## はじめに — 激動の3月を振り返る

2026年3月は、Google Gemini にとって歴史的な転換点となる1か月でした。Gemini 3.1 Pro / Flash の正式リリース、月間アクティブユーザー7.5億人突破、そして ChatGPT からの乗り換えを促すチャット履歴インポート機能の発表と、毎週のように大きなニュースが飛び込んできました。

Gemini Lab でも今月は 270 本を超える記事を公開し、入門ガイドから本番運用まで幅広いトピックをカバーしてきました。この月間まとめでは、3月の主要アップデートを整理しつつ、Gemini Lab で特に読まれた人気記事、そして4月に向けた注目ポイントをお伝えします。

## 今月の主要アップデート

### Gemini 3.1 Pro / Flash が正式リリース

3月最大のニュースは、Gemini 3.1 シリーズの正式公開です。Gemini 3.1 Pro は複雑なタスクにおける推論精度が大幅に向上し、`gemini-3.1-pro-preview-customtools` エンドポイントでは bash ツールとカスタム Function Calling を1つの API コールで併用できるようになりました。

一方、[Gemini 3.1 Flash 高速推論APIの実装テクニック](/articles/gemini-api/gemini-3-1-flash-api-implementation-techniques)で詳しく解説しているとおり、Flash モデルはストリーミング・バッチ処理の最適化が進み、本番環境での採用がさらに現実的になっています。

### Gemini 3.1 Flash-Lite — 史上最もコスト効率の高いモデル

3月3日にプレビュー公開された Flash-Lite は、入力 $0.25/100万トークン・出力 $1.50/100万トークンという破格の価格設定です。Gemini 2.5 Flash と比較して初回トークン応答が 2.5 倍高速化し、出力速度も 45% 向上しています。大量のリクエストを処理するプロダクションワークロードにとって、コスト削減の切り札となるモデルです。

### ChatGPT / Claude チャット履歴インポート機能

3月26日に発表されたこの機能は、AI 業界に大きなインパクトを与えました。ユーザーは ChatGPT や Claude から会話ログを .zip 形式でエクスポートし、`gemini.google.com/import` にアップロードすることで、過去の会話をシームレスに引き継げます。

「メモリインポート」機能では、既存のチャットボットに専用プロンプトを入力してユーザープロフィールの要約を生成し、それを Gemini に取り込むことで、初日から「あなたのことを理解した AI」として使い始めることができます。

### Google AI Studio の大幅リニューアル

3月中旬に実施された [Google AI Studio の UI 刷新](https://www.androidsage.com/2026/03/19/google-ai-studios-biggest-update-yet/)では、Gemini、GenMedia（Veo 3.1）、TTS、Live モデルを1つの統合プレイグラウンドで利用できるようになりました。タブを切り替えることなく、テキスト生成から音声合成、動画生成まで一気通貫で試せる環境が整っています。

さらに、ビルトインツールとカスタム Function Calling を1つの API コールで組み合わせられる新機能も追加され、エージェント開発の柔軟性が飛躍的に向上しました。

### Workspace 連携の進化

3月10日には Google Workspace 全体で Gemini 機能が強化されました。Docs ではドキュメントの作成・編集をAIがアシスト、Sheets ではデータ分析と関数生成が自動化、Slides ではAIベースのプレゼンデザインが可能になり、Drive では複数ファイルやメールを横断した質問応答ができるようになっています。

Gemini Lab では [Gemini × Google Sheets AI 実践ガイド](/articles/gemini-workspace/gemini-google-sheets-ai-data-analysis)や [Google Slides × Gemini AI 実践ガイド](/articles/gemini-workspace/gemini-google-slides-ai-presentation-guide)で、これらの新機能の活用法を詳しく解説しています。

### 月間ユーザー7.5億人突破

Google は3月に Gemini の月間アクティブユーザーが 7.5 億人を突破したことを発表しました。2025 年末時点の推定 3.5 億人から倍増以上のペースで、ChatGPT との差を急速に縮めています。この急成長の背景には、Android / Pixel へのネイティブ統合、Workspace 連携の深化、そして Google TV や Home デバイスへの展開があります。

## よくある質問（FAQ）

**Q: Gemini 3.1 Pro と 3.0 Pro の違いは何ですか？**

A: 3.1 Pro は推論精度が向上し、カスタムツールと bash ツールの同時使用に対応した新エンドポイントが追加されています。また、コンテキストウィンドウの効率的な活用やトークン処理速度も改善されており、本番環境でのパフォーマンスが大幅に向上しています。

**Q: ChatGPT からの履歴インポートはどの地域で利用できますか？**

A: 2026年3月末時点では、EEA（欧州経済領域）、スイス、英国を除くほとんどの地域で利用可能です。データ規制の関係で一部地域では提供が遅れていますが、今後段階的に拡大される見込みです。インポートは `gemini.google.com/import` から行えます。

**Q: 4月に Gemini で最も注目すべきことは何ですか？**

A: Gemini 3.1 Pro の一般提供（GA）移行が最大の注目ポイントです。GA 化に伴い、SLA（サービスレベル契約）が付与され、本番環境での利用がより安心して行えるようになります。また、Google I/O 2026 に向けたプレアナウンスメントにも注目です。
