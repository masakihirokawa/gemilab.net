## Recipe 1: Email Summarization and Reply Drafting

Tame your inbox by having Gemini summarize emails and draft replies for you.

```python
# recipe_01_email_summary.py — Summarize emails + draft replies
from common_setup import client, MODEL

def summarize_email(email_text: str) -> str:
    """Summarize an email and generate a reply draft."""
    prompt = f"""Analyze the following email and return the output in this format:

**Summary** (3 lines max)
**Urgency**: High / Medium / Low
**Action Required**:
**Draft Reply**:

---
{email_text}
"""
    response = client.models.generate_content(
        model=MODEL,
        contents=prompt
    )
    return response.text

# Example usage
email = """
Hi Alex,

The budget for Project Alpha has been approved by finance.
Could you submit the implementation plan by next Monday?

This also affects next fiscal year's budget, so a quick
turnaround would be appreciated.

Best,
Jordan
"""

result = summarize_email(email)
print(result)
# Expected output:
# **Summary**: Project Alpha budget approved. Implementation plan due next Monday.
# **Urgency**: High
# **Action Required**: Submit implementation plan by Monday
# **Draft Reply**: Hi Jordan, Thanks for the update! I'll have the plan ready by Monday...
```

## Recipe 4: Image Captions and Alt Text Generation

Automatically generate captions, alt text, and hashtags for your images.

```python
# recipe_04_image_caption.py — Generate captions + alt text from images
import pathlib
from common_setup import client, MODEL

def generate_image_caption(image_path: str) -> str:
    """Generate captions and alt text for an image."""
    image_data = pathlib.Path(image_path).read_bytes()
    ext = pathlib.Path(image_path).suffix.lower()
    mime_map = {".png": "image/png", ".jpg": "image/jpeg", ".jpeg": "image/jpeg", ".webp": "image/webp"}
    mime_type = mime_map.get(ext, "image/jpeg")

    response = client.models.generate_content(
        model=MODEL,
        contents=[
            {
                "role": "user",
                "parts": [
                    {"inline_data": {"mime_type": mime_type, "data": image_data}},
                    {"text": """For this image, please generate:
1. A social media caption (under 50 words)
2. Alt text for web accessibility (under 100 words)
3. An SEO-friendly description (under 150 words)
4. 5 relevant hashtags"""}
                ]
            }
        ]
    )
    return response.text
```

## Recipe 8: Daily Status Report Generator

```python
# recipe_08_daily_report.py — Generate daily status reports
from datetime import datetime
from common_setup import client, MODEL

def generate_daily_report(tasks: list[str], issues: list[str] = None) -> str:
    """Generate a daily status report from task lists."""
    today = datetime.now().strftime("%B %d, %Y")

    issues_text = "\n".join(f"- {i}" for i in (issues or []))

Date: {today}
Completed tasks:
{tasks_text}

Issues/blockers:
{issues_text if issues_text else "None"}

**Report format:**
■ Today's Accomplishments (progress % and outcomes for each)
■ Issues & Risks (impact and mitigation)
■ Tomorrow's Plan (prioritized)
■ Notes
"""
    response = client.models.generate_content(
        model=MODEL,
        contents=prompt
    )
    return response.text

# Example usage
tasks = [
    "Implemented authentication API endpoints",
```

## Recipe 10: Batch Processing Multiple Files

Process an entire folder of files in one go with configurable tasks.

```python
# recipe_10_batch_processing.py — Batch process multiple files
import pathlib
import json
from common_setup import client, MODEL

def batch_process_files(
    input_dir: str,
    output_dir: str,
    task: str = "summarize"
) -> list[dict]:
    """Batch process text files in a directory."""
    input_path = pathlib.Path(input_dir)
    output_path = pathlib.Path(output_dir)
    output_path.mkdir(parents=True, exist_ok=True)

    task_prompts = {
        "summarize": "Summarize the following text in under 200 words.",
        "translate": "Translate the following text to Japanese.",
        "keywords": "Extract the 10 most important keywords from the following text.",
    }
    prompt_template = task_prompts.get(task, task)
    results = []

    for file in sorted(input_path.glob("*.txt")):
        content = file.read_text(encoding="utf-8")
        prompt = f"{prompt_template}\n\n---\n{content}"

        response = client.models.generate_content(
            model=MODEL,
            contents=prompt
        )

        output_file = output_path / f"{file.stem}_{task}.txt"
        output_file.write_text(response.text, encoding="utf-8")
        results.append({"file": file.name, "status": "success"})
        print(f"✅ {file.name} → {output_file.name}")

    # Save a processing summary
    summary_file = output_path / "batch_summary.json"
    summary_file.write_text(
        json.dumps(results, ensure_ascii=False, indent=2),
        encoding="utf-8"
    )
    return results

# Example usage
# batch_process_files("./input_docs", "./output_docs", task="summarize")
# Expected output:
# ✅ report_01.txt → report_01_summarize.txt
# ✅ report_02.txt → report_02_summarize.txt
# ✅ report_03.txt → report_03_summarize.txt
```

## Frequently Asked Questions

### Can I run all these recipes on the free tier?

Yes — every recipe works within the Gemini API free tier. Gemini 2.5 Flash offers a generous request allowance that's more than enough for personal use and learning. If you plan on running heavy batch jobs, review the [API Rate Limiting and Quota Management Guide](/en/articles/gemini-api/gemini-api-rate-limiting-quota-management) for details on limits and best practices.

### What should I consider before using these in production?

For production deployments, you'll want to add proper error handling, secure API key management (use environment variables or a secret manager), and rate-limit retry logic. Validating API responses before acting on them is also a must. For a deeper dive, see the [Gemini API Cost Optimization Guide](/en/articles/gemini-api/gemini-api-cost-optimization).

### Can I use these recipes in other programming languages?

Absolutely. The Gemini API is available as a REST API, so any language that can make HTTP requests will work. Google provides official SDKs for Python, JavaScript/TypeScript, Go, and Dart. While this article uses Python, the logic translates easily to other stacks.

### How can I improve the quality of the generated results?

The biggest lever is prompt specificity. Clearly define the output format, include concrete examples, and state your constraints explicitly. These small changes can dramatically improve output quality. For a comprehensive guide, check out our [Gemini Prompt Engineering Guide](/en/articles/gemini-api/gemini-prompt-engineering-guide).
