## Gemini Multimodal Embedding - A New Era Begins

In March 2026, Google introduced **gemini-embedding-2-preview**, the industry's first multimodal embedding model capable of mapping text, images, video, audio, and PDF documents into a single unified vector space. Previously, separate models were required for different data modalities, but gemini-embedding-2 eliminates this constraint entirely.

## Understanding Multimodal Embeddings

### Core Concepts

Embedding is the process of converting data like text or images into numerical vectors of hundreds or thousands of dimensions. Multimodal embedding converts multiple different data formats (text, images, video, etc.) into **the same vector space**.

```
Text: "A scenic view of Mount Fuji in Japan"
  ↓
[0.23, -0.54, 0.81, ..., 0.12]  ← 768-dimensional vector

Image: Mount Fuji photograph
  ↓
[0.24, -0.53, 0.79, ..., 0.11]  ← 768-dimensional vector (nearly identical!)

Video: Mount Fuji sunrise footage
  ↓
[0.25, -0.52, 0.82, ..., 0.10]  ← 768-dimensional vector (very close!)
```

Since they're converted to the same space, you can now calculate **semantic similarity** between text and images, images and video, and more.

### Comparison with Traditional Approaches

| Feature | Traditional Embeddings | gemini-embedding-2 |
|---------|---------------------|-----------------|
| Modalities | Text only | Text, Image, Video, Audio, PDF |
| Vector Space | Different per modality | Unified |
| Cross-modal Search | Not possible | Fully supported |
| Semantic Matching | Within modality only | Across all modalities |
| Required APIs | 5+ | Just 1 |

## Practical Applications

### 1. Next-Generation RAG (Retrieval-Augmented Generation)

Traditional RAG was limited to text documents. With gemini-embedding-2, you can search across text, images, and video simultaneously.

```python
import numpy as np
from google.generativeai import embed_content, generate_content

class MultimodalRAG:
    def __init__(self, documents, model="models/gemini-embedding-2-preview"):
        self.model = model
        self.documents = documents
        self.embeddings = self._embed_all_documents()

    def _embed_all_documents(self):
        """Embed all documents (text, images, videos) into vector space"""
        embeddings = []
        for doc in self.documents:
            if doc["type"] == "text":
                response = embed_content(
                    model=self.model,
                    content={"parts": [{"text": doc["content"]}]}
                )
            elif doc["type"] == "image":
                response = embed_content(
                    model=self.model,
                    content={
                        "parts": [{
                            "inline_data": {
                                "mime_type": "image/jpeg",
                                "data": doc["content"]
                            }
                        }]
                    }
                )
            elif doc["type"] == "video":
                response = embed_content(
                    model=self.model,
                    content={
                        "parts": [{
                            "file_data": {
                                "mime_type": "video/mp4",
                                "file_uri": doc["content"]
                            }
                        }]
                    }
                )
            embeddings.append(response["embedding"])
        return embeddings

    def retrieve(self, query, top_k=3):
        """Retrieve top k documents most relevant to query"""
        query_embedding = embed_content(
            model=self.model,
            content={"parts": [{"text": query}]}
        )["embedding"]

        # Calculate cosine similarity
        similarities = []
        for i, doc_embedding in enumerate(self.embeddings):
            similarity = np.dot(query_embedding, doc_embedding) / (
                np.linalg.norm(query_embedding) * np.linalg.norm(doc_embedding)
            )
            similarities.append((i, similarity))

        # Return top k
        top_indices = sorted(similarities, key=lambda x: x[1], reverse=True)[:top_k]
        return [self.documents[idx] for idx, _ in top_indices]

    def generate_with_rag(self, query):
        """Generate answer using RAG with LLM"""
        relevant_docs = self.retrieve(query)

        context = "Reference materials:\n"
        for doc in relevant_docs:
            context += f"- [{doc['type']}] {doc['title']}\n"

        prompt = f"{context}\nQuestion: {query}"

        response = generate_content(
            model="gemini-pro",
            contents=[{"parts": [{"text": prompt}]}]
        )
        return response.text

# Usage example
documents = [
    {
        "type": "text",
        "title": "Japanese Spring Traditions",
        "content": "Cherry blossom viewing is a traditional spring activity in Japan..."
    },
    {
        "type": "image",
        "title": "Kyoto Cherry Blossoms",
        "content": base64_cherry_blossom_kyoto
    },
    {
        "type": "video",
        "title": "Tokyo Spring Landscapes",
        "content": "gs://bucket/tokyo_spring.mp4"
    }
]

rag = MultimodalRAG(documents)
result = rag.generate_with_rag("Tell me about spring in Japan")
print(result)
```

### 3. Multimodal Recommendation System

Recommend similar content based on user viewing history spanning text articles, images, and videos.

```python
class MultimodalRecommender:
    def __init__(self, user_history, all_content):
        self.user_history = user_history  # Content user has viewed
        self.all_content = all_content    # All available content
        self.user_profile = self._build_user_profile()

    def _build_user_profile(self):
        """Build comprehensive user profile from viewing history"""
        embeddings = []
        for item in self.user_history:
            emb = embed_content(
                model="models/gemini-embedding-2-preview",
                content=_prepare_doc_content(item)
            )["embedding"]
            embeddings.append(emb)

        # Use mean vector as profile
        return np.mean(embeddings, axis=0)

    def recommend(self, top_k=10):
        """Recommend top_k pieces of content based on user profile"""
        recommendations = []

        for content in self.all_content:
            # Skip already viewed
            if content["id"] in [h["id"] for h in self.user_history]:
                continue

            # Embed content
            content_embedding = embed_content(
                model="models/gemini-embedding-2-preview",
                content=_prepare_doc_content(content)
            )["embedding"]

            # Similarity with user profile
            similarity = np.dot(self.user_profile, content_embedding) / (
                np.linalg.norm(self.user_profile) * np.linalg.norm(content_embedding)
            )

            recommendations.append((content, similarity))

        return sorted(recommendations, key=lambda x: x[1], reverse=True)[:top_k]
```

```python
    """Automatically split video into chapters"""

    chapters = []
    current_chapter = {
        "start": 0,
        "frames": [],
        "embeddings": []
    }

    # Extract frames from video (simplified)
    frames = extract_frames_from_video(video_uri, interval_sec=10)

    for frame in frames:
        # Embed frame
        frame_emb = embed_content(
            model="models/gemini-embedding-2-preview",
            content={
                "parts": [{
                    "inline_data": {
                        "mime_type": "image/jpeg",
                        "data": frame["data"]
                    }
                }]
            }
        )["embedding"]

        current_chapter["frames"].append(frame)
        current_chapter["embeddings"].append(frame_emb)

        # Check chapter length
        if len(current_chapter["frames"]) >= chapter_duration_sec // 10:
            # Calculate scene change (distance between first and last frame)
            first_emb = np.array(current_chapter["embeddings"][0])
            last_emb = np.array(current_chapter["embeddings"][-1])
            scene_change = 1 - np.dot(first_emb, last_emb) / (
                np.linalg.norm(first_emb) * np.linalg.norm(last_emb)
            )

            chapters.append({
                "start": current_chapter["start"],
                "end": frame["timestamp"],
                "scene_change_score": scene_change
            })

            current_chapter = {
                "start": frame["timestamp"],
                "frames": [],
                "embeddings": []
            }

    return chapters
```

## Vector Database Integration

Vectors generated by gemini-embedding-2 integrate seamlessly with vector search databases.

### Pinecone Integration Example

```python
import pinecone
from google.generativeai import embed_content

# Initialize Pinecone index
pinecone.init(api_key="your-api-key")
index = pinecone.Index("gemini-multimodal")

# Embed and upload documents
def index_documents(documents):
    for i, doc in enumerate(documents):
        embedding = embed_content(
            model="models/gemini-embedding-2-preview",
            content=_prepare_doc_content(doc)
        )["embedding"]

        index.upsert(
            vectors=[
                (str(i), embedding, {
                    "type": doc["type"],
                    "title": doc["title"],
                    "source": doc["source"]
                })
            ]
        )

# Execute query
def search(query):
    query_embedding = embed_content(
        model="models/gemini-embedding-2-preview",
        content={"parts": [{"text": query}]}
    )["embedding"]

    results = index.query(
        vector=query_embedding,
        top_k=10,
        include_metadata=True
    )
    return results
```

<Callout type="warning">
Vector search performs approximate nearest neighbor search, not exact matching. Production systems should include filtering and re-ranking of results for quality assurance.
</Callout>

## Performance Optimization

### Latency Optimization

```python
# Batch processing multiple documents simultaneously
def batch_embed(documents, batch_size=32):
    """Batch processing optimizes both API cost and latency"""
    embeddings = []

    for i in range(0, len(documents), batch_size):
        batch = documents[i:i+batch_size]

        # Process multiple parts in single request
        parts = []
        for doc in batch:
            if doc["type"] == "text":
                parts.append({"text": doc["content"]})
            elif doc["type"] == "image":
                parts.append({
                    "inline_data": {
                        "mime_type": "image/jpeg",
                        "data": doc["content"]
                    }
                })

        response = embed_content(
            model="models/gemini-embedding-2-preview",
            content={"parts": parts}
        )

        embeddings.extend(response["embeddings"])

    return embeddings
```

### Cost Reduction Strategies

- Use dedicated text embedding models for text-only similarity calculations
- Embed images and videos only when needed (enable caching)
- Use batch processing to minimize API calls
