## Overview

In March 2026, Google achieved a groundbreaking technological advancement in the AI field. **gemini-embedding-2-preview** — the first multimodal embedding model that maps text, images, video, audio, and PDFs into a unified vector space — has been released.

Previous embedding technologies were specialized for single modalities (text only, etc.). However, real-world data exists in diverse formats. gemini-embedding-2-preview solves this challenge by projecting all modalities into the same vector space, enabling true cross-modal search and reasoning capabilities.

This complete guide covers every aspect of gemini-embedding-2-preview, from technical specifications to practical implementation.

## Understanding Multimodal Embeddings

### Limitations of Traditional Embeddings

Traditional text-only embedding models converted strings into vector space:

```
Text → Encoder → Vector [0.234, -0.512, 0.789, ...]
```

Problems with this approach:
- Cannot directly embed images or videos
- Cannot capture relationships between different modalities
- Cannot understand that "a cat video" and "a cat image" are semantically related

### The Innovation of Multimodal Embeddings

gemini-embedding-2-preview projects multiple modalities into the **same vector space**:

```
Text        ┐
Image       ├→ Multimodal ├→ Unified Vector Space
Video       │  Encoder    │  [0.234, -0.512, 0.789, ...]
Audio       │             │
PDF         ┘             ↓
          (All vectors are
           mutually comparable)
```

This design enables innovative capabilities:

1. **Cross-Modal Search**: Search images with text queries, search videos with image queries
2. **Multimodal RAG**: Search documents, images, and videos using a unified vector basis
3. **Semantic Similarity**: Discover meaningful relationships across different modalities

## Understanding the Unified Vector Space

### Mathematical Foundation

The core of gemini-embedding-2-preview is normalizing multiple modality encoder outputs into a single vector space:

```
Modality → Specialized Encoder → Intermediate → Normalization → Unified Space Vector
 (Text)     (Text conversion)   (Variable dims)  (768 dims)
 (Image)    (Visual features)   (Variable dims)
 (Video)    (Temporal features) (Variable dims)
```

### Cosine Similarity for Search

In the unified vector space, all vectors are comparable using cosine similarity:

```python
# Similarity score (0 to 1)
similarity = (embedding1 · embedding2) / (||embedding1|| × ||embedding2||)

# Score interpretation
# 0.95+ : Nearly identical content
# 0.80-0.95 : Strongly related
# 0.60-0.80 : Related
# 0.40-0.60 : Weakly related
# 0.00-0.40 : Unrelated
```

### Real-World Example


### Embedding Images

```python
import base64
from pathlib import Path

def embed_image(image_path: str):
    """Embed an image file"""
    with open(image_path, 'rb') as f:
        image_data = base64.standard_b64encode(f.read()).decode('utf-8')

    # Determine file type
    suffix = Path(image_path).suffix.lower()
    media_type_map = {
        '.jpg': 'image/jpeg',
        '.jpeg': 'image/jpeg',
        '.png': 'image/png',
        '.webp': 'image/webp',
        '.heic': 'image/heic'
    }

    media_type = media_type_map.get(suffix, 'image/jpeg')

    embedding = genai.embed_content(
        model="models/embedding-2-preview",
        content={
            "mime_type": media_type,
            "data": image_data
        }
    )

    return embedding['embedding']

# Example usage
cat_image_embedding = embed_image("cat.jpg")
print(f"Image vector obtained: {len(cat_image_embedding)} dimensions")
```

### Cross-Modal Similarity Calculation

```python
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

def calculate_similarity(embedding1, embedding2):
    """Calculate cosine similarity between two vectors"""
    embedding1 = np.array(embedding1).reshape(1, -1)
    embedding2 = np.array(embedding2).reshape(1, -1)
    return cosine_similarity(embedding1, embedding2)[0][0]

# Calculate similarity between text and image
text_query = "Yellow dog running in the park"
text_query_embedding = genai.embed_content(
    model="models/embedding-2-preview",
    content=text_query
)['embedding']

image_embedding = embed_image("dog_in_park.jpg")

similarity = calculate_similarity(
    text_query_embedding,
    image_embedding
)
print(f"Text-Image similarity: {similarity:.3f}")
```

## Practical Use Cases

### 1. Cross-Modal Search Engine

```python
import json
from typing import List, Dict

class MultimodalSearchEngine:
    """Multimodal search engine"""

    def __init__(self, model_name: str = "models/embedding-2-preview"):
        self.model_name = model_name
        self.indexed_items = []  # Use vector DB in production

    def index_content(self, items: List[Dict]):
        """Index multiple content items"""
        for item in items:
            content_type = item['type']  # 'text', 'image', 'video', etc.
            content_value = item['value']

            if content_type == 'text':
                embedding = genai.embed_content(
                    model=self.model_name,
                    content=content_value,
                    task_type="retrieval_document"
                )['embedding']
            elif content_type == 'image':
                with open(content_value, 'rb') as f:
                    img_data = base64.standard_b64encode(f.read()).decode()
                embedding = genai.embed_content(
                    model=self.model_name,
                    content={"mime_type": "image/jpeg", "data": img_data}
                )['embedding']

            self.indexed_items.append({
                'content': item,
                'embedding': embedding
            })

    def search(self, query: str, query_type: str = 'text', top_k: int = 5):
        """Perform cross-modal search with text, image, or video"""
        if query_type == 'text':
            query_embedding = genai.embed_content(
                model=self.model_name,
                content=query,
                task_type="retrieval_query"
            )['embedding']

        # Calculate similarity scores and sort by top K results
        scores = []
        for item in self.indexed_items:
            similarity = calculate_similarity(
                query_embedding,
                item['embedding']
            )
            scores.append({
                'item': item['content'],
                'score': similarity
            })

        return sorted(scores, key=lambda x: x['score'], reverse=True)[:top_k]

# Example usage
search_engine = MultimodalSearchEngine()
results = search_engine.search("dog running", query_type='text', top_k=5)
```

### 2. Image-Inclusive RAG System

```python
class MultimodalRAG:
    """Multimodal document search and generation"""

    def __init__(self):
        self.documents = []
        self.embeddings = []

    def add_document(self, doc_content: str, doc_metadata: Dict):
        """Add document (text + images)"""
        embedding = genai.embed_content(
            model="models/embedding-2-preview",
            content=doc_content,
            task_type="retrieval_document"
        )['embedding']

        self.documents.append({
            'content': doc_content,
            'metadata': doc_metadata
        })
        self.embeddings.append(embedding)

    def retrieve(self, query: str, top_k: int = 3) -> List[Dict]:
        """Retrieve related documents"""
        query_embedding = genai.embed_content(
            model="models/embedding-2-preview",
            content=query,
            task_type="retrieval_query"
        )['embedding']

        similarities = []
        for idx, doc_embedding in enumerate(self.embeddings):
            sim = calculate_similarity(query_embedding, doc_embedding)
            similarities.append((idx, sim))

        top_indices = sorted(similarities, key=lambda x: x[1], reverse=True)[:top_k]
        return [self.documents[idx] for idx, _ in top_indices]

    def generate_response(self, query: str):
        """Generate response based on retrieved documents"""
        retrieved_docs = self.retrieve(query)

        context = "\n".join([
            f"Document {i+1}:\n{doc['content'][:500]}"
            for i, doc in enumerate(retrieved_docs)
        ])

        response = genai.generate_content(
            f"""Based on the following relevant documents, answer the question.

【Relevant Documents】
{context}

【Question】
{query}

【Answer】"""
        )

        return response.text
```

```python
class VideoRecommendationSystem:
    """Video content recommendation engine"""

        self.videos.append({
            'path': video_path,
            'metadata': metadata
        })
        self.embeddings.append(embedding)

    def recommend(self, video_path: str, top_k: int = 5):
        """Recommend similar videos"""
        with open(video_path, 'rb') as f:
            video_data = base64.standard_b64encode(f.read()).decode()

        query_embedding = genai.embed_content(
            model="models/embedding-2-preview",
            content={"mime_type": "video/mp4", "data": video_data}
        )['embedding']

        similarities = []
        for idx, emb in enumerate(self.embeddings):
            sim = calculate_similarity(query_embedding, emb)
            similarities.append((idx, sim))

        top_videos = sorted(similarities, key=lambda x: x[1], reverse=True)[1:top_k+1]

        return [
            {
                'video': self.videos[idx],
                'similarity_score': score
            }
            for idx, score in top_videos
        ]
```

```python
import pinecone

def setup_pinecone_with_multimodal(api_key: str, index_name: str):
    """Set up multimodal index in Pinecone"""
    pinecone.init(api_key=api_key, environment="gcp-starter")

    # Create 768-dimension index

def index_to_pinecone(items: List[Dict], index):
    """Index multimodal items to Pinecone"""
    vectors = []

    for item in items:
        if item['type'] == 'text':
            embedding = genai.embed_content(
                model="models/embedding-2-preview",
                content=item['content'],
                task_type="retrieval_document"
            )['embedding']
        elif item['type'] == 'image':
            with open(item['content'], 'rb') as f:
                img_data = base64.standard_b64encode(f.read()).decode()
            embedding = genai.embed_content(
                model="models/embedding-2-preview",
                content={"mime_type": "image/jpeg", "data": img_data}
            )['embedding']

        vectors.append((
            item['id'],
            embedding,
            {"type": item['type'], "source": item['source']}
        ))

    index.upsert(vectors=vectors)
```

## RAG Pipeline Architecture Evolution

### Traditional Text-Only RAG

```
Documents → Text Extraction → Text Embedding → Vector DB
                                    ↓
Query → Text Embedding → Vector Search → Top K Retrieval → LLM Generation
```

### New Multimodal RAG Architecture

```
Document Collection (text, images, videos)
    ↓
Multimodal Embedding (unified vector space)
    ↓
Hybrid Vector DB (with metadata)
    ↓
    ┌─────────────────────────┐
    │ Support Diverse Queries │
    │ (text/image/video)      │
    └─────────────────────────┘
    ↓
Multimodal Content Search
    ↓
    ┌─────────────────────────────────────┐
    │ Retrieve All Related Modalities      │
    │ (text + images + video chunks)      │
    └─────────────────────────────────────┘
    ↓
LLM with Vision Capabilities for Synthesis
    ↓
```

### Implementation Example

```python
class AdvancedMultimodalRAG:
    """Next-generation multimodal RAG system"""

    def __init__(self, vector_db_client):
        self.vector_db = vector_db_client
        self.model_name = "models/embedding-2-preview"

    def process_multimodal_document(self, doc_path: str):
        """Process composite document (text + images)"""
        chunks = self.extract_chunks(doc_path)
        processed_chunks = []

        for chunk in chunks:
            if chunk['type'] == 'text':
                embedding = self.embed_text(chunk['content'])
            elif chunk['type'] == 'image':
                embedding = self.embed_image(chunk['path'])

            processed_chunks.append({
                'content': chunk,
                'embedding': embedding,
                'modality': chunk['type']
            })

        return processed_chunks

    def multimodal_query(self, query: str, query_type: str = 'text'):
        """Search based on query modality"""
        if query_type == 'text':
            query_embedding = genai.embed_content(
                model=self.model_name,
                content=query,
                task_type="retrieval_query"
            )['embedding']

        # Retrieve related chunks from vector DB
        results = self.vector_db.query(
            query_embedding,
            top_k=10,
            include_metadata=True
        )

        # Organize results by modality
        multimodal_context = self.organize_by_modality(results)

        return multimodal_context

    def generate_enriched_response(self, query: str):
        """Generate rich response from multimodal context"""
        context = self.multimodal_query(query)

        # Prepare text context
        text_context = "\n".join([
            chunk['content']
            for chunk in context.get('text', [])[:3]
        ])

        # Generate with LLM (vision-capable)
        response = genai.generate_content([
            f"Based on the following relevant information, provide an answer:\n{text_context}\n\nQuestion: {query}",
            # Add images if available
            *[{"mime_type": "image/jpeg", "data": img}
              for img in context.get('images', [])[:2]]
        ])

        return response.text
```
