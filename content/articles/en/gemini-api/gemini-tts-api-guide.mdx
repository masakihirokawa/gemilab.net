## Introduction: What Is Gemini 2.5 TTS?

Google significantly upgraded the text-to-speech (TTS) capabilities of the Gemini API in December 2025, and the March 2026 update added **enhanced expressivity, precision pacing, and seamless dialogue generation** — pushing AI-powered speech synthesis into a new era.

Gemini 2.5 TTS is available in two models:

- **`gemini-2.5-flash-preview-tts`** — Optimized for low latency. Best for real-time assistants and high-volume narration.
- **`gemini-2.5-pro-preview-tts`** — Optimized for quality. Best for podcasts, premium content, and rich emotional delivery.

In this guide, you'll learn how to use both models via the Python SDK with practical code examples.

## Basic Usage: Your First TTS Generation

Here's the simplest way to generate speech from text. Pass in a string and receive audio binary back.

```python
import os
import wave
from google import genai
from google.genai import types

# Initialize the client
client = genai.Client(api_key=os.environ["GOOGLE_API_KEY"])

# Generate speech from text
response = client.models.generate_content(
    model="gemini-2.5-flash-preview-tts",
    contents="Hello! This is a Gemini TTS test.",
    config=types.GenerateContentConfig(
        response_modalities=["AUDIO"],
        speech_config=types.SpeechConfig(
            voice_config=types.VoiceConfig(
                prebuilt_voice_config=types.PrebuiltVoiceConfig(
                    voice_name="Kore"
                )
            )
        )
    )
)

# Extract audio data
audio_data = response.candidates[0].content.parts[0].inline_data.data

# Save as WAV file
with wave.open("output.wav", "wb") as wav_file:
    wav_file.setnchannels(1)   # Mono
    wav_file.setsampwidth(2)   # 16-bit PCM
    wav_file.setframerate(24000)  # 24kHz
    wav_file.writeframes(audio_data)

print("✅ Saved to output.wav")
# Expected output: ✅ Saved to output.wav
```

## Controlling Speech Style with Style Prompts

One of the most powerful features of Gemini TTS is the ability to use **natural language style prompts** to control tone, pacing, and atmosphere. Structure your prompt with three components:

1. **Audio Profile** — Defines the character's core identity and archetype
2. **Scene** — Establishes the physical environment and emotional "vibe"
3. **Director's Notes** — Provides precise performance guidance

```python
import os
import wave
from google import genai
from google.genai import types

client = genai.Client(api_key=os.environ["GOOGLE_API_KEY"])

# TTS generation with style prompt
style_prompt = """
Audio Profile: A professional podcast host. Calm, intelligent, and authoritative.
Scene: Studio recording environment. Quiet and focused atmosphere.
Director's Notes: Speak slowly and clearly. Slightly emphasize key technical terms.
"""

text_to_speak = (
    "Today, we're diving into the latest features of the Gemini API. "
    "The most notable advancement is the dramatic evolution of text-to-speech capabilities."
)

response = client.models.generate_content(
    model="gemini-2.5-pro-preview-tts",  # Use the quality-focused model
    contents=f"{style_prompt}\n\n{text_to_speak}",
    config=types.GenerateContentConfig(
        response_modalities=["AUDIO"],
        speech_config=types.SpeechConfig(
            voice_config=types.VoiceConfig(
                prebuilt_voice_config=types.PrebuiltVoiceConfig(
                    voice_name="Aoede"
                )
            )
        )
    )
)

audio_data = response.candidates[0].content.parts[0].inline_data.data

with wave.open("podcast_style.wav", "wb") as wav_file:
    wav_file.setnchannels(1)
    wav_file.setsampwidth(2)
    wav_file.setframerate(24000)
    wav_file.writeframes(audio_data)

print("✅ Saved to podcast_style.wav")
# Expected output: ✅ Saved to podcast_style.wav
```

### Style Prompt Examples

| Style Instruction | Use Case |
|-------------------|----------|
| `Upbeat and energetic, fast pace` | Ads, promo videos |
| `Deep and authoritative, calm tone` | News narration |
| `Warm and approachable` | Support bots, virtual assistants |
| `Tense and dramatic` | Game narration, trailers |
| `Slow and meditative` | Wellness, meditation content |

## Multi-Speaker Dialogue Generation

Gemini 2.5 TTS supports multi-speaker dialogue — generate a full conversation with distinct voices in a single API call. Perfect for podcasts, interviews, or interactive content.

```python
import os
import wave
from google import genai
from google.genai import types

client = genai.Client(api_key=os.environ["GOOGLE_API_KEY"])

# Generate multi-speaker dialogue
response = client.models.generate_content(
    model="gemini-2.5-pro-preview-tts",
    contents="""
    TTS_SPEAKERS:
    - Speaker1: Aoede (Female, bright and energetic)
    - Speaker2: Charon (Male, calm and measured)

    DIALOGUE:
    [Speaker1]: Have you tried the new Gemini TTS features yet?
    [Speaker2]: I have — the expressivity improvements are genuinely impressive.
    [Speaker1]: The precision pacing is my favorite part. It adjusts automatically based on context.
    [Speaker2]: For developers, the style prompt system is a game changer. So much easier to control.
    """,
    config=types.GenerateContentConfig(
        response_modalities=["AUDIO"],
        speech_config=types.SpeechConfig(
            multi_speaker_voice_config=types.MultiSpeakerVoiceConfig(
                speaker_voice_configs=[
                    types.SpeakerVoiceConfig(
                        speaker="Speaker1",
                        voice_config=types.VoiceConfig(
                            prebuilt_voice_config=types.PrebuiltVoiceConfig(
                                voice_name="Aoede"
                            )
                        )
                    ),
                    types.SpeakerVoiceConfig(
                        speaker="Speaker2",
                        voice_config=types.VoiceConfig(
                            prebuilt_voice_config=types.PrebuiltVoiceConfig(
                                voice_name="Charon"
                            )
                        )
                    )
                ]
            )
        )
    )
)

audio_data = response.candidates[0].content.parts[0].inline_data.data

with wave.open("dialogue.wav", "wb") as wav_file:
    wav_file.setnchannels(1)
    wav_file.setsampwidth(2)
    wav_file.setframerate(24000)
    wav_file.writeframes(audio_data)

print("✅ Multi-speaker audio saved to dialogue.wav")
# Expected output: ✅ Multi-speaker audio saved to dialogue.wav
```

## Streaming Mode: Real-Time Audio Generation

For real-time assistants and live-response applications, streaming mode minimizes perceived latency.

```python
import os
import wave
from google import genai
from google.genai import types

client = genai.Client(api_key=os.environ["GOOGLE_API_KEY"])

# Receive audio via streaming
audio_chunks = []

for chunk in client.models.generate_content_stream(
    model="gemini-2.5-flash-preview-tts",  # Low-latency model
    contents="Streaming TTS test. Audio is generated in real time as you receive chunks.",
    config=types.GenerateContentConfig(
        response_modalities=["AUDIO"],
        speech_config=types.SpeechConfig(
            voice_config=types.VoiceConfig(
                prebuilt_voice_config=types.PrebuiltVoiceConfig(
                    voice_name="Kore"
                )
            )
        )
    )
):
    if (chunk.candidates and
        chunk.candidates[0].content.parts and
        chunk.candidates[0].content.parts[0].inline_data):
        audio_chunks.append(
            chunk.candidates[0].content.parts[0].inline_data.data
        )
        print(".", end="", flush=True)

print("\n✅ Streaming complete")

# Combine chunks and save
full_audio = b"".join(audio_chunks)
with wave.open("streaming_output.wav", "wb") as wav_file:
    wav_file.setnchannels(1)
    wav_file.setsampwidth(2)
    wav_file.setframerate(24000)
    wav_file.writeframes(full_audio)

print("✅ Saved to streaming_output.wav")
# Expected output:
# ......
# ✅ Streaming complete
# ✅ Saved to streaming_output.wav
```

## Common Errors and How to Fix Them

### Error 1: `INVALID_ARGUMENT: voice_name not supported`

```
google.api_core.exceptions.InvalidArgument: 400 INVALID_ARGUMENT: voice_name not supported
```

**Cause**: The specified voice name isn't available for the selected model.
**Fix**: Check the official Gemini API documentation for the current list of supported voice names.

---

### Error 2: Missing `response_modalities`

```
AttributeError: 'NoneType' object has no attribute 'data'
```

**Cause**: Omitting `response_modalities=["AUDIO"]` causes the response to contain no audio data.
**Fix**: Always include `response_modalities=["AUDIO"]` in `GenerateContentConfig`.

---

### Error 3: Using the Wrong Model Name

**Cause**: Confusing `gemini-2.5-flash` (standard model) with `gemini-2.5-flash-preview-tts` (TTS model).
**Fix**: Always use the `-tts` suffixed model name for speech generation tasks.

## Flash vs Pro: Which Model Should You Choose?

| Comparison | Flash TTS | Pro TTS |
|------------|-----------|---------|
| **Latency** | Low (real-time friendly) | Higher |
| **Voice Quality** | High | Maximum |
| **Expressiveness** | Rich | Very Rich |
| **Best For** | Chatbots, live responses | Podcasts, premium content |
| **Cost** | Lower | Higher |

**Bottom line**: Use Flash for prototyping and real-time use cases; use Pro for final, high-quality deliverables.

## FAQ

**Q1: Does Gemini TTS support languages other than English?**

A: Yes, Gemini TTS supports multiple languages including Japanese, Spanish, French, German, and more. Language quality varies by voice; check the official documentation for language-specific voice recommendations and test a few options to find the best fit for your target language.

**Q2: Is there a limit on how much text I can synthesize in one request?**

A: Yes. Each TTS API call has an input token limit (typically a few thousand tokens depending on the model). For longer narration, split your text into logical sections and make multiple API calls, then concatenate the resulting audio files using a library like `pydub`.

**Q3: What audio format does the API return?**

A: Gemini TTS returns raw **PCM audio data** — 24kHz sample rate, 16-bit, mono channel. You can wrap it in a WAV container using Python's built-in `wave` module, or convert it to MP3 or other formats using `pydub` or `ffmpeg`.
