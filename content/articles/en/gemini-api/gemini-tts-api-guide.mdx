## Introduction

Adding text-to-speech functionality to your application significantly improves accessibility and user engagement. Google's Gemini TTS API leverages cutting-edge voice synthesis technology to generate natural, expressive audio in multiple languages.

This guide walks you through everything you need to know about Gemini TTS API, from basic setup to advanced implementation patterns.

## What is Gemini TTS API?

The Gemini TTS API converts text into speech files in MP3 or other formats. Key features include:

- **Natural voice output**: State-of-the-art synthesis models produce human-like pronunciation
- **Multiple voice styles**: Choose from different tones, genders, and regional accents
- **Streaming support**: Real-time voice generation for interactive applications
- **Multilingual**: Supports Japanese, English, and many other languages with native-level quality

## Basic Implementation

### Simple Python Example

```python
import google.generativeai as genai
import os

# Configure API
genai.configure(api_key=os.environ.get("GOOGLE_API_KEY"))

# Convert text to speech
text = "Welcome to the Gemini TTS API tutorial."

# Use TTS API
response = genai.generate_speech(
    model="models/gemini-1.5-flash",
    text=text,
    voice="en-US-Neural2-C"  # Female English voice
)

# Save audio file
with open("output.mp3", "wb") as f:
    f.write(response.audio_data)

print("✅ Audio file created: output.mp3")
```

**Expected output:**
- `output.mp3` file containing natural English speech

### JavaScript Implementation

```javascript
const genai = require('@google/generative-ai');

const client = new genai.GoogleGenerativeAI(
  process.env.GOOGLE_API_KEY
);

async function textToSpeech() {
  const text = "Gemini TTS API is a powerful tool for speech generation.";
  
  try {
    const response = await client.generateSpeech({
      model: "models/gemini-1.5-flash",
      input: { text: text },
      voice: {
        languageCode: "en-US",
        name: "en-US-Neural2-C"
      },
      audioConfig: {
        audioEncoding: "MP3"
      }
    });

    const fs = require('fs');
    fs.writeFileSync('output.mp3', response.audioContent);
    console.log('✅ Audio file generated: output.mp3');
  } catch (error) {
    console.error('❌ Error:', error);
  }
}

textToSpeech();
```

**Expected output:**
- Playable MP3 file with natural speech synthesis

## Controlling Voice Styles

Gemini TTS API provides multiple voice parameters for fine-tuned speech synthesis:

**English voice options:**
- `en-US-Neural2-A` (Male, natural)
- `en-US-Neural2-C` (Female, natural)
- `en-US-Neural2-E` (Male, alternative)
- `en-US-Neural2-F` (Female, alternative)

**Japanese voice options:**
- `ja-JP-Neural2-A` (Male)
- `ja-JP-Neural2-C` (Female)
- `ja-JP-Neural2-D` (Female, higher pitch)

## Integration Patterns

### Pattern 1: Web Application Integration

```python
from flask import Flask, request, send_file
import google.generativeai as genai
import io

app = Flask(__name__)
genai.configure(api_key=os.environ.get("GOOGLE_API_KEY"))

@app.route('/generate-speech', methods=['POST'])
def generate_speech():
    data = request.json
    text = data.get('text', '')
    voice = data.get('voice', 'en-US-Neural2-C')
    
    try:
        response = genai.generate_speech(
            model="models/gemini-1.5-flash",
            text=text,
            voice=voice
        )
        
        audio_file = io.BytesIO(response.audio_data)
        audio_file.seek(0)
        
        return send_file(
            audio_file,
            mimetype='audio/mpeg',
            as_attachment=True,
            download_name='speech.mp3'
        )
    except Exception as e:
        return {'error': str(e)}, 400

if __name__ == '__main__':
    app.run(debug=True)
```

### Pattern 2: Batch Processing

```python
import time

texts = [
    "First text sample.",
    "Second text sample.",
    "Third text sample."
]

for i, text in enumerate(texts):
    try:
        response = genai.generate_speech(
            model="models/gemini-1.5-flash",
            text=text,
            voice="en-US-Neural2-C"
        )
        
        filename = f"batch_{i+1:03d}.mp3"
        with open(filename, "wb") as f:
            f.write(response.audio_data)
        
        print(f"✅ Created: {filename}")
        time.sleep(1)  # Rate limiting
    except Exception as e:
        print(f"❌ Error for text {i+1}: {e}")
```

## Common Errors and Troubleshooting

### Error 1: Invalid API Key

```
Error: 403 Forbidden - Invalid API key
```

**Solution:**
- Verify your API key is correctly set
- Regenerate a new key in Google AI Studio
- Confirm environment variables are loaded properly

### Error 2: Text Length Exceeded

```
Error: 400 Bad Request - Input text exceeds maximum length
```

**Solution:**
- Split text into multiple API calls
- Maximum text length is typically around 5,000 characters

### Error 3: Unsupported Audio Format

```
Error: 400 Bad Request - Audio encoding not supported
```

**Solution:**
- Use MP3 format (most reliable)
- Convert output using ffmpeg if needed

## FAQ

### Q1: How much does Gemini TTS API cost?

Gemini TTS API is currently free through Google AI Studio (experimental phase) and operates on pay-as-you-go pricing through Vertex AI. Check the official documentation for current rates.

### Q2: How can I store and reuse generated audio?

You can save generated audio files locally. For commercial use, review Google's terms of service regarding audio rights and usage restrictions.

### Q3: Can I use Gemini TTS API offline?

No, the API requires an internet connection. However, once generated, audio files can be played offline.

### Q4: Is real-time generation possible?

API response times are typically 1-3 seconds, which works well for chatbots and voice assistants but may not suit live streaming applications.
