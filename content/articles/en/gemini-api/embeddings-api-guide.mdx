## What Are Embeddings?

**Embeddings** convert text into numerical vectors. Text with similar meaning ends up close together in vector space, enabling "semantic similarity" searches that keyword matching can't handle.

The Gemini Embeddings API converts text to high-quality vectors for use in your applications.

## Use Cases

- **Semantic search**: Knowledge bases that search by meaning
- **Document similarity**: Duplicate detection, related content recommendations
- **RAG (Retrieval Augmented Generation)**: Search docs first, then pass context to an LLM
- **Text classification**: Classification with minimal training data
- **Clustering**: Grouping large volumes of feedback or comments

## Basic Usage

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

result = genai.embed_content(
    model="models/text-embedding-004",
    content="What is machine learning?",
    task_type="retrieval_query"
)

embedding = result["embedding"]
print(f"Vector dimension: {len(embedding)}")  # 768 dimensions
```

## task_type Options

Setting the right `task_type` improves embedding quality:

| task_type | Use case |
|-----------|----------|
| `retrieval_query` | Search queries / questions |
| `retrieval_document` | Documents to be searched |
| `semantic_similarity` | Sentence similarity scoring |
| `classification` | Text classification |
| `clustering` | Text grouping |

## Semantic Search with Cosine Similarity

```python
import numpy as np

def cosine_similarity(a, b):
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

documents = [
    "Gemini 3 Flash is Google's latest AI model",
    "Python is a popular programming language",
    "Machine learning trains models with large amounts of data",
    "You can try Gemini API for free in Google AI Studio",
]

doc_embeddings = [
    genai.embed_content(
        model="models/text-embedding-004",
        content=doc,
        task_type="retrieval_document"
    )["embedding"]
    for doc in documents
]

# Search with a query
query = "What is Google's newest AI?"
query_emb = genai.embed_content(
    model="models/text-embedding-004",
    content=query,
    task_type="retrieval_query"
)["embedding"]

scores = [(cosine_similarity(query_emb, emb), doc)
          for emb, doc in zip(doc_embeddings, documents)]
scores.sort(reverse=True)

for score, doc in scores[:3]:
    print(f"{score:.3f}: {doc}")
```

## Simple RAG Implementation

```python
def rag_search(query, docs, doc_embeddings, model_name="gemini-3-flash-preview"):
    # 1. Embed the query
    query_emb = genai.embed_content(
        model="models/text-embedding-004",
        content=query,
        task_type="retrieval_query"
    )["embedding"]

    # 2. Find the most relevant documents
    scores = [(cosine_similarity(query_emb, emb), doc)
              for emb, doc in zip(doc_embeddings, docs)]
    scores.sort(reverse=True)
    top_docs = [doc for _, doc in scores[:3]]

    # 3. Pass retrieved docs as context to the LLM
    context = "\n\n".join(top_docs)
    prompt = f"""Answer the question using the provided context.

Context:
{context}

Question: {query}"""

    model = genai.GenerativeModel(model_name)
    response = model.generate_content(prompt)
    return response.text
```

## Summary

The Gemini Embeddings API makes it straightforward to build semantically-aware applications.

- `text-embedding-004` delivers high-quality 768-dimension vectors
- Setting `task_type` correctly improves accuracy
- Cosine similarity enables fast nearest-neighbor search
- RAG patterns augment LLM knowledge with up-to-date retrieved content

Pair it with a vector database (Pinecone, Weaviate, pgvector) for production-scale search systems.
