## Introduction

The Gemini API includes a **File API** that lets you upload files separately from your prompt. This is especially useful when working with large files or when you want to reuse media across multiple requests.

In this guide, we'll walk through the Gemini File API from the basics to practical use cases, complete with Python code examples.

---

## Prerequisites

- Python 3.9 or higher
- A Gemini API key (available from [Google AI Studio](https://aistudio.google.com/))
- The `google-genai` package installed

```bash
pip install google-genai
```

## What Is the File API?

The File API is a **media file management service** built into the Gemini API. Its key characteristics include:

- **Free to use** (no additional cost for the File API itself)
- **48-hour storage**: uploaded files are retained for 48 hours
- **Max file size**: up to 2 GB per file (use the File API when the total request exceeds 100 MB)
- **Supported formats**: images, PDFs, videos, audio, text, and more

### When Should You Use the File API?

If the total request size (files + text prompt + system instructions) **exceeds 100 MB**, using the File API is required. Below that threshold you can pass files inline as Base64-encoded data, but the File API improves reusability across requests.

## Basic Usage

### 1. Initialize the Client

```python
import os
from google import genai

client = genai.Client(api_key=os.environ.get("GEMINI_API_KEY"))
```

### 2. Upload a File

```python
# Upload an image file
myfile = client.files.upload(file="./photo.jpg")
print(f"Upload complete: {myfile.uri}")
```

On success, `myfile.uri` contains the unique URI for your file.

### 3. Generate Content Using the Uploaded File

```python
# Use the uploaded file in a generation request
response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[myfile, "Please describe what you see in this image in detail."]
)
print(response.text)
```

## Working with PDFs

PDFs are one of the most common use cases. You can process PDFs up to **50 MB** in size.

```python
# Upload a PDF file
pdf_file = client.files.upload(
    file="./report.pdf",
    config={"mime_type": "application/pdf"}
)

# Ask questions about the PDF content
response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[
        pdf_file,
        "Summarize the three key points from this PDF."
    ]
)
print(response.text)
```

## Working with Videos

Video processing takes time, so you need to **wait until the file enters the ACTIVE state** before using it.

```python
import time

# Upload a video file
video_file = client.files.upload(file="./presentation.mp4")

# Wait for processing to complete
while video_file.state.name == "PROCESSING":
    print("Processing video...")
    time.sleep(5)
    video_file = client.files.get(name=video_file.name)

if video_file.state.name == "FAILED":
    raise ValueError("Video processing failed")

print(f"Processing complete: {video_file.uri}")

# Analyze the video content
response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[
        video_file,
        "Please summarize the content of this video."
    ]
)
print(response.text)
```

## Combining Multiple Files

You can also combine multiple files with text in a single request.

```python
# Upload multiple images
image1 = client.files.upload(file="./before.jpg")
image2 = client.files.upload(file="./after.jpg")

# Compare the two images
response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[
        "Compare these two images and list the differences.\n\nImage 1 (before):",
        image1,
        "\n\nImage 2 (after):",
        image2
    ]
)
print(response.text)
```

## Managing Files

### List All Uploaded Files

```python
# List all files
for file in client.files.list():
    print(f"Name: {file.name}, URI: {file.uri}, State: {file.state}")
```

### Get File Details

```python
# Retrieve details for a specific file
file_info = client.files.get(name="files/xxxxxx")
print(f"Created: {file_info.create_time}")
print(f"Expires: {file_info.expiration_time}")
```

### Delete a File

```python
# Manually delete a file before the 48-hour expiry
client.files.delete(name=myfile.name)
print("File deleted successfully")
```

## Frequently Asked Questions

**Q: Is the File API free?**
A: Yes, uploading and managing files with the File API is free. However, standard Gemini API pricing applies when you use uploaded files in content generation requests.

**Q: How long are files stored?**
A: Files are automatically deleted **48 hours** after upload. You can delete them earlier using `client.files.delete()`.

**Q: What file formats are supported?**
A: Images (JPEG, PNG, WebP, HEIC, HEIF), video (MP4, MOV, AVI, etc.), audio (MP3, WAV, FLAC, etc.), PDF, and text formats (TXT, HTML, CSS, Markdown, etc.).

**Q: Can I use a file's URI from a different project?**
A: No. File URIs are tied to the project associated with the API key used for the upload. Keep your API key secure to prevent unauthorized access to your uploaded files.
