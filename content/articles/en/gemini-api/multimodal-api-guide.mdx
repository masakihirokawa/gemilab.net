## What Is Multimodal?

One of Gemini's greatest strengths is its ability to process multiple input types simultaneously — images, video, audio, PDFs, and more. This article covers how to implement multimodal processing with the Gemini API.

## Supported Media Types

| Media | Formats | Max Size |
|---|---|---|
| Images | PNG, JPEG, WEBP, GIF | 20MB |
| Video | MP4, MPEG, MOV, AVI | 2GB |
| Audio | MP3, WAV, AIFF, FLAC | ~9.5 hours |
| Documents | PDF | Up to 300 pages |

## Image Examples

### Sending with Base64 Encoding

```python
import google.genai as genai
import base64

client = genai.Client(api_key="YOUR_API_KEY")

with open("photo.jpg", "rb") as f:
    image_data = base64.b64encode(f.read()).decode("utf-8")

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[
        {"text": "Describe what you see in this image"},
        {
            "inline_data": {
                "mime_type": "image/jpeg",
                "data": image_data
            }
        }
    ]
)

print(response.text)
```

### Direct URL Reference

Images stored in Google Cloud Storage can be referenced directly by URI.

```python
response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[
        {"text": "Analyze this image"},
        {
            "file_data": {
                "mime_type": "image/jpeg",
                "file_uri": "gs://your-bucket/image.jpg"
            }
        }
    ]
)
```

## Video Analysis

Gemini can understand video content and provide summaries or answer questions about it.

```python
# Upload video using File API
video_file = client.files.upload(
    file="presentation.mp4",
    config={"display_name": "My Presentation"}
)

# Wait for processing to complete
import time
while video_file.state.name == "PROCESSING":
    time.sleep(5)
    video_file = client.files.get(name=video_file.name)

# Ask about the video
response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[
        {"text": "List the 5 key points from this video"},
        {"file_data": {"file_uri": video_file.uri, "mime_type": "video/mp4"}}
    ]
)
```

## Audio Processing

Transcription and summarization of audio files is also supported.

```python

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[
        {"text": "Transcribe this meeting audio and list the action items"},
        {"file_data": {"file_uri": audio_file.uri, "mime_type": "audio/mp3"}}
    ]
)
```

## PDF Analysis

Upload PDF documents for content summarization and Q&A.

```python

response = client.models.generate_content(

        {"text": "Summarize the key findings and recommendations in this report"},
        {"file_data": {"file_uri": pdf_file.uri, "mime_type": "application/pdf"}}
    ]

```

Gemini truly shines when processing multiple media types simultaneously.


## Best Practices

- **Model selection**: Use `gemini-2.5-pro` for high-accuracy analysis, `gemini-2.5-flash` for speed
- **File size**: Use the File API for large files, inline_data for small ones
- **Prompt design**: Specific instructions dramatically improve accuracy even with multimodal input
- **Cost management**: Video and audio consume more tokens — trim to essential segments

## Summary

Gemini's multimodal API enables use cases that were impossible with text alone. From image recognition and video summarization to audio transcription, tasks that once required specialized tools can now be handled through a unified API.
