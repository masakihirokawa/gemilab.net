## Google AI Studio Is Now a Development Environment

On March 20, 2026, Google added **full-stack vibe coding** to Google AI Studio. What was once a prompt playground has evolved into a complete development environment where you can build, preview, and deploy apps — all from natural language prompts.

We've already covered [the announcement and overview](/en/articles/gemini-dev/google-ai-studio-fullstack-vibe-coding) in a previous article. This one is a **hands-on walkthrough**. By the end, you'll have generated a working app with Firebase backend integration using nothing but prompts.

## Accessing Build Mode

### Getting Started

Navigate to [aistudio.google.com](https://aistudio.google.com) and select "**Build**" from the left sidebar. Gemini (2.5 Pro recommended) is auto-selected as the model, and you'll see a prompt input area alongside a live preview panel.

### Your First Generated App

Type your app requirements in natural language. Gemini interprets them and generates React (or Angular/Next.js) code automatically.

```
Example prompt:
"Build a reading tracker app. Users can add books with title, author,
and completion date. Show a bar chart of books finished per month.
Use a clean, minimal design with muted colors."
```

Within seconds, the preview panel shows a working app with UI components, state management, and responsive layout already in place.

### Reviewing and Editing Code

Click the "**Code**" tab below the preview to inspect the generated source. You can edit it manually, or use follow-up prompts like "change the button color to blue" for iterative refinement.

## Firebase Auto-Integration — Databases and Auth from Prompts

### How Antigravity Detects Backend Needs

Build Mode's standout feature is the **Antigravity coding agent**, which analyzes your app requirements and automatically suggests Firebase integration when a backend is needed.

Add this to the reading tracker prompt:

```
"Save data per user. Let users sign in with their Google account."
```

The Antigravity agent detects the need for:

1. **Cloud Firestore** — to persist reading data
2. **Firebase Authentication** — for Google sign-in

Click "Approve Firebase integration" and the following happens automatically:

```javascript
// Auto-generated Firebase configuration (reference)
import { initializeApp } from "firebase/app";
import { getFirestore, collection, addDoc, getDocs } from "firebase/firestore";
import { getAuth, signInWithPopup, GoogleAuthProvider } from "firebase/auth";

const firebaseConfig = {
  // AI Studio creates the project and fills in config values
  apiKey: "auto-configured",
  authDomain: "auto-configured",
  projectId: "auto-configured",
};

const app = initializeApp(firebaseConfig);
const db = getFirestore(app);
const auth = getAuth(app);

// Function to add a book
export async function addBook(userId, book) {
  const docRef = await addDoc(collection(db, `users/${userId}/books`), {
    title: book.title,
    author: book.author,
    finishedDate: book.finishedDate,
    createdAt: new Date(),
  });
  return docRef.id;
}

// Expected behavior:
// addBook("user123", { title: "Deep Learning", author: "Ian Goodfellow", finishedDate: "2026-03-15" })
// → Adds a document to the users/user123/books collection in Firestore
```

### What Happens Behind the Scenes

When you approve Firebase integration, the system automatically:

- Creates a new Firebase project (or lets you select an existing one)
- Enables Cloud Firestore with initial security rules
- Enables Firebase Authentication with Google provider
- Integrates Firebase SDK into the app code

Steps that used to require manual Firebase Console configuration are now fully automated.

## Hands-On: Build a Task Manager in 30 Minutes

Let's walk through building a practical app step by step.

### Step 1: Describe Your Requirements

```
"Build a team task management app.
- Manage tasks with title, assignee, deadline, and status (todo/in-progress/done)
- Display as a Kanban board
- Drag and drop to change status
- Real-time collaboration for multiple users
- Google sign-in"
```

### Step 2: Approve Firebase Integration

The Antigravity agent suggests Firestore (real-time database) and Authentication (Google sign-in). Approve it.

### Step 3: Refine Iteratively

```
"Show task count in each Kanban column header"
"Highlight overdue tasks in red"
"Add a dropdown filter for assignees"
```

### Step 4: Deploy

Hit the "Deploy" button. Your app is deployed to Google Antigravity and you receive a URL. Share it with your team to start collaborating immediately.

## Build Mode vs. Direct Gemini API Usage

Build Mode is for interactive app assembly. The [Gemini API called from code](/en/articles/gemini-api/gemini-api-quickstart) is better when you want to add AI features to an existing project.

| Aspect | Build Mode | Direct Gemini API |
|--------|-----------|-------------------|
| Audience | Prototyping, low-code approach | AI integration in existing projects |
| Output | Complete application | API responses (text, images, etc.) |
| Customization | Prompts + code editing | Full code control |
| Firebase | Auto-provisioning | Manual setup |
| Deployment | One-click (Antigravity) | Your own infrastructure |

## Frequently Asked Questions

### Is Build Mode free to use?

It's available within Google AI Studio's free tier. However, Firebase integration uses Firebase's free Spark plan. If your app generates heavy Firestore reads/writes or authentication requests, you may need to upgrade to the Blaze plan (pay-as-you-go).

### Can I download the generated code locally?

Yes. Export the full source code from the "Code" tab. Run `npm install && npm run dev` locally to continue development. Apps deployed to Antigravity can also be cloned as Git repositories.

### Does it support frameworks besides React?

Currently React, Angular, and Next.js are supported. Specify "build this with Angular" in your prompt to use a different framework. Vue.js and Svelte support is planned for future updates.

### Can I connect to an existing Firebase project?

Yes. During the Firebase approval step, choose "Select existing project" to link with a Firebase project you've already set up. Your existing Firestore data and Authentication configuration will be available to the generated app.

Google AI Studio's Build Mode combines Gemini's language understanding, Antigravity's coding agent, and Firebase's backend infrastructure into a unified full-stack app generation environment.
