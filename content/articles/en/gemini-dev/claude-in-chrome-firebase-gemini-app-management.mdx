## Best Practices

### 1. Authentication State

Ensure Claude in Chrome has logged in beforehand. Never include passwords in prompts.

### 2. Data Granularity Balance

Adjust extraction frequency by priority:

| Frequency | Scope |
|-----------|-------|
| Daily | Analytics summary, Crashlytics, AdMob |
| Weekly | Comprehensive analysis |
| Monthly | Detailed breakdown |

### 3. Error Handling

Enable Cowork notifications to catch failures immediately.

### 4. Data Security

Confirm sensitive data exclusion:

✓ Safe: Analytics aggregates, crash summaries, revenue totals
✗ Unsafe: User PII, individual Firestore documents, API keys

## FAQ

**Q: Can Claude in Chrome manage multiple Firebase projects?**

A: Yes, but create separate tasks per project to avoid confusion. Sequential execution prevents timeout issues.

**Q: How do I integrate Gemini analysis?**

A: After data collection, send extracted metrics to Gemini API for interpretation and recommendation generation.

**Q: What about real-time critical monitoring?**

A: Firebase Cloud Functions + Pub/Sub + Cloud Monitoring better suit real-time alerts. Claude in Chrome excels at scheduled summaries.
