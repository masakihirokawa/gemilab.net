## Complete Your Development Within the Google AI Ecosystem

Stitch and Gemini are both AI tools from Google. Combining them creates a seamless development flow where **front-end design generation (Stitch)** and **back-end logic implementation (Gemini API)** both happen inside the Google AI ecosystem.

Minimize third-party dependencies while letting AI lead your full-stack development — that's what Stitch × Gemini makes possible.

---

## What is Google Stitch?

**Google Stitch** ([stitch.withgoogle.com](https://stitch.withgoogle.com)) is a **generative AI-powered UI design tool** from Google.

Key features:
- Generate Material Design 3-compliant UI from natural language prompts
- Export as React or HTML/CSS
- Generate and customize at the component level
- Fully integrated with Google's design language

---

## The Basic Stitch × Gemini Workflow

### Overall Flow

```
[Stitch] → Generate UI components (React / HTML)
    ↓
[Gemini API] → Optimize code and generate backend integration code
    ↓
[Firebase / Vertex AI] → Deploy and add AI features
```

## Connecting Stitch via MCP

Adding the Stitch server to Gemini CLI or Claude Code's MCP lets you generate UIs seamlessly within an AI conversation.

### MCP Configuration (mcp_config.json)

```json
{
  "mcpServers": {
    "stitch": {
      "command": "npx",
      "args": ["-y", "@google/stitch-mcp-server@latest"],
      "env": {
        "GOOGLE_API_KEY": "YOUR_GOOGLE_API_KEY"
      }
    }
  }
}
```

### Usage After Configuration (Gemini CLI)

```bash
# Use Stitch MCP with Gemini CLI
gemini "Generate a dashboard screen with Stitch, then convert it to a
React component that integrates with Firebase Firestore"
```

## Python Integration: Stitch × Gemini

### Generate Components with Stitch API, then Extend with Gemini

```python
import google.generativeai as genai
import requests
import json

genai.configure(api_key="YOUR_GEMINI_API_KEY")
model = genai.GenerativeModel("gemini-2.0-flash")

def generate_ui_with_stitch(description: str, components: list[str]) -> dict:
    """
    Generate UI components using the Google Stitch API
    (Adjust to match the official API endpoint)
    """
    stitch_payload = {
        "description": description,
        "components": components,
        "style": "material-design-3",
        "platform": "react",
        "dark_mode": True
    }
    return stitch_payload  # Simplified

def enhance_stitch_with_gemini(stitch_output: str, requirements: str) -> str:
    """
    Further extend Stitch-generated UI with the Gemini API
    """
    prompt = f"""
Add the specified features to the following Google Stitch-generated React code:

Stitch-generated code:
{stitch_output}

Additional requirements:
{requirements}

Output format:
1. Extended React component (TypeScript)
2. Required type definitions
3. Mock data for testing
"""
    response = model.generate_content(prompt)
    return response.text

# Example: Generate and extend a dashboard UI
ui_description = """
SaaS app dashboard:
- Monthly revenue chart (line graph)
- User count card (month-over-month)
- Latest activity feed
- Material Design 3, dark/light mode
"""

enhancements = """
- Fetch real-time data from Firebase Firestore
- Render charts with recharts
- Cache management with React Query
- Loading skeleton animation
"""

stitch_result = generate_ui_with_stitch(ui_description, ["Chart", "Card", "Feed"])
enhanced_code = enhance_stitch_with_gemini(str(stitch_result), enhancements)
print(enhanced_code)
```

**2. Automate contact form responses with Gemini API**

```python
def create_ai_contact_form(form_data: dict) -> dict:
    """
    Receive a contact form submission and generate an
    automatic reply using the Gemini API
    """
    prompt = f"""
Write a polite, professional reply email for the following inquiry:

Sender name: {form_data.get('name')}
Email: {form_data.get('email')}
Inquiry: {form_data.get('message')}

Reply should include:
- Thank the sender for reaching out
- Specific response to the inquiry or next steps
- Note that a representative will follow up within 24 hours
- Company: Dolice Labs
"""
    response = model.generate_content(prompt)

    return {
        "auto_reply": response.text,
        "original_query": form_data,
        "status": "processed"
    }
```

**3. Integrate with Stitch code**

```python
def integrate_ai_with_stitch_form(stitch_form_code: str) -> str:
    """
    Add Gemini AI integration to a Stitch-generated form
    """
    prompt = f"""
Add Gemini API integration to the following Stitch-generated form code:

Form code:
{stitch_form_code}

Features to add:
1. POST to /api/contact on form submission
2. Loading animation while submitting (match Stitch style)
3. Success message on completion (display Gemini's auto-reply)
4. Validation (email format, required fields)
5. Error handling (network errors etc.)

Implement with TypeScript + React Hook Form.
"""
    response = model.generate_content(prompt)
    return response.text
```

## Triple Integration: Stitch × Gemini × Vertex AI

Take it further by adding Vertex AI for an enterprise-grade three-layer architecture.

```python
def stitch_gemini_vertex_pipeline(
    ui_description: str,
    business_logic: str,
    model_type: str = "gemini-2.0-flash"
) -> dict:
    """
    Integrated pipeline: Stitch (UI) × Gemini API (logic) × Vertex AI (scale)
    """

    # 1. Generate UI with Stitch
    ui_code = f"# Stitch generated UI for: {ui_description}"

    # 2. Implement business logic with Gemini API
    gemini_prompt = f"""
UI spec: {ui_description}
Business logic requirements: {business_logic}

Implement:
- React component (Stitch style compliant)
- API client (Vertex AI endpoint calls)
- Error handling
- TypeScript type definitions
"""
    gemini_model = genai.GenerativeModel(model_type)
    logic_code = gemini_model.generate_content(gemini_prompt).text

    # 3. Configure scalable backend with Vertex AI
    vertex_config = {
        "endpoint": "us-central1-aiplatform.googleapis.com",
        "model": "gemini-2.0-flash-001",
        "features": ["streaming", "function_calling", "grounding"]
    }

    return {
        "ui": ui_code,
        "logic": logic_code,
        "vertex_config": vertex_config,
        "deployment_target": "Cloud Run + Firebase Hosting"
    }
```
