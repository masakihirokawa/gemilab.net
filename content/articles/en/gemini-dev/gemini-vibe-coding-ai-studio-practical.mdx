## Introduction

**Vibe Coding** is a modern paradigm where you guide AI through intuitive instructions rather than rigid specifications. By combining Google AI Studio with Gemini 2.5, you reduce development time by 60–70%.

## What Is Vibe Coding?

Traditional: 50-100 page specs, 2-4 weeks
Vibe Coding: 5-10 lines of natural language, 2-4 days

### Why Gemini + AI Studio?

- **Multimodal**: Screenshot → code generation
- **Context-aware**: Long conversation history
- **Fast iteration**: Seconds to prototype changes

## Setting Up Google AI Studio

1. Open https://aistudio.google.com
2. Start a new chat
3. Configure system prompt

```
You are a 10-year full-stack developer.
Your task is to understand user requests in natural language
and generate production-ready code (HTML/CSS/JavaScript/Python).
```

## Full-Stack Application Flow

### Step 1: Natural Language Instruction

```markdown
Create a task management app with:
- Add, delete, complete task functionality
- Local storage persistence
- Modern UI with gradients
```

### Step 2: Gemini Generates Complete Code

Gemini produces production-ready HTML/CSS/JavaScript in seconds.
