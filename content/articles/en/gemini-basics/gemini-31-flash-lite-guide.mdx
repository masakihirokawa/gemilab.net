## What Is Gemini 3.1 Flash Lite?

Gemini 3.1 Flash Lite is Google's lightest-weight model, positioned as the most cost-effective option in the Gemini lineup. It's available as a developer preview through the Gemini API on Google AI Studio, and also offered via Vertex AI for enterprise use cases.

### Key Features

- **Ultra-low cost**: $0.25/1M input tokens, $1.50/1M output tokens — among the most affordable in the industry
- **Super-fast response**: Time to first token (TTFT) is **2.5x faster** than Gemini 2.5 Flash
- **High throughput**: Output speed is **45% higher** compared to Gemini 2.5 Flash
- **Multimodal support**: Handles text, images, and video inputs



## Cost Optimization Tips

### 1. Route Tasks to the Right Model

Not every request needs the most powerful model. Routing tasks based on complexity can dramatically cut costs:

```python
def get_appropriate_model(task_type: str) -> str:
    """Return the right model based on task complexity."""
    simple_tasks = ["classify", "summarize", "translate", "tag"]
    complex_tasks = ["code generation", "reasoning", "analysis report"]

    if any(task in task_type.lower() for task in simple_tasks):
        return "gemini-3.1-flash-lite"  # Cost-focused
    elif any(task in task_type.lower() for task in complex_tasks):
        return "gemini-3.1-pro"  # Accuracy-focused
    else:
        return "gemini-2.5-flash"  # Balanced
```

### 2. Optimize Your Prompts

Concise, clear prompts reduce token count and lower costs:

```python
# Verbose prompt (not recommended)
prompt_bad = "You are a highly capable AI. Please summarize the following text as thoroughly and helpfully as possible: " + text

# Concise prompt (recommended)
prompt_good = f"Summarize in 100 words:\n{text}"
```

### 3. Use Batch Processing

```python
import asyncio
from google import genai

client = genai.Client(api_key=os.getenv("GEMINI_API_KEY"))

async def batch_process(texts: list[str]) -> list[str]:
    """Process multiple texts in parallel."""
    tasks = []
    for text in texts:
        task = asyncio.create_task(
            asyncio.to_thread(
                lambda t=text: client.models.generate_content(
                    model="gemini-3.1-flash-lite",
                    contents=f"Summarize: {t}"
                ).text
            )
        )
        tasks.append(task)

    return await asyncio.gather(*tasks)

# Process multiple articles at once
texts = ["Article 1 content...", "Article 2 content...", "Article 3 content..."]
results = asyncio.run(batch_process(texts))
```
