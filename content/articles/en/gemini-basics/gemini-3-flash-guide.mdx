## How to Use Gemini 3 Flash

### Try It in Google AI Studio

The quickest way to get started is [Google AI Studio](https://aistudio.google.com/). Select `gemini-3-flash-preview` from the model selector and start prompting — no API key required for testing.

### Gemini API (Python)

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-3-flash-preview")

response = model.generate_content(
    "Write a Python function to generate the Fibonacci sequence."
)
print(response.text)
```

### Setting Reasoning Levels

Gemini 3 Flash lets you control thinking depth across four levels. Use higher reasoning for complex tasks and lower for latency-sensitive applications.

```python
from google.generativeai.types import GenerationConfig

config = GenerationConfig(
    thinking_config={"thinking_budget": "high"}  # minimal / low / medium / high
)

response = model.generate_content(
    "Solve the following complex math problem: ...",
    generation_config=config
)
print(response.text)
```

### Multimodal Input (Images)

```python
import PIL.Image

image = PIL.Image.open("screenshot.png")

response = model.generate_content(
    ["Identify the bug in this screenshot and suggest a fix.", image]
)
print(response.text)
```
