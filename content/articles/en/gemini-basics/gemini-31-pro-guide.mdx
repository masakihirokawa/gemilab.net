## The Headline Feature: Three-Tier Thinking System

The most important addition in Gemini 3.1 Pro is the **three-tier thinking budget system**. Previous Gemini models offered only a binary choice between fast (low compute) and deep (high compute) modes. Gemini 3.1 Pro introduces a **Medium** tier, allowing developers to strike a mathematically balanced trade-off between response latency and reasoning depth.

### Choosing the Right Thinking Mode

| Mode | Characteristics | Best For |
|------|----------------|----------|
| **Low** | Fast, low cost | Q&A, text generation, simple tasks |
| **Medium** | Balanced | Summarization, code review, standard reasoning |
| **High** | Deep, precise | Math proofs, complex agent workflows |

## Getting Started: Calling the API

### Basic Python Example

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

# Specify Gemini 3.1 Pro Preview
model = genai.GenerativeModel("gemini-3.1-pro-preview")

response = model.generate_content(
    "Explain the basics of quantum computing for a beginner.",
    generation_config=genai.GenerationConfig(
        temperature=0.7,
        max_output_tokens=2048,
    )
)

print(response.text)
```

### Specifying the Thinking Budget

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel(
    "gemini-3.1-pro-preview",
    generation_config={
        "thinking_config": {
            "thinking_budget": "medium"  # "low" / "medium" / "high"
        }
    }
)

response = model.generate_content(
    "Analyze the pattern in this sequence and predict the next 5 values: 1, 1, 2, 3, 5, 8, 13, 21"
)

print(response.text)
```

### REST API Call

```bash
curl -X POST \
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-3.1-pro-preview:generateContent?key=YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "parts": [{"text": "What are the best practices for implementing async processing in Python?"}]
    }],
    "generationConfig": {
      "thinkingConfig": {
        "thinkingBudget": "high"
      }
    }
  }'
```

## Reasoning Breakthrough: 77.1% on ARC-AGI-2

Gemini 3.1 Pro achieved **77.1%** on **ARC-AGI-2**, a rigorous benchmark designed to test genuine general reasoning — patterns a model has never seen before. This is more than double the score of Gemini 3 Pro and represents a top-tier result at the time of release.

ARC-AGI-2 is considered especially hard because it demands novel pattern recognition and logic that can't be memorized — making this score a meaningful signal that AI is approaching more flexible, human-like reasoning.

## Multimodal Capabilities in Practice

Gemini 3.1 Pro handles text, images, audio, video, PDFs, and full code repositories through its unified 1M-token context window.

### Analyzing an Image

```python
import google.generativeai as genai
import PIL.Image

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-3.1-pro-preview")

# Load a local image
image = PIL.Image.open("architecture_diagram.png")

response = model.generate_content([
    "Analyze this diagram in detail and suggest any improvements.",
    image
])

print(response.text)
```

### Parsing a PDF Document

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

# Upload the PDF
uploaded_file = genai.upload_file("quarterly_report.pdf", mime_type="application/pdf")

model = genai.GenerativeModel("gemini-3.1-pro-preview")

response = model.generate_content([
    "Summarize the three key takeaways from this report and suggest actionable next steps.",
    uploaded_file
])

print(response.text)
```
