## Gemini's Video Understanding Capabilities

Gemini's 1M token context window makes it uniquely capable of analyzing long videos. One minute of video corresponds to roughly 100 frames, so even hour-long videos can be processed in a single request.

Key video analysis use cases:
- Video summarization and transcription
- Timeline and chapter generation
- Scene detection and extraction
- Security footage anomaly detection
- Meeting recording analysis

## Upload and Analyze a Video

Use the File API to upload videos before analysis:

```python
import google.generativeai as genai
import time

genai.configure(api_key="YOUR_API_KEY")

print("Uploading video...")
video_file = genai.upload_file(
    path="meeting_recording.mp4",
    mime_type="video/mp4"
)

# Wait for processing
while video_file.state.name == "PROCESSING":
    print(".", end="", flush=True)
    time.sleep(2)
    video_file = genai.get_file(video_file.name)

if video_file.state.name == "FAILED":
    raise ValueError("Video processing failed")

print(f"\nReady: {video_file.uri}")
```

## Generate a Summary

```python
model = genai.GenerativeModel("gemini-3-flash-preview")

response = model.generate_content([
    video_file,
    "Summarize this video. List the key points as bullet points."
])

print(response.text)
```

## Generate Chapters and Timeline

```python
response = model.generate_content([
    video_file,
    """Create a detailed chapter timeline for this video.
    For each section, return:
    - start_time (MM:SS format)
    - title
    - brief_description (under 100 chars)
    Return as JSON."""
])

import json
timeline = json.loads(response.text)
for chapter in timeline:
    print(f"{chapter['start_time']} - {chapter['title']}: {chapter['brief_description']}")
```

## Search for Specific Content

```python
response = model.generate_content([
    video_file,
    "Are there any moments in this video where code is shown on screen? If so, at approximately what timestamp?"
])
print(response.text)
```

## Transcribe Audio

```python
response = model.generate_content([
    video_file,
    "Transcribe the audio in this video, organized by speaker."
])
print(response.text)
```

## Analyze YouTube Videos Directly

Pass a YouTube URL directly — no file upload needed:

```python
response = model.generate_content([
    {
        "file_data": {
            "mime_type": "video/youtube",
            "file_uri": "https://www.youtube.com/watch?v=XXXXXXXXX"
        }
    },
    "Summarize the main topics and key points from this video."
])
print(response.text)
```

## Summary

Gemini's Video Understanding makes video analysis accessible with minimal code.

- Upload via File API → analyze with GenerativeModel
- Handles summaries, timelines, transcription, and scene search
- YouTube URLs can be passed directly
- Gemini 3 Flash is the most cost-effective choice for long videos

Practical applications include meeting transcription, seminar summarization, and video content indexing.
