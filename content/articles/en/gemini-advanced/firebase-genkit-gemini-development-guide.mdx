## Introduction

Firebase Genkit is Google's open-source framework for building AI applications, with first-class support for Gemini. Optimized for serverless deployment, it enables you to build complex AI workflows, agents, and RAG systems with minimal boilerplate.

This guide covers everything from initial setup to production deployment, with real-world patterns and optimization strategies.

## What is Firebase Genkit?

Firebase Genkit is a TypeScript/JavaScript framework for accelerated AI development:

**Key Features:**
- **Multi-model support**: Use Gemini, OpenAI, Claude, and other LLMs with a unified API
- **Flow definition**: Declare complex AI workflows declaratively
- **Agents**: Autonomous tool selection and execution
- **RAG integration**: Seamless vector search and knowledge base integration
- **Firebase integration**: Works with Realtime DB, Firestore, Cloud Functions
- **Developer experience**: Build in days what typically takes weeks

## Define Flows

A **Flow** combines multiple steps into an AI workflow, automatically deployable to Cloud Functions.

### Simple Flow Example

```typescript
// src/flows/greeting.ts
import { defineFlow, runFlow } from "@firebase/genkit";
import { gemini15Flash } from "@firebase/genkit-plugins";

export const greetingFlow = defineFlow(
  {
    name: "greeting",
    inputSchema: {
      type: "object",
      properties: {
        name: { type: "string" },
        language: { type: "string", enum: ["en", "ja"] },
      },
    },
  },
  async (input) => {
    const prompt = 
      input.language === "en" 
        ? `Create a friendly greeting for ${input.name}.`
        : `${input.name}さんへの親切な挨拶を日本語で作成してください。`;

    const response = await gemini15Flash.generateText({ prompt });
    return response.text();
  }
);

// Local test
if (require.main === module) {
  runFlow(greetingFlow, { name: "Alice", language: "en" });
}
```

**Run:**
```bash
npx ts-node src/flows/greeting.ts

# Output:
# Hello Alice! Hope you have a wonderful day!
```

### Complex Multi-Step Flow

```typescript
// src/flows/documentAnalysis.ts
import { defineFlow } from "@firebase/genkit";
import { gemini15Flash } from "@firebase/genkit-plugins";

interface DocumentInput {
  documentText: string;
  analysisType: "summary" | "sentiment" | "keywords";
}

export const documentAnalysisFlow = defineFlow(
  {
    name: "documentAnalysis",
    inputSchema: {
      type: "object",
      properties: {
        documentText: { type: "string" },
        analysisType: { 
          type: "string", 
          enum: ["summary", "sentiment", "keywords"] 
        },
      },
    },
  },
  async (input: DocumentInput) => {
    // Step 1: Clean text
    const cleanedText = input.documentText.trim().slice(0, 5000);

    // Step 2: Create analysis prompt
    const prompts = {
      summary: `Summarize this document in 3 sentences:\n${cleanedText}`,
      sentiment: `Analyze sentiment (negative/neutral/positive):\n${cleanedText}`,
      keywords: `Extract 5 key topics:\n${cleanedText}`,
    };

    // Step 3: Run analysis with Gemini
    const analysisPrompt = prompts[input.analysisType];
    const response = await gemini15Flash.generateText({ 
      prompt: analysisPrompt 
    });

    // Step 4: Return structured result
    return {
      analysisType: input.analysisType,
      result: response.text(),
      timestamp: new Date().toISOString(),
      tokenUsage: response.usage,
    };
  }
);
```

## Agent Implementation

### Agent Flow

```typescript
// src/flows/researchAgent.ts
import { defineFlow } from "@firebase/genkit";
import { gemini15Flash } from "@firebase/genkit-plugins";
import { searchWeb, getWeather } from "../tools/webTools";

export const researchAgentFlow = defineFlow(
  {
    name: "researchAgent",
    inputSchema: {
      type: "object",
      properties: {
        question: { type: "string" },
      },
    },
  },
  async (input) => {
    const tools = [searchWeb, getWeather];

    const agent = gemini15Flash.createAgent({
      tools: tools,
      systemPrompt: `
        You are a helpful research assistant.
        Use available tools to gather information and provide comprehensive answers.
      `,
    });

    const response = await agent.run({
      message: input.question,
    });

    return {
      question: input.question,
      answer: response.text(),
      toolsUsed: response.toolCalls,
    };
  }
);
```

```typescript
  {
    name: "documentRAG",
    inputSchema: {
      type: "object",
      properties: {
        query: { type: "string" },
        topK: { type: "number", default: 3 },
      },
    },
  },
  async (input) => {
    // Step 1: Vectorize query
    const queryEmbedding = await gemini15Flash.embedText(input.query);

    // Step 2: Semantic search for relevant documents
    const relevantDocs = await searchSimilarDocuments(
      queryEmbedding.embedding,
      input.topK

      ${context}

      Question: ${input.query}
    `;

    const response = await gemini15Flash.generateText({ 
      prompt: ragPrompt

      query: input.query,
```

```typescript
// src/index.ts

import { greetingFlow } from "./flows/greeting";
import { documentAnalysisFlow } from "./flows/documentAnalysis";
import { researchAgentFlow } from "./flows/researchAgent";
import { documentRAGFlow } from "./flows/documentRAG";

// Export to Cloud Functions
export const greeting = functions.https.onCall(async (data, context) => {
  return await greetingFlow(data);
});

  return await documentAnalysisFlow(data);
});

export const researchAgent = functions.https.onCall(async (data, context) => {
  return await researchAgentFlow(data);
});

export const documentRAG = functions.https.onCall(async (data, context) => {
  return await documentRAGFlow(data);
});
```

## Best Practices

### Error Handling

```typescript
export const robustFlow = defineFlow(
  { name: "robustFlow" },
  async (input) => {
    try {
      const response = await gemini15Flash.generateText({ 
        prompt: input.prompt 
      });
      return { success: true, result: response.text() };
    } catch (error) {
      if (error.code === "RATE_LIMIT_EXCEEDED") {
        await new Promise(resolve => setTimeout(resolve, 1000));
        return robustFlow(input);
      }
      return { success: false, error: error.message };
    }
  }
);
```

### Cost Optimization

```typescript
import { gemini15Pro, gemini15Flash } from "@firebase/genkit-plugins";

const efficientFlow = defineFlow(
  { name: "efficientFlow" },
  async (input) => {
    if (input.complexity === "low") {
      return await gemini15Flash.generateText({ prompt: input.prompt });
    }
    return await gemini15Pro.generateText({ prompt: input.prompt });
  }
);
```

### Caching

```typescript
export const cachedFlow = defineFlow(
  { name: "cachedFlow" },
  async (input) => {
    const cacheKey = `flow:${JSON.stringify(input)}`;
    
    const cached = await getFromCache(cacheKey);
    if (cached) return cached;

    const result = await gemini15Flash.generateText({ 
      prompt: input.prompt 
    });

    await saveToCache(cacheKey, result, 3600);
    return result;
  }
);
```

## FAQ

### Q1: Can I use multiple AI models in one Flow?

Yes. Configure multiple model plugins and use them conditionally within your Flow.

### Q2: Which languages does Genkit support?

Primarily TypeScript/JavaScript. Go and Python versions are in development.

### Q3: How does it scale with high traffic?

Cloud Functions auto-scales automatically based on demand.

### Q4: How is data consistency guaranteed?

Firestore transactions ensure atomic execution across multiple steps.
