Google Agent Development Kit (ADK) is a comprehensive platform that streamlines AI agent development and deployment. In this article, we explore how to combine ADK with the Gemini API to build production-grade multi-agent systems using TypeScript, with practical patterns and real-world implementation strategies.

## Understanding Google ADK

### The Three Core Layers of ADK

Google ADK is structured in three essential layers:

**Agent Definition Layer**

The foundational "Agent" class provides unified management of:

- System prompts (role definitions)
- Tools (external API and database integration)
- Memory (session context preservation)
- Inference engine (Gemini)

**Workflow Integration Layer**

Orchestrates collaboration between multiple agents:

- Sequential processing (Agent A → B → C)
- Parallel execution (concurrent agents)
- Conditional branching and routing
- Retry and fallback mechanisms

**Execution and Monitoring Layer**

Supports production operations through:

- Local testing environments
- Deployment to Cloudflare Workers / Cloud Run
- Observability (tracing, logging, metrics)
- Cost and rate limit management

## Fundamental Agent Design Patterns

### Type-Safe Agent Definition

```typescript
import { GoogleGenerativeAI } from "@google/generative-ai";
import { z } from "zod";

interface AgentConfig {
  name: string;
  role: string;
  systemPrompt: string;
  tools: Tool[];
  model: string;
}

class GeminiAgent {
  private client: GoogleGenerativeAI;
  private config: AgentConfig;
  private conversationHistory: Array<{
    role: "user" | "assistant";
    content: string;
  }> = [];

  constructor(apiKey: string, config: AgentConfig) {
    this.client = new GoogleGenerativeAI(apiKey);
    this.config = config;
  }

  async execute(userMessage: string): Promise<string> {
    this.conversationHistory.push({
      role: "user",
      content: userMessage,
    });

    const model = this.client.getGenerativeModel({
      model: this.config.model,
      systemInstruction: this.config.systemPrompt,
    });

    const response = await model.generateContent({
      contents: this.conversationHistory.map((msg) => ({
        role: msg.role,
        parts: [{ text: msg.content }],
      })),
    });

    const assistantMessage =
      response.response.content.parts[0].text || "";
    this.conversationHistory.push({
      role: "assistant",
      content: assistantMessage,
    });

    return assistantMessage;
  }

  resetContext(): void {
    this.conversationHistory = [];
  }
}
```

## Multi-Agent System Architecture

### Communication Patterns

**Pattern 1: Sequential Handoff**

```typescript
interface TaskResult {
  success: boolean;
  data: Record<string, any>;
  context: string;
}

class OrchestrationEngine {
  private agents: Map<string, GeminiAgent>;

  constructor() {
    this.agents = new Map();
  }

  registerAgent(name: string, agent: GeminiAgent): void {
    this.agents.set(name, agent);
  }

  async executeSequentialWorkflow(
    workflow: Array<{
      agentName: string;
      instruction: string;
    }>
  ): Promise<TaskResult> {
    let context = "";

    for (const step of workflow) {
      const agent = this.agents.get(step.agentName);
      if (!agent) {
        throw new Error(`Agent ${step.agentName} not found`);
      }

      const instruction = `${step.instruction}\n\nPrevious results: ${context}`;
      const result = await agent.execute(instruction);

      context += `\n[${step.agentName} result]\n${result}`;
    }

    return {
      success: true,
      data: { finalOutput: context },
      context,
    };
  }
}
```

**Pattern 2: Parallel Execution**

```typescript
async function executeParallelAgents(
  agents: Map<string, GeminiAgent>,
  task: string
): Promise<Record<string, string>> {
  const promises: Promise<[string, string]>[] = [];

  for (const [name, agent] of agents) {
    promises.push(agent.execute(task).then((result) => [name, result]));
  }

  const results = await Promise.all(promises);
  return Object.fromEntries(results);
}
```

**Pattern 3: Dynamic Routing**

```typescript
class IntelligentRouter {
  private agents: Map<string, GeminiAgent>;

  constructor(agents: Map<string, GeminiAgent>) {
    this.agents = agents;
  }

  async routeTask(input: string): Promise<TaskResult> {
    const routerPrompt = `
Classify the user request into one of these categories:
1. "sales_query" - Sales data and strategy questions
2. "trend_analysis" - Market trends and competitive analysis
3. "forecast" - Demand forecasts and scenario planning

User input: "${input}"

Output only the classification.
    `;

    const routerAgent = this.agents.get("router")!;
    const classification = await routerAgent.execute(routerPrompt);

    const targetAgent = this.agents.get(
      classification.toLowerCase().trim()
    );
    if (!targetAgent) {
      throw new Error(`No agent for: ${classification}`);
    }

    const result = await targetAgent.execute(input);
    return {
      success: true,
      data: { response: result },
      context: result,
    };
  }
}
```

## Session Management and Context Preservation

```typescript
interface SessionContext {
  sessionId: string;
  userId: string;
  startedAt: Date;
  conversationHistory: Array<{
    agentName: string;
    role: "user" | "assistant";
    message: string;
    timestamp: Date;
  }>;
  sharedMemory: Map<string, any>;
  metadata: Record<string, any>;
}

class SessionManager {
  private sessions: Map<string, SessionContext> = new Map();
  private sessionTTL = 30 * 60 * 1000; // 30 minutes

  createSession(userId: string): SessionContext {
    const sessionId = `sess_${Date.now()}_${Math.random().toString(36)}`;
    const context: SessionContext = {
      sessionId,
      userId,
      startedAt: new Date(),
      conversationHistory: [],
      sharedMemory: new Map(),
      metadata: {},
    };

    this.sessions.set(sessionId, context);

    setTimeout(
      () => this.sessions.delete(sessionId),
      this.sessionTTL
    );

    return context;
  }

  getSession(sessionId: string): SessionContext | undefined {
    return this.sessions.get(sessionId);
  }

  updateSharedMemory(
    sessionId: string,
    key: string,
    value: any
  ): void {
    const session = this.sessions.get(sessionId);
    if (session) {
      session.sharedMemory.set(key, value);
    }
  }
}
```

## Error Handling and Recovery Patterns

```typescript
type RetryStrategy = "exponential_backoff" | "linear" | "fixed";

interface RetryConfig {
  maxAttempts: number;
  strategy: RetryStrategy;
  baseDelayMs: number;
  maxDelayMs: number;
}

async function executeWithRetry(
  fn: () => Promise<TaskResult>,
  config: RetryConfig
): Promise<TaskResult> {
  let lastError: Error | null = null;

  for (let attempt = 0; attempt < config.maxAttempts; attempt++) {
    try {
      return await fn();
    } catch (error) {
      lastError = error as Error;
      console.warn(
        `Attempt ${attempt + 1} failed: ${lastError.message}`
      );

      if (attempt < config.maxAttempts - 1) {
        const delay = calculateDelay(attempt, config);
        await new Promise((resolve) =>
          setTimeout(resolve, delay)
        );
      }
    }
  }

  throw new Error(
    `Failed after ${config.maxAttempts} attempts: ${lastError?.message}`
  );
}

function calculateDelay(
  attempt: number,
  config: RetryConfig
): number {
  let delay: number;

  switch (config.strategy) {
    case "exponential_backoff":
      delay = config.baseDelayMs * Math.pow(2, attempt);
      break;
    case "linear":
      delay = config.baseDelayMs * (attempt + 1);
      break;
    case "fixed":
      delay = config.baseDelayMs;
      break;
  }

  return Math.min(delay, config.maxDelayMs);
}
```

## Cost Monitoring and Rate Limiting

```typescript
interface CostMetrics {
  inputTokens: number;
  outputTokens: number;
  estimatedCostUSD: number;
  timestamp: Date;
}

class CostMonitor {
  private metrics: CostMetrics[] = [];
  private readonly INPUT_COST_PER_1M = 0.00075;
  private readonly OUTPUT_COST_PER_1M = 0.003;
  private dailyBudgetUSD = 10;

  recordUsage(
    inputTokens: number,
    outputTokens: number
  ): CostMetrics {
    const cost =
      (inputTokens * this.INPUT_COST_PER_1M) / 1_000_000 +
      (outputTokens * this.OUTPUT_COST_PER_1M) / 1_000_000;

    const metric: CostMetrics = {
      inputTokens,
      outputTokens,
      estimatedCostUSD: cost,
      timestamp: new Date(),
    };

    this.metrics.push(metric);
    return metric;
  }

  getDailyCost(): number {
    const today = new Date();
    today.setHours(0, 0, 0, 0);

    return this.metrics
      .filter((m) => m.timestamp >= today)
      .reduce((sum, m) => sum + m.estimatedCostUSD, 0);
  }

  isWithinBudget(): boolean {
    return this.getDailyCost() < this.dailyBudgetUSD;
  }
}

class RateLimiter {
  private requestCounts: Map<string, number[]> = new Map();
  private readonly maxRequestsPerMinute = 60;

  async checkAndWait(userId: string): Promise<void> {
    const now = Date.now();
    const oneMinuteAgo = now - 60_000;

    const requests = this.requestCounts
      .get(userId)
      ?.filter((time) => time > oneMinuteAgo) || [];

    if (requests.length >= this.maxRequestsPerMinute) {
      const oldestRequest = Math.min(...requests);
      const waitTime = oldestRequest + 60_000 - now;
      await new Promise((resolve) =>
        setTimeout(resolve, waitTime)
      );
    }

    requests.push(now);
    this.requestCounts.set(userId, requests);
  }
}
```

## Observability Implementation

```typescript
interface TraceSpan {
  traceId: string;
  spanId: string;
  name: string;
  startTime: Date;
  endTime?: Date;
  status: "pending" | "success" | "error";
  attributes: Record<string, any>;
  error?: string;
}

class ObservabilityService {
  private spans: Map<string, TraceSpan> = new Map();

  startSpan(
    traceId: string,
    name: string,
    attributes: Record<string, any> = {}
  ): string {
    const spanId = `span_${Date.now()}_${Math.random()}`;
    const span: TraceSpan = {
      traceId,
      spanId,
      name,
      startTime: new Date(),
      status: "pending",
      attributes,
    };

    this.spans.set(spanId, span);
    return spanId;
  }

  endSpan(
    spanId: string,
    status: "success" | "error" = "success",
    error?: string
  ): void {
    const span = this.spans.get(spanId);
    if (span) {
      span.endTime = new Date();
      span.status = status;
      if (error) span.error = error;

      this.logSpan(span);
    }
  }

  private logSpan(span: TraceSpan): void {
    console.log(
      JSON.stringify({
        traceId: span.traceId,
        spanId: span.spanId,
        name: span.name,
        duration:
          (span.endTime?.getTime() || Date.now()) -
          span.startTime.getTime(),
        status: span.status,
        attributes: span.attributes,
        error: span.error,
      })
    );
  }
}
```

## Key Takeaways

Successfully building multi-agent systems with Google ADK and Gemini API depends on:

1. **Type Safety**: Use TypeScript strict mode to prevent bugs early
2. **Communication Patterns**: Choose sequential, parallel, or dynamic routing based on requirements
3. **Session Management**: Maintain context across extended interactions
4. **Error Handling**: Implement retry, fallback, and graceful degradation strategies
5. **Operational Foundation**: Integrate cost monitoring, rate limiting, and observability

By combining these implementation patterns, you can build production-grade systems that are both reliable and scalable.
