### #1: Gemini 3.1 Pro Custom Tools — Complete Developer Guide

Google's announcement of **Gemini 3.1 Pro** this mid-March introduced custom tool definitions at the API level. Developers can now define granular control over executable actions, making agent-based AI implementations remarkably streamlined.

This topic dominated our traffic for three key reasons:

1. **High Practical Value** — Build sophisticated agents in under 50 lines of JavaScript
2. **Surging Search Volume** — "Gemini 3.1 Pro API" searches increased 320% week-over-week
3. **Lower Implementation Barrier** — No more complex state management machinery

**Basic Custom Tool Definition Pattern:**

```javascript
const customTool = {
  name: "weather_lookup",
  description: "Fetch current weather for a specified city",
  inputSchema: {
    type: "object",
    properties: {
      city: {
        type: "string",
        description: "City name (e.g., Tokyo, New York)"
      },
      units: {
        type: "string",
        enum: ["celsius", "fahrenheit"]
      }
    },
    required: ["city"]
  }
};

const response = await genAI.getGenerativeModel({
  model: "gemini-1.5-pro",
  tools: [customTool]
}).generateContent(userQuery);
```
