Building a presentation from scratch is one of those tasks that eats up far more time than it should. Between outlining, writing, designing, finding visuals, and preparing speaker notes, a single deck can consume an entire afternoon. Gemini AI in Google Slides changes that equation dramatically.

This guide walks you through everything Gemini can do inside Google Slides — from generating entire decks to polishing designs and translating slides for global audiences. Whether you're new to Gemini or looking to level up your workflow, you'll find practical techniques you can use right away.

## What Gemini Can Do in Google Slides

As of March 2026, Gemini AI in Google Slides offers a powerful set of features that cover the entire presentation lifecycle.

**Auto-generate slide decks**: Describe your topic or provide an outline, and Gemini creates a multi-slide presentation for you. Instructions like "Create a 10-slide investor pitch deck for a SaaS product" work out of the box.

**Design suggestions and improvements**: Gemini analyzes your existing slides and recommends better layouts, color schemes, and typography. You can ask it to "make this look more professional" or "use a dark theme with blue accents."

**Image generation and insertion**: Powered by Imagen 3, Gemini can create custom visuals directly in your slides — no more hunting through stock photo libraries.

**Automatic speaker notes**: Gemini reads your slide content and generates speaker notes that help you deliver a polished presentation, complete with transition cues and emphasis points.

**Multilingual translation**: Translate your entire deck into other languages while preserving the layout and design.

## Getting Started: Generating Slides from a Prompt

The simplest way to use Gemini in Google Slides is through the side panel.

**Step 1**: Open Google Slides and click the Gemini icon (✨) in the right-side panel. You'll need a Google AI Pro / AI Ultra subscription or a Workspace Business plan with Gemini enabled.

**Step 2**: Write a detailed prompt. The more specific you are, the better the output.

```
Weak prompt: "Make a presentation"
Strong prompt: "Create a 12-slide investor pitch deck for a SaaS product.
               Include: problem statement, solution, market size,
               business model, competitive advantage, team intro,
               financial projections, and roadmap.
               Keep the tone professional and concise."
```

**Step 3**: Review the generated slides and iterate. You can give follow-up instructions like "Make slide 3 more data-driven" or "Change the chart colors to match our brand."

### Prompting Tips for Better Results

A few principles that consistently produce better slide decks:

1. **Specify your audience**: "For C-suite executives" vs. "For the engineering team" leads to very different content and depth
2. **Set a slide count**: Saying "around 10 slides" helps Gemini calibrate information density
3. **List required elements**: Mention if you need charts, case studies, FAQs, or specific sections
4. **Describe the visual direction**: "Minimal design," "dark theme," or "photo-heavy" all influence the output

## Improving Existing Slide Designs

Already have a deck that needs polish? Gemini can help with that too.

In the side panel, type something like "Improve the layout of this slide," and Gemini will optimize text placement, font sizing, and image positioning. Here are some particularly effective use cases:

**Text-heavy slides**: Ask Gemini to "Split this slide into two and add headings to each." It will redistribute the content into a more digestible format.

**Inconsistent design**: When multiple people have edited a deck, visual consistency often suffers. "Unify the fonts and color palette across all slides" fixes this in seconds.

**Data visualization**: Tell Gemini to "Convert this table into a chart," and it will select an appropriate chart type and create the visualization for you.

## Automating Slides with Apps Script and Gemini API

For power users, combining Google Apps Script with the Gemini API unlocks a new level of automation. Here's an example that generates slides from spreadsheet data:

```javascript
// Apps Script: Auto-generate slides from spreadsheet data
function generateSlidesFromSheet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName("PresentationData");
  const data = sheet.getDataRange().getValues();
  
  // Use Gemini API to generate slide structure
  const prompt = `Based on the following data, create a presentation structure in JSON.
Each slide should include title, body, and notes (speaker notes).

Data:
${data.map(row => row.join(", ")).join("\n")}

Return as JSON array:
[{"title": "...", "body": "...", "notes": "..."}, ...]`;

  const apiKey = PropertiesService.getScriptProperties()
    .getProperty("GEMINI_API_KEY");
  
  const response = UrlFetchApp.fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=${apiKey}`,
    {
      method: "post",
      contentType: "application/json",
      payload: JSON.stringify({
        contents: [{ parts: [{ text: prompt }] }],
        generationConfig: {
          responseMimeType: "application/json"
        }
      })
    }
  );

  const result = JSON.parse(response.getContentText());
  const slides = JSON.parse(
    result.candidates[0].content.parts[0].text
  );

  // Create slides in Google Slides
  const presentation = SlidesApp.create("Auto-Generated Presentation");
  
  // Remove the default blank slide
  presentation.getSlides()[0].remove();

  slides.forEach((slideData, index) => {
    const slide = presentation.appendSlide(
      SlidesApp.PredefinedLayout.TITLE_AND_BODY
    );
    
    // Set title and body text
    const shapes = slide.getShapes();
    shapes[0].getText().setText(slideData.title);
    shapes[1].getText().setText(slideData.body);
    
    // Set speaker notes
    slide.getNotesPage().getSpeakerNotesShape()
      .getText().setText(slideData.notes);
  });

  Logger.log(
    "Presentation created: " + presentation.getUrl()
  );
  // Expected output: Presentation created: https://docs.google.com/presentation/d/xxxxx/edit
}
```

This script is especially useful for recurring presentations like monthly reports, quarterly reviews, or weekly status updates where the structure stays the same but the data changes.

## Enhancing Slides with AI-Generated Images

Gemini's image generation capability (powered by Imagen 3) lets you create custom visuals without leaving Google Slides.

In the side panel, try prompts like "Generate an illustration that fits this slide." For better results, be specific: "A flat-design illustration of a business meeting, using blue and white colors."

**Practical scenarios**:

- Concept visualization: "An icon-style illustration representing cloud computing"
- Data imagery: "An upward-trending graph with people celebrating, modern flat style"
- Background images: "An abstract technology-themed background in dark blue tones"

The real advantage here isn't just saving time on image searches — it's getting original visuals that perfectly match your presentation's theme and brand.

## Auto-Generating Speaker Notes

Speaker notes are often an afterthought, but they make a huge difference in delivery quality. Gemini can read your slide content and create structured notes for each slide.

Try this prompt: "Add speaker notes to every slide. Keep each note 3–4 sentences long and ensure smooth transitions between slides."

The generated notes typically include:

- The key message for each slide
- Natural transitions from the previous slide
- Emphasis points for numerical data
- Suggestions for audience engagement or questions

You can fine-tune the style too: "Make the notes more conversational" or "Add technical context for a developer audience."

## Creating Multilingual Presentations

For global teams and international clients, Gemini can translate your entire deck while preserving the layout.

```
Example prompt:
"Translate this entire presentation into Japanese.
 Keep the layout and design exactly the same.
 Also translate the speaker notes.
 Use a formal, business-appropriate tone."
```

**Pro tip**: After translation, always run a review pass. Ask Gemini to "Check the translated text for unnatural phrasing and fix any issues." This second pass catches most of the rough edges.

## Frequently Asked Questions

### Is Gemini in Google Slides free to use?

Gemini features in Google Workspace require a Google AI Pro subscription ($19.99/month) or higher, or a Workspace Business/Enterprise plan that includes Gemini. Free Google accounts have limited access to Gemini features.

### Is it safe to use Gemini for presentations with confidential information?

Under the Google Workspace Gemini terms, your business data is not used to train Google's models. That said, always follow your organization's security policies when working with sensitive content.

### What if I don't like the design Gemini generates?

You can iterate as many times as you need. Give specific feedback like "Use a simpler layout," "Switch to sans-serif fonts," or "Use a monochrome color scheme." You can also manually apply a theme or master slide and then ask Gemini to "Reformat all slides to match this new theme."

### Does using the Gemini API via Apps Script cost extra?

Yes. API calls through Apps Script are billed through Google AI Studio pricing. However, there's a free tier (15 requests per minute for Gemini 2.5 Flash), so small-scale automation can be tested at no cost.
