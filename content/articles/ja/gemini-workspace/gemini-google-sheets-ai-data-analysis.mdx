## はじめに

Google スプレッドシートにGemini AIが統合されたことで、データ分析の作業が大きく変わりつつあります。これまで関数の書き方を調べたり、複雑な数式を試行錯誤しながら組み立てていた時間が、自然な言葉で指示するだけで済むようになりました。

この記事では、Gemini AIをGoogle スプレッドシートで活用するための基本操作から、日常業務で役立つ実践的なテクニックまでを丁寧に解説します。

## Gemini サイドパネルの基本

### サイドパネルを開く

Google スプレッドシートでGeminiを使うには、画面右上の **Geminiアイコン**（✦マーク）をクリックします。サイドパネルが開き、チャット形式でAIに指示を出せるようになります。

Google Workspace の Business / Enterprise プラン、または Google One AI Premium プランに加入していれば、追加設定なしで利用可能です。

### できることの概要

Geminiサイドパネルでは、主に以下のことが行えます。

- **データの要約**: 「このシートの売上傾向を教えて」のように、データ全体の傾向や特徴を自然言語で質問
- **関数の生成**: 「A列とB列を掛け算してC列に入れる関数を作って」のように、やりたいことを言葉で伝えるだけで数式を提案
- **表の作成**: 「月別の予算テンプレートを作って」のように、ゼロからテーブル構造を生成
- **データの整理**: 重複削除、フォーマット統一、カテゴリ分類などの整形作業を依頼

## 実践テクニック① — 自然言語で関数を生成する

スプレッドシートの関数は種類が多く、慣れていないと書くのに時間がかかります。Geminiを使えば、やりたいことを日本語で伝えるだけで適切な関数を提案してもらえます。

### 基本的な使い方

サイドパネルに以下のように入力します。

```
B列の数値がすべて100以上かどうかを判定する関数を作って
```

Geminiは `=AND(ARRAYFORMULA(B2:B100>=100))` のような関数を提案し、ワンクリックでセルに挿入できます。

### よく使うプロンプト例

| やりたいこと | プロンプト例 |
|---|---|
| 条件付き合計 | 「カテゴリがAの行だけ売上を合計して」 |
| 日付の計算 | 「今日から各締切日までの残り日数を計算して」 |
| テキスト整形 | 「名前の列を姓と名に分割して」 |
| 重複チェック | 「メールアドレスの重複を見つけて」 |
| VLOOKUP生成 | 「Sheet2の商品コードに対応する商品名をこのシートに持ってきて」 |

ポイントは、**セル範囲や列の名前を具体的に伝えること**です。「A列の値を使って…」のように明示すると、より正確な関数が返ってきます。

## 実践テクニック② — データ分析を会話で進める

大量のデータを前にして「まず何を見ればいいか」が分からないとき、Geminiに相談するのが効果的です。

### 分析の進め方

まずは全体像を把握する質問から始めます。

```
このデータセットの概要を教えて。主要な傾向やパターンはある？
```

Geminiがデータを読み取り、行数・列数・データ型の確認とともに、目立つ傾向を要約してくれます。そこから深掘りしていきましょう。

```
売上が最も高い月はどれ？前年同月比も出せる？
```

```
地域別の平均単価を比較して、表にまとめて
```

このように、対話を重ねながら段階的に分析を進められるのがGeminiの大きな利点です。

### グラフの自動生成

分析結果を視覚化したい場合も、Geminiに依頼できます。

```
月別売上の推移を折れ線グラフで作って
```

Geminiがデータ範囲を判断し、適切なグラフをシート上に挿入してくれます。グラフの種類（棒グラフ、円グラフなど）やタイトルも指定可能です。

## 実践テクニック③ — テーブルの自動生成

ゼロから表を作りたいときにも、Geminiは頼りになります。

### テンプレート生成の例

```
週次のプロジェクト進捗管理表を作って。列はタスク名、担当者、ステータス、期限、備考
```

Geminiが構造化されたテーブルを生成し、シートに挿入してくれます。ステータスのドロップダウンリストまで提案してくれることもあります。

### サンプルデータの生成

テスト用のデータが必要なときにも便利です。

```
架空の顧客データを20件作って。名前、メールアドレス、購入金額、購入日を含めて
```

プロトタイピングやダッシュボードの検証に使えるリアルなサンプルデータをすぐに用意できます。

## 実践テクニック④ — データのクリーニング

実務では、外部システムからエクスポートしたデータの整形に多くの時間を取られがちです。Geminiを使えば、この作業も効率化できます。

### よくあるクリーニング作業

```
電話番号の列をハイフン付きの統一フォーマットに揃えて
```

```
空白行を見つけてハイライトして
```

```
日付の列を「YYYY/MM/DD」形式に統一して
```

手作業では見落としやすいフォーマットの不統一も、Geminiに任せることで正確に処理できます。

## 活用のコツとベストプラクティス

Geminiをスプレッドシートで効果的に使うために、いくつかのポイントを押さえておきましょう。

**具体的に指示する**: 「データを分析して」よりも「B列の売上データを月別に集計して、前月比の増減率も出して」のように具体的な方が、精度の高い回答が得られます。

**ヘッダー行を整えておく**: Geminiはヘッダー行を読み取ってデータ構造を理解します。「列A」「列B」のような曖昧な名前より、「商品名」「販売数量」のように意味のある名前にしておくと、より正確な処理が期待できます。

**段階的に依頼する**: 一度に複雑な処理を頼むよりも、ステップを分けて依頼する方が成功率が高くなります。まず集計、次にフィルタ、最後にグラフ化、というように進めると良いでしょう。

**結果を確認する習慣をつける**: Geminiが生成した関数や分析結果は、必ず目視で確認しましょう。特に数値計算では、サンプルデータで検算しておくと安心です。

Google スプレッドシートとGemini AIの組み合わせは、データ作業の効率を大きく向上させてくれるツールです。関数を覚えていなくても、分析手法に詳しくなくても、やりたいことを言葉で伝えるだけで作業を進められます。

まずはサイドパネルを開いて、簡単な質問から試してみてください。使い慣れるほど、Geminiがデータ作業の心強いパートナーになってくれるはずです。
