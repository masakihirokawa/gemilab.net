プレゼン資料の作成は、多くのビジネスパーソンにとって時間のかかる作業です。構成を考え、テキストをまとめ、デザインを整え、画像を探し、スピーカーノートを準備する——これらの工程すべてに Gemini AI が対応できるようになりました。

この記事では、Google Slides に統合された Gemini AI の機能を網羅的に紹介し、プレゼン資料作成のワークフローを大幅に効率化する方法を解説します。初めて Gemini を使う方でも安心して取り組める内容です。

## Gemini × Google Slides でできること

2026年3月現在、Google Slides の Gemini AI は以下の機能を提供しています。

**スライドの自動生成**: テーマやアウトラインを伝えるだけで、複数ページのスライドを一括作成できます。「新規事業の社内提案資料を10枚で作って」のような自然言語の指示に対応しています。

**デザインの提案と改善**: 既存のスライドに対して、配色・レイアウト・フォントの改善を Gemini が提案します。「もっとプロフェッショナルな印象にして」「コーポレートカラーの青を基調にして」といった指示が可能です。

**画像の生成と挿入**: Imagen 3 ベースの画像生成機能により、スライドに合ったビジュアルをその場で作成できます。ストック画像を探す手間が不要になります。

**スピーカーノートの自動作成**: スライドの内容に基づいて、発表時に使えるスピーカーノートを自動生成します。発表の流れや強調すべきポイントも含めて作成されます。

**多言語への翻訳**: 日本語で作成したプレゼン資料を英語や他の言語に翻訳できます。レイアウトを維持したまま、テキストのみを変換します。

## 基本の使い方：テーマからスライドを自動生成する

Google Slides で Gemini を使う最も基本的な方法は、サイドパネルからの指示です。

**手順 1**: Google Slides を開き、右側のサイドパネルにある Gemini アイコン（✨）をクリックします。Google Workspace に Gemini が有効化されている必要があります（Google AI Pro / AI Ultra サブスクリプション、または Workspace Business 以上のプラン）。

**手順 2**: プロンプトを入力します。具体的であるほど、期待に近い結果が得られます。

```
悪い例：「プレゼンを作って」
良い例：「SaaS プロダクトの投資家向けピッチデッキを12枚で作成してください。
       内容は、課題提起、解決策、市場規模、ビジネスモデル、
       競合優位性、チーム紹介、財務計画、ロードマップを含めてください。
       トーンはプロフェッショナルかつ簡潔に。」
```

**手順 3**: Gemini が生成したスライドを確認し、必要に応じて微調整します。「3枚目のスライドをもっとデータ中心にして」「グラフの色を変えて」など、追加の指示でイテレーションできます。

### プロンプトのコツ

Gemini にプレゼン資料を作成させる際、以下のポイントを押さえておくとクオリティが上がります。

1. **対象読者を明示する**: 「経営層向け」「技術チーム向け」「顧客向け」など、読み手を指定すると適切なトーンと粒度になります
2. **スライド枚数を指定する**: 「10枚程度」と伝えることで、情報量が適切に調整されます
3. **含めたい要素を列挙する**: 「グラフ」「事例紹介」「FAQ」など、必須の要素を事前に伝えます
4. **デザインの方向性を伝える**: 「ミニマル」「ダーク基調」「写真を多用」など、ビジュアルの好みも指示できます

## 既存スライドのデザインを改善する

すでに作成済みのスライドがある場合、Gemini にデザインの改善を依頼できます。

サイドパネルで「このスライドのレイアウトを改善して」と入力すると、テキストの配置やフォントサイズ、画像の位置などを最適化してくれます。特に以下のケースで効果的です。

**テキストが多すぎるスライド**: 「このスライドの情報を2枚に分割して、それぞれに見出しをつけて」と指示すると、読みやすく再構成されます。

**統一感のないデザイン**: 複数の人が編集したスライドにありがちなデザインの不統一を「全スライドのフォントと配色を統一して」で解消できます。

**データの可視化**: 「この表をグラフに変換して」と依頼すると、データに適したチャート形式を Gemini が選択して視覚化してくれます。

## Apps Script で Gemini API を使ったスライド自動化

Google Slides の UI から Gemini を使うだけでなく、Apps Script と Gemini API を組み合わせることで、より高度な自動化が実現できます。

以下は、スプレッドシートのデータから自動でスライドを生成する例です。

```javascript
// Apps Script: スプレッドシートのデータからスライドを自動生成
function generateSlidesFromSheet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName("プレゼンデータ");
  const data = sheet.getDataRange().getValues();
  
  // Gemini API でスライド構成を生成
  const prompt = `以下のデータを基に、プレゼンテーションの構成を JSON で返してください。
各スライドには title, body, notes（スピーカーノート）を含めてください。

データ:
${data.map(row => row.join(", ")).join("\n")}

JSON形式で返してください:
[{"title": "...", "body": "...", "notes": "..."}, ...]`;

  const apiKey = PropertiesService.getScriptProperties()
    .getProperty("GEMINI_API_KEY");
  
  const response = UrlFetchApp.fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=${apiKey}`,
    {
      method: "post",
      contentType: "application/json",
      payload: JSON.stringify({
        contents: [{ parts: [{ text: prompt }] }],
        generationConfig: {
          responseMimeType: "application/json"
        }
      })
    }
  );

  const result = JSON.parse(response.getContentText());
  const slides = JSON.parse(
    result.candidates[0].content.parts[0].text
  );

  // Google Slides にスライドを作成
  const presentation = SlidesApp.create("自動生成プレゼン");
  
  // デフォルトの空スライドを削除
  presentation.getSlides()[0].remove();

  slides.forEach((slideData, index) => {
    const slide = presentation.appendSlide(
      SlidesApp.PredefinedLayout.TITLE_AND_BODY
    );
    
    // タイトルとボディを設定
    const shapes = slide.getShapes();
    shapes[0].getText().setText(slideData.title);
    shapes[1].getText().setText(slideData.body);
    
    // スピーカーノートを設定
    slide.getNotesPage().getSpeakerNotesShape()
      .getText().setText(slideData.notes);
  });

  Logger.log(
    "プレゼン作成完了: " + presentation.getUrl()
  );
  // 期待される出力: プレゼン作成完了: https://docs.google.com/presentation/d/xxxxx/edit
}
```

このスクリプトを実行すると、スプレッドシートのデータに基づいたプレゼン資料が自動で作成されます。定例レポートや月次報告など、定型のプレゼンに特に有効です。

## 画像生成でスライドのビジュアルを強化する

Gemini の画像生成機能（Imagen 3 ベース）を使えば、スライドに挿入するビジュアルをその場で作成できます。

サイドパネルで「このスライドに合うイラストを生成して」と指示するだけです。より具体的に「フラットデザインのビジネスミーティングのイラスト、青と白の配色」と伝えると、イメージ通りの画像が生成されます。

**活用シーン例**:

- コンセプトの視覚化: 「クラウドコンピューティングの概念を表すアイコン風イラスト」
- データのイメージ化: 「成長を表す右肩上がりのグラフと人々のイラスト」
- 背景画像: 「テクノロジーをテーマにした抽象的な背景画像、ダークブルー基調」

ストック画像サイトで検索する時間を節約できるだけでなく、プレゼンのテーマに完全にマッチしたオリジナル画像が使えるのが大きなメリットです。

## スピーカーノートの自動生成と発表準備

プレゼンの準備で見落とされがちなのがスピーカーノートです。Gemini はスライドの内容を読み取り、発表に最適なノートを自動生成します。

サイドパネルで「すべてのスライドにスピーカーノートを追加して。1スライドあたり3〜4文で、発表の流れが自然になるようにして」と指示します。

生成されるスピーカーノートには以下が含まれます。

- そのスライドで伝えるべき主要メッセージ
- 前のスライドからの自然なつなぎ
- 数値データがある場合は、強調すべきポイントの補足
- 聴衆への問いかけや、インタラクションの提案

「もっとカジュアルなトーンで」「技術的な補足を多めに」など、スタイルの調整も可能です。

## 多言語プレゼンの作成

グローバルなチームやクライアントに向けたプレゼンでは、多言語対応が求められます。Gemini を使えば、レイアウトを維持したまま、スライドのテキストを他言語に翻訳できます。

```
プロンプト例:
「このプレゼンテーション全体を英語に翻訳してください。
  レイアウトとデザインはそのまま維持し、
  スピーカーノートも合わせて翻訳してください。
  ビジネス文書にふさわしいフォーマルなトーンで。」
```

**注意点**: 翻訳後は必ずネイティブスピーカーまたは Gemini 自身にレビューを依頼してください。「翻訳結果を確認して、不自然な表現があれば修正して」とフォローアップの指示を出すと、品質が向上します。

## よくある質問（FAQ）

### Q: Gemini × Google Slides は無料で使えますか？

Google Workspace の Gemini 機能を利用するには、Google AI Pro（月額 ¥2,900）以上のサブスクリプション、または Gemini が含まれる Workspace Business / Enterprise プランが必要です。無料の Google アカウントでは一部機能に制限があります。

### Q: 機密情報を含むプレゼンを Gemini で作成しても安全ですか？

Google Workspace 版の Gemini では、ビジネスデータの保護に関する規約が適用されます。ユーザーの入力データがモデルのトレーニングに使用されないことが明記されています。ただし、社内のセキュリティポリシーに沿って運用することを推奨します。

### Q: Gemini が生成したスライドのデザインが気に入らない場合は？

「レイアウトをもっとシンプルにして」「フォントをゴシック体に変えて」「配色をモノトーンにして」など、具体的なフィードバックを何度でも指示できます。また、手動でテーマやマスタースライドを変更した後に「この新しいテーマに合わせて全スライドを再構成して」と依頼することも可能です。

### Q: Apps Script での Gemini API 利用に追加費用はかかりますか？

はい。Apps Script から Gemini API を呼び出す場合は、Google AI Studio の API 利用料金が別途発生します。ただし、無料枠（Gemini 2.5 Flash で毎分15リクエスト）もあるため、小規模な自動化であれば無料で試せます。
