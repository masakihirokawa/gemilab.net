## Gemini エージェントの概要

Gemini エージェントフレームワークは、自律的に判断し、複数のツールを組み合わせて、複雑なタスクを解決するAIシステムを構築するためのフレームワークです。

### エージェントとは

エージェントは以下の特性を持つAIシステムです：

- **自律性**: ユーザーの指示なしで自ら判断して行動
- **目標指向性**: 与えられたゴールを達成するための計画を立案
- **ツール活用**: 外部APIやデータベースなどのツールを適切に選択・実行
- **反復的思考**: 結果を評価し、改善するために再計画

### 実世界での応用例

- **カスタマーサポート**: 自動的に顧客の問題を診断・解決
- **データ分析**: 複数のデータソースから自動的に分析を実施
- **コンテンツ管理**: 記事の執筆、編集、公開を自動化
- **タスク自動化**: 複数のビジネスプロセスの統合・自動実行

## エージェント開発の基本構造

### シンプルなエージェントの構築

```python
import google.generativeai as genai
from typing import Callable, Any
import json

# APIの初期化
genai.configure(api_key="YOUR_API_KEY")

class SimpleAgent:
    """基本的なAIエージェント"""

    def __init__(self, model: str = "gemini-2.0-flash"):
        self.client = genai.Client()
        self.model = model
        self.tools = {}
        self.conversation_history = []

    def register_tool(self, name: str, tool_fn: Callable, description: str):
        """エージェントにツールを登録"""
        self.tools[name] = {
            "function": tool_fn,
            "description": description
        }

    def execute_tool(self, tool_name: str, **kwargs) -> Any:
        """登録されたツールを実行"""
        if tool_name not in self.tools:
            return f"エラー: ツール '{tool_name}' は登録されていません"

        try:
            result = self.tools[tool_name]["function"](**kwargs)
            return result
        except Exception as e:
            return f"ツール実行エラー: {str(e)}"

    def process_request(self, user_request: str) -> str:
        """ユーザーリクエストを処理"""
        # 会話履歴に追加
        self.conversation_history.append({
            "role": "user",
            "content": user_request
        })

        # 利用可能なツールのリストを作成
        tools_description = "\n".join([
            f"- {name}: {info['description']}"
            for name, info in self.tools.items()
        ])

        # システムプロンプト
        system_prompt = f"""
        あなたは自律的なAIエージェントです。
        以下のツールを使用できます：
        {tools_description}

        タスクを完了するために必要なツールを判断し、実行してください。
        ツール実行の結果に基づいて、次のステップを決定してください。
        """

        # Gemini APIを呼び出し
        response = self.client.models.generate_content(
            model=self.model,
            contents=user_request,
            system_instruction=system_prompt
        )

        assistant_response = response.text

        # 会話履歴に追加
        self.conversation_history.append({
            "role": "assistant",
            "content": assistant_response
        })

        return assistant_response

# 使用例
def example_search_web(query: str) -> str:
    """Webから情報を検索（シミュレーション）"""
    return f"検索結果: '{query}' に関連する10件の結果を取得しました"

def example_fetch_data(source: str) -> str:
    """データソースから情報を取得"""
    return f"{source} からデータを取得しました: sample_data"

# エージェントの初期化
agent = SimpleAgent()
agent.register_tool("search", example_search_web, "Webを検索して情報を取得します")
agent.register_tool("fetch_data", example_fetch_data, "指定されたデータソースからデータを取得します")

# リクエストの処理
result = agent.process_request("AI技術の最新動向について調べてください")
print("エージェントの応答:")
print(result)

# 出力例:
# エージェントの応答:
# AI技術の最新動向について調べるため、Web検索を実行します。
# 検索結果: 'AI技術の最新動向' に関連する10件の結果を取得しました
# これらの結果に基づいて、以下のポイントが重要です：...
```

## マルチエージェント協調システム

### 複数エージェントの連携

```python
from typing import List, Dict

class MultiAgentSystem:
    """複数エージェントが協調して動作するシステム"""

    def __init__(self):
        self.agents: Dict[str, SimpleAgent] = {}
        self.coordinator_agent = SimpleAgent()

    def register_agent(self, agent_name: str, specialization: str) -> SimpleAgent:
        """特定の領域に特化したエージェントを登録"""
        agent = SimpleAgent()
        self.agents[agent_name] = {
            "agent": agent,
            "specialization": specialization
        }
        return agent

    def orchestrate_task(self, task: str) -> Dict[str, str]:
        """複数エージェントの協調による複雑なタスク実行"""

        # コーディネータがタスクを分解
        decomposition_prompt = f"""
        次のタスクを、複数のエージェントで協力して完成させるために分解してください：
        タスク: {task}

        登録されているエージェント：
        {json.dumps({name: info['specialization'] for name, info in self.agents.items()}, ensure_ascii=False)}

        各エージェントの役割分担を提案してください。
        """

        response = self.coordinator_agent.process_request(decomposition_prompt)

        # 各エージェントにタスクを割り当て
        results = {}
        for agent_name, agent_info in self.agents.items():
            agent = agent_info["agent"]
            specialization = agent_info["specialization"]

            task_prompt = f"""
            {specialization} の専門家として、次のタスクに貢献してください：
            {task}

            {response}
            """

            results[agent_name] = agent.process_request(task_prompt)

        return results

# 使用例
multi_agent = MultiAgentSystem()

# 異なる専門分野のエージェントを登録
multi_agent.register_agent("analyst", "データ分析と統計的インサイト")
multi_agent.register_agent("developer", "技術的な実装と最適化")
multi_agent.register_agent("strategist", "ビジネス戦略と計画")

# 複雑なタスクを実行
task = "新しいAIアシスタントシステムの開発計画を作成してください"
results = multi_agent.orchestrate_task(task)

print("マルチエージェント協調の結果:")
for agent_name, result in results.items():
    print(f"\n[{agent_name}の視点]")
    print(result[:300] + "...")

# 出力例:
# マルチエージェント協調の結果:
#
# [analystの視点]
# データ分析の観点からは、ユーザー行動分析が重要です。
# 既存システムの利用パターンを分析することで...
#
# [developerの視点]
# 技術的実装としては、モジュール化されたアーキテクチャが推奨されます...
```

## ツール統合とAPI連携

### 外部ツールとの連携

```python
class ToolIntegrationAgent(SimpleAgent):
    """複数の外部ツールと統合するエージェント"""

    def __init__(self):
        super().__init__()
        self.api_integrations = {}

    def integrate_api(self, api_name: str, api_endpoint: str, schema: Dict):
        """APIをエージェントに統合"""
        self.api_integrations[api_name] = {
            "endpoint": api_endpoint,
            "schema": schema
        }

    def execute_with_tools(self, task: str) -> str:
        """ツール統合を使用してタスクを実行"""

        # 利用可能なAPIのスキーマを作成
        api_schema = json.dumps(self.api_integrations, ensure_ascii=False)

        # ツール利用を考慮したプロンプト
        enhanced_prompt = f"""
        次のタスクを完了してください：{task}

        利用可能なAPI：
        {api_schema}

        必要に応じて、適切なAPIを選択して実行してください。
        """

        response = self.client.models.generate_content(
            model=self.model,
            contents=enhanced_prompt
        )

        return response.text

# 使用例
tool_agent = ToolIntegrationAgent()

# API統合を追加
tool_agent.integrate_api(
    "weather_api",
    "https://api.weather.example.com",
    {
        "location": "string",
        "unit": "celsius|fahrenheit"
    }
)

tool_agent.integrate_api(
    "database_api",
    "https://api.database.example.com",
    {
        "query": "string",
        "table": "string"
    }
)

# ツール統合を使用してタスクを実行
task = "東京の天気情報を取得し、過去1年間の気温傾向を分析してください"
result = tool_agent.execute_with_tools(task)

print("ツール統合による実行結果:")
print(result)
```

## エージェントの評価と最適化

### パフォーマンス測定

```python
class AgentEvaluator:
    """エージェントのパフォーマンスを測定・最適化"""

    def __init__(self, agent: SimpleAgent):
        self.agent = agent
        self.metrics = {
            "total_requests": 0,
            "successful_tasks": 0,
            "failed_tasks": 0,
            "average_response_time": 0,
            "tool_usage_counts": {}
        }

    def evaluate_agent(self, test_cases: List[str]) -> Dict:
        """複数のテストケースでエージェントを評価"""
        import time

        for test_case in test_cases:
            start_time = time.time()

            try:
                response = self.agent.process_request(test_case)

                execution_time = time.time() - start_time

                self.metrics["total_requests"] += 1
                self.metrics["successful_tasks"] += 1

                # 平均応答時間を更新
                total_time = (self.metrics["average_response_time"] *
                              (self.metrics["total_requests"] - 1) + execution_time)
                self.metrics["average_response_time"] = (
                    total_time / self.metrics["total_requests"]
                )

            except Exception as e:
                self.metrics["total_requests"] += 1
                self.metrics["failed_tasks"] += 1

        return self.metrics

    def generate_report(self) -> str:
        """評価結果レポートを生成"""
        success_rate = (
            self.metrics["successful_tasks"] / self.metrics["total_requests"] * 100
            if self.metrics["total_requests"] > 0 else 0
        )

        report = f"""
        === エージェント評価レポート ===
        総リクエスト数: {self.metrics['total_requests']}
        成功したタスク: {self.metrics['successful_tasks']}
        失敗したタスク: {self.metrics['failed_tasks']}
        成功率: {success_rate:.1f}%
        平均応答時間: {self.metrics['average_response_time']:.2f}秒
        """

        return report

# 使用例
evaluator = AgentEvaluator(agent)

test_cases = [
    "簡単な計算をしてください: 125 + 456",
    "昨日の天気について調べてください",
    "複雑なデータ分析タスクを実行してください"
]

metrics = evaluator.evaluate_agent(test_cases)
report = evaluator.generate_report()

print(report)

# 出力例:
# === エージェント評価レポート ===
# 総リクエスト数: 3
# 成功したタスク: 2
# 失敗したタスク: 1
# 成功率: 66.7%
# 平均応答時間: 2.15秒
```

## よくある質問

### Q1: エージェントシステムとチャットボットの違いは何ですか？

**A:** チャットボットは主にユーザーの質問に応答するための会話システムです。一方、エージェントは自律的に目標達成に向けて行動し、複数のツールを組み合わせて複雑なタスクを解決できます。エージェントは計画立案と意思決定能力が備わっています。

### Q2: マルチエージェントシステムの調整はどうすればいいですか？

**A:** マルチエージェントシステムを効果的に調整するには、以下が重要です：
1. 各エージェントの役割を明確に定義
2. エージェント間の通信プロトコルを設計
3. コンフリクト解決メカニズムを実装
4. 継続的な監視と最適化

### Q3: エージェントのセキュリティに関する考慮事項は？

**A:** エージェントにツール統合を行う場合、以下の対策が必須です：
- APIキーやクレデンシャルの安全な管理
- 外部API呼び出しの検証と認可
- エージェントの行動ログ記録と監査
- 予期しない動作の制限とサンドボックス化
