# Gemini Function Calling完全ガイド — 外部API連携の実装

## はじめに

Function Callingは、Gemini APIの最も強力な機能の一つです。この機能を使用することで、AIに外部システムやAPIへのアクセスを許可し、リアルタイムデータ取得や複雑なビジネスロジックの実行が可能になります。本記事では、Function Callingの概念から実装、そして複雑なユースケースまで、体系的に解説します。

## Function Callingとは

Function Callingは、ユーザーの質問や指示に対して、Geminiが自動的に外部の関数やAPI呼び出しを実行する仕組みです。従来のAIは単に応答テキストを生成するのみでしたが、Function Callingにより、AIが実際のアクション（データベース参照、外部API呼び出しなど）を実行できるようになりました。

### Function Callingの流れ

1. ユーザーが質問や指示をGeminiに送信
2. Geminiが質問に基づいて、呼び出すべき関数を判断
3. 関数名とパラメータをJSON形式で返却
4. アプリケーション側が実際に関数を実行
5. 実行結果をGeminiに送り返す
6. Geminiが結果を処理して、ユーザーに最終的な応答を返す

## ステップ1：関数定義（Function Declarations）

### 関数定義の構造

Function Callingを使用するには、あらかじめGeminiに対して、利用可能な関数の定義を提供する必要があります。これを「Function Declaration」と呼びます。

```python
from google import genai
from google.genai import types

# 利用可能な関数を定義
tools = [
    types.Tool(
        function_declarations=[
            {
                "name": "get_weather",
                "description": "指定された都市の現在の天気情報を取得します",
                "parameters": {
                    "type": "object",
                    "properties": {
                        "city": {
                            "type": "string",
                            "description": "都市名（例：東京、大阪）"
                        },
                        "unit": {
                            "type": "string",
                            "enum": ["celsius", "fahrenheit"],
                            "description": "温度単位（デフォルト：celsius）"
                        }
                    },
                    "required": ["city"]
                }
            },
            {
                "name": "search_products",
                "description": "ECサイトで商品を検索します",
                "parameters": {
                    "type": "object",
                    "properties": {
                        "query": {
                            "type": "string",
                            "description": "検索キーワード"
                        },
                        "category": {
                            "type": "string",
                            "description": "商品カテゴリ（オプション）"
                        },
                        "max_price": {
                            "type": "number",
                            "description": "最大価格（オプション）"
                        }
                    },
                    "required": ["query"]
                }
            }
        ]
    )
]
```

### パラメータスキーマの詳細

関数定義の `parameters` フィールドは、JSON Schema形式で記述されます。以下の要素が重要です：

- **type**：パラメータの型（"object"、"string"、"number"、"array"など）
- **properties**：各パラメータの詳細定義
- **required**：必須パラメータのリスト
- **description**：パラメータの説明（Geminiがパラメータを選択する際に参考にします）

## ステップ2：Python での Function Calling 実装

### 基本的な実装パターン

```python
from google import genai
import json

api_key = os.getenv("GEMINI_API_KEY")
client = genai.Client(api_key=api_key)

# 実際の関数実装
def get_weather(city: str, unit: str = "celsius") -> dict:
    """実際の天気API呼び出しをシミュレート"""
    weather_data = {
        "東京": {"temperature": 22, "condition": "晴れ"},
        "大阪": {"temperature": 20, "condition": "曇り"},
        "京都": {"temperature": 18, "condition": "雨"}
    }
    data = weather_data.get(city, {"temperature": 15, "condition": "不明"})
    if unit == "fahrenheit":
        data["temperature"] = data["temperature"] * 9/5 + 32
    return data

def search_products(query: str, category: str = None, max_price: float = None) -> list:
    """ECサイトの商品検索をシミュレート"""
    # 実装省略 - 実際には外部APIを呼び出す
    return [
        {"id": 1, "name": f"{query}商品A", "price": 1000, "category": category or "一般"},
        {"id": 2, "name": f"{query}商品B", "price": 2000, "category": category or "一般"}
    ]

# 関数ディクショナリー
available_functions = {
    "get_weather": get_weather,
    "search_products": search_products
}

def process_function_call(function_name: str, function_args: dict):
    """関数を実行して結果を返す"""
    if function_name in available_functions:
        return available_functions[function_name](**function_args)
    else:
        return {"error": f"関数 {function_name} は利用できません"}
```

### Function Calling フロー

```python
def chat_with_function_calling(user_message: str):
    """Function Callingを含むチャット実装"""

    # Tool定義を含める
    tools = [
        types.Tool(
            function_declarations=[
                {
                    "name": "get_weather",
                    "description": "都市の天気情報を取得",
                    "parameters": {
                        "type": "object",
                        "properties": {
                            "city": {"type": "string"}
                        },
                        "required": ["city"]
                    }
                },
                {
                    "name": "search_products",
                    "description": "商品を検索",
                    "parameters": {
                        "type": "object",
                        "properties": {
                            "query": {"type": "string"}
                        },
                        "required": ["query"]
                    }
                }
            ]
        )
    ]

    # 初回リクエスト
    response = client.models.generate_content(
        model="gemini-2.5-pro",
        contents=user_message,
        tools=tools,
        config=types.GenerateContentConfig(
            temperature=0.5
        )
    )

    # レスポンスを処理
    while True:
        if response.candidates[0].content.parts:
            for part in response.candidates[0].content.parts:
                if part.function_call:
                    # Function Callingが必要な場合
                    function_name = part.function_call.name
                    function_args = dict(part.function_call.args)

                    print(f"呼び出し関数: {function_name}")
                    print(f"パラメータ: {function_args}")

                    # 関数を実行
                    result = process_function_call(function_name, function_args)

                    # 結果をGeminiに送信
                    response = client.models.generate_content(
                        model="gemini-2.5-pro",
                        contents=[
                            {"role": "user", "parts": [{"text": user_message}]},
                            response.candidates[0].content,
                            {
                                "role": "user",
                                "parts": [{
                                    "function_response": {
                                        "name": function_name,
                                        "response": result
                                    }
                                }]
                            }
                        ],
                        tools=tools
                    )
                elif hasattr(part, 'text') and part.text:
                    # 最終的なテキスト応答
                    print(f"Gemini: {part.text}")
                    return
        else:
            return

# 使用例
chat_with_function_calling("東京の天気を教えてください")
```

## ステップ3：TypeScript での Function Calling 実装

### TypeScript での関数定義と実装

```typescript
import { GoogleGenerativeAI } from "@google/generative-ai";

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY!);

// 関数定義
const tools = [
  {
    name: "get_weather",
    description: "指定都市の天気情報を取得",
    parameters: {
      type: "object",
      properties: {
        city: {
          type: "string",
          description: "都市名"
        }
      },
      required: ["city"]
    }
  },
  {
    name: "search_products",
    description: "商品検索",
    parameters: {
      type: "object",
      properties: {
        query: {
          type: "string",
          description: "検索キーワード"
        }
      },
      required: ["query"]
    }
  }
];

// 実装関数
function getWeather(city: string): object {
  const weatherData: { [key: string]: object } = {
    "東京": { temperature: 22, condition: "晴れ" },
    "大阪": { temperature: 20, condition: "曇り" }
  };
  return weatherData[city] || { temperature: 15, condition: "不明" };
}

function searchProducts(query: string): object[] {
  return [
    { id: 1, name: `${query}商品A`, price: 1000 },
    { id: 2, name: `${query}商品B`, price: 2000 }
  ];
}

const functionRegistry: { [key: string]: Function } = {
  get_weather: getWeather,
  search_products: searchProducts
};

async function executeFunction(
  functionName: string,
  args: { [key: string]: string }
): Promise<object> {
  const func = functionRegistry[functionName];
  if (!func) {
    return { error: `Unknown function: ${functionName}` };
  }
  return func(...Object.values(args));
}
```

### TypeScript での Function Calling フロー

```typescript
async function chatWithFunctionCalling(userMessage: string): Promise<void> {
  const model = genAI.getGenerativeModel({
    model: "gemini-2.5-pro"
  });

  let messages: any[] = [
    {
      role: "user",
      parts: [{ text: userMessage }]
    }
  ];

  let continueLoop = true;

  while (continueLoop) {
    const result = await model.generateContent({
      contents: messages,
      tools: [
        {
          functionDeclarations: tools as any
        }
      ]
    });

    const response = result.response;

    if (response.candidates && response.candidates[0]) {
      const candidate = response.candidates[0];

      // 最後のメッセージをhistoryに追加
      if (candidate.content && candidate.content.parts) {
        messages.push({
          role: "model",
          parts: candidate.content.parts
        });

        let hasFunction = false;

        for (const part of candidate.content.parts) {
          if ("functionCall" in part && part.functionCall) {
            hasFunction = true;
            const functionName = part.functionCall.name;
            const args = part.functionCall.args as { [key: string]: string };

            console.log(`Calling: ${functionName}`);
            console.log(`Args:`, args);

            // 関数実行
            const result = await executeFunction(functionName, args);

            // レスポンスを追加
            messages.push({
              role: "user",
              parts: [
                {
                  functionResponse: {
                    name: functionName,
                    response: result
                  }
                }
              ]
            });
          } else if ("text" in part) {
            console.log(`Gemini: ${part.text}`);
            continueLoop = false;
          }
        }

        if (!hasFunction) {
          continueLoop = false;
        }
      }
    }
  }
}

// 使用例
chatWithFunctionCalling("京都と大阪の天気を比較してください");
```

## Automatic Function Calling

複数の関数呼び出しが必要な場合、手動で処理するのは煩雑です。Automatic Function Callingを使用することで、この処理を自動化できます。

### Automatic Function Calling の実装

```python
def automatic_function_calling(user_message: str):
    """自動関数呼び出し対応版"""

    tools = [
        types.Tool(
            function_declarations=[...]  # 関数定義
        )
    ]

    messages = [
        {"role": "user", "parts": [{"text": user_message}]}
    ]

    while True:
        response = client.models.generate_content(
            model="gemini-2.5-pro",
            contents=messages,
            tools=tools,
            config=types.GenerateContentConfig(
                temperature=0.5
            )
        )

        # レスポンスを処理
        assistant_message = {"role": "model", "parts": []}
        tool_calls = []

        for part in response.candidates[0].content.parts:
            if part.function_call:
                tool_calls.append(part.function_call)
                assistant_message["parts"].append(part)
            elif hasattr(part, 'text'):
                assistant_message["parts"].append({"text": part.text})

        messages.append(assistant_message)

        # Function Callがない場合、終了
        if not tool_calls:
            # 最終応答を出力
            if assistant_message["parts"]:
                for part in assistant_message["parts"]:
                    if "text" in part:
                        print(f"Gemini: {part['text']}")
            break

        # すべてのFunction Callを処理
        for tool_call in tool_calls:
            result = process_function_call(
                tool_call.name,
                dict(tool_call.args)
            )
            messages.append({
                "role": "user",
                "parts": [{
                    "function_response": {
                        "name": tool_call.name,
                        "response": result
                    }
                }]
            })

# 使用例
automatic_function_calling("東京と大阪の天気を教えてください")
```
