## Gemini Advanced を使うべき理由

ASO成功の鍵はデータドリブンな意思決定です。App Annie/App Figures のキーワードデータだけでなく、Google Trends による市場トレンド分析を組み合わせることで、より正確なキーワード選定が可能になります。

**Gemini Advanced** の強みは：
- **大規模データ処理**: 複数ソースのデータを統合分析
- **多言語分析**: 30言語以上の自然言語処理
- **市場インサイト抽出**: トレンドデータから戦略的な示唆を抽出
- **カスタムプロンプト**: 特定の市場・言語に最適化された分析

## Google Trends を活用した ASO キーワード調査

### Google Trends で何がわかるか

Google Trends は Google 検索の集計データを提供。ASO にとって重要な情報：

1. **検索トレンド**: キーワードの検索量推移（過去5年）
2. **地域別トレンド**: 国別の検索ボリューム比較
3. **関連キーワード**: 同じカテゴリで検索される用語
4. **季節変動**: 時期ごとの検索パターン

### ASO リサーチでの活用方法

**例：ゲームアプリのキーワード調査**

```
検索キーワード: "puzzle game"
結果:
- 全世界での検索トレンド: 右肩上がり（2021-2026）
- 地域別: 米国（最高）> インド > ブラジル
- 季節: 冬期（12月）が最高、夏期（8月）が最低
- 関連キーワード: match 3, brain game, casual game
```

このデータから以下を推断：
- ゲームアプリは右肩上がり市場（長期投資に値する）
- インド・ブラジル市場の成長性が高い
- 冬期キャンペーンが効果的

## Gemini Advanced での複合データ分析

### Google Trends × App Annie / App Figures の統合分析

Gemini Advanced に以下のデータを入力すると、自動で洞察を抽出：

```markdown

## Input Data

### Google Trends
- 「puzzle game」: 2021年100 → 2026年150（+50%成長）
- 地域別トレンド:
  * 米国: 100
  * インド: 85
  * ブラジル: 72
  * 日本: 45

### App Annie / App Figures
- 「puzzle game」: 月間検索 100,000、難易度 80/100
- 「match 3」: 月間検索 50,000、難易度 70/100
- 「brain training」: 月間検索 30,000、難易度 50/100

## Task
各地域でのキーワード選定戦略と、タイムリーなキャンペーン時期を提案してください
```

**Gemini Advanced の出力例**：

```
1. 米国市場（高難易度、競争激戦）
   推奨キーワード: match-3, brain game, casual game（複合キーワードで差別化）
   推奨時期: 12月初旬（冬休み前）

2. インド市場（成長中、チャンスあり）
   推奨キーワード: free puzzle game, brain training（低コスト訴求）
   推奨時期: 1月（新年目標立てシーズン）

3. ブラジル市場（ポルトガル語で分析必須）
   推奨キーワード: jogo de puzzle, jogo casual
   推奨時期: 7月（冬休み）
```

## Google Play Console データの統合分析

### App のランキング・ダウンロード推移を Gemini で分析

Google Play Console から以下のデータをエクスポート：

```
日付, キーワード, 順位, ダウンロード, CVR
2026-01-01, puzzle game, 150, 500, 2.1%
2026-01-02, puzzle game, 145, 520, 2.2%
2026-01-03, puzzle game, 142, 550, 2.3%
...
```

このデータを Gemini Advanced に入力：

```markdown
# ランキングと成長の相関分析

上記CSVデータから以下を分析してください:
1. キーワード「puzzle game」のランキング改善による
   ダウンロード増加の相関
2. 改善の理由（メタデータ更新、広告キャンペーンなど）
3. 他のキーワードへの応用可能性
```

**分析結果の例**：

```
発見:
- 順位150位 → 142位への改善で、ダウンロード 500 → 550
  （+10%のダウンロード増加）
- CVR も 2.1% → 2.3% に改善

示唆:
- 他の低順位キーワードでも同様の改善が期待できる
- メタデータ最適化の ROI は月 +50 DL / キーワード
```

## 季節トレンド分析と キャンペーン最適化

### Google Trends の季節パターン認識

Gemini Advanced で Google Trends データから季節パターンを抽出：

```markdown
# 季節トレンド分析

Google Trends データ（過去5年）から、各四半期の
検索ボリューム変動を分析してください。

特に以下について:
1. Q4（11-12月）の需要ピークの理由
2. 各地域での季節変動の違い
3. キャンペーン実施タイミングの最適化
```

**分析出力例**:

```
Q1 (1-3月): 10-15%増加
- 新年の目標設定ブーム（特に脳トレ・学習アプリ）
- 推奨: 「brain training」「educational」をテーマに

Q2 (4-6月): 5%減少
- 学校のテスト期間が多い（教育リソース分散）
- 推奨: カジュアルゲーム、リラックステーマで対抗

Q3 (7-9月): 5-10%減少
- 夏休み（ゲーム以外の活動増加）
- 推奨: 「casual」「quick game」で短時間プレイ訴求

Q4 (11-12月): 30-40%増加
- クリスマス、冬休みシーズン
- 推奨: 「gift」「family game」でギフトニーズに対応
```

## 競合分析の高度化

### 上位競合アプリの メタデータ逆算分析

Gemini Advanced は複雑な文字列処理が可能。競合アプリのメタデータを分析：

```markdown
# 競合分析リクエスト

以下の3つの上位アプリのメタデータから、
成功パターンを抽出してください:

【App A - ランク1位】
タイトル: Puzzle Master - Match 3 Games
説明: Award-winning puzzle game with 500+ levels...
キーワード: puzzle game, match 3, brain training, casual games...

【App B - ランク3位】
タイトル: Brain Games - Logic Puzzle Master
説明: Train your brain with addictive puzzles...
キーワード: brain game, puzzle, logic, brain training...

【App C - ランク5位】
タイトル: Match 3 Puzzle Blast
説明: Blast through colorful puzzle levels...
キーワード: match 3, puzzle, casual game...

分析項目:
1. タイトル構成パターン（ブランド + カテゴリ + 説明語）
2. 説明文の開始パターン（感動詞 vs 機能説明）
3. キーワード選定の優先度（ボリューム重視 vs ニッチ重視）
```

**Gemini Advanced の分析結果**:

```
成功パターン:
1. タイトル構成: [カテゴリ] - [特徴] の形式が80%
   例: "Puzzle Master - Match 3 Games"

2. 説明文の開始: 感動詞（Award-winning, Enjoy, Train）
   が最初の単語の75%（ユーザー感情に訴える）

3. キーワード戦略:
   - ランク1位: 一般キーワード70% + ニッチ30%
   - ランク5位: ニッチキーワード80% + 一般20%
   （ランク1位は一般キーワードで大量ダウンロード）

推奨:
- 新規アプリなら「ニッチ重視」でスタート
- 一定ダウンロード後「一般キーワード」に転換
```

## リアルタイム市場インサイト抽出

### 新トレンドの早期発見

Gemini Advanced でリアルタイム検索トレンドを分析：

```markdown
# 新トレンドキーワード検出

Google Trends の過去1ヶ月データで、
前月比 50% 以上伸びているキーワードを抽出し、
ASO戦略への活用を提案してください。

データ:
- 「AI game」: 前月比 +250%
- 「NFT puzzle」: 前月比 +180%
- 「multiplayer puzzle」: 前月比 +45%
```

**出力例**:

```
【高優先度】AI game (+250%)
- トレンド発生: 3ヶ月以上継続の可能性
- 推奨: AI要素を説明文に追加、キーワード化検討

【中優先度】NFT puzzle (+180%)
- トレンド発生: バブル可能性（警戒）
- 推奨: NFT機能がない場合は追加しない

【低優先度】multiplayer puzzle (+45%)
- 既知キーワードの伸長
- 推奨: 既に対応していれば維持
```
