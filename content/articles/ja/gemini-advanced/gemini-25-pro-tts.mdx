## Gemini が音声合成に本格参入

2026年3月、GoogleはGemini 2.5 Pro TTSプレビューを公開しました。これは品質に最適化された新しいText-to-Speech（テキスト読み上げ）モデルで、従来のGoogle Cloud TTSとは一線を画す表現力を持っています。

これまでのTTSモデルが「テキストを音声に変換するツール」だったのに対し、Gemini 2.5 Pro TTSは「テキストの意図を理解して自然に話すAI」です。

## 主な特徴

### 高い表現力（Enhanced Expressivity）

感情やニュアンスを理解し、文脈に応じた抑揚をつけた発話が可能です。ニュースの読み上げ、物語のナレーション、ビジネスプレゼンなど、用途に合わせたトーンが自動的に適用されます。

### 精密なペーシング制御

文章の構造を理解し、適切な位置で間（ま）を入れます。カンマ、ピリオド、段落の切れ目だけでなく、意味的な区切りを認識して自然なリズムで読み上げます。

### シームレスな対話表現

会話文を自然に読み分ける能力が大幅に向上しています。複数の話者がいる対話でも、それぞれのキャラクターに合わせた声色とトーンで読み上げます。

## APIでの使い方

### 基本的なTTSリクエスト

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-2.5-pro-tts-preview")

response = model.generate_content(
    "以下のテキストを自然な日本語で読み上げてください: "
    "本日は晴天なり。散歩に出かけるには絶好の天気です。",
    generation_config={
        "response_modalities": ["audio"],
        "speech_config": {
            "voice_config": {
                "prebuilt_voice_config": {
                    "voice_name": "Kore"
                }
            }
        }
    }
)

# 音声データを保存
with open("output.wav", "wb") as f:
    f.write(response.candidates[0].content.parts[0].inline_data.data)
```

### スタイルの指定

プロンプトでスタイルを指示することで、読み上げのトーンを制御できます。

```python
prompt = """
以下のニュース原稿をプロのアナウンサーのように読み上げてください。
落ち着いた声で、重要な部分は少しゆっくりと:

Google は本日、Gemini 2.5 Pro の新しいTTS機能を発表しました。
この機能により、開発者は高品質な音声合成を
アプリケーションに簡単に組み込むことができます。
"""
```

## 活用シーン

### ポッドキャストの自動生成

ブログ記事やレポートを入力として、ポッドキャスト風の音声コンテンツを自動生成できます。NotebookLMのAudio Overviewと同様のアプローチですが、APIから直接利用できる点が強みです。

### 多言語対応アプリ

日本語、英語、中国語など複数言語での音声出力をアプリに組み込む場合、Gemini TTSは各言語のネイティブ品質で読み上げを行います。

### アクセシビリティ

視覚障害を持つユーザー向けのスクリーンリーダーや、学習支援ツールでの活用が期待されます。従来のTTSと比べて格段に聞き取りやすい音声を提供できます。

## 従来のCloud TTSとの違い

| 機能 | Gemini 2.5 Pro TTS | Cloud TTS |
|------|-------------------|-----------|
| 感情表現 | 文脈理解による自動調整 | SSML タグで手動指定 |
| 対話表現 | 自然な話者分離 | 単一話者 |
| カスタマイズ | プロンプトベース | SSMLベース |
| 価格 | API従量課金 | 文字数ベース |

## 注意点

- 現在はプレビュー版のため、本番利用には注意が必要
- 音声のリアルタイムストリーミングは今後のアップデートで対応予定
- 一部の言語では品質にばらつきがある場合がある
