## Gemini の動画理解能力

Gemini は 1M トークンのコンテキストウィンドウを持つ唯一のモデルファミリーの一つで、**長時間動画の分析**に優れています。1 分間の動画は約 100 フレームに相当し、数十分の動画も一度に処理できます。

主な動画分析ユースケース:
- 動画の要約・テキスト化
- タイムライン・チャプター生成
- 特定シーンの検出と抽出
- 動画内の音声文字起こし
- セキュリティ映像の異常検出

## 動画のアップロードと分析

Gemini API への動画は File API でアップロードしてから分析します。

```python
import google.generativeai as genai
import time

genai.configure(api_key="YOUR_API_KEY")

# 動画ファイルのアップロード
print("動画をアップロード中...")
video_file = genai.upload_file(
    path="meeting_recording.mp4",
    mime_type="video/mp4"
)

# 処理完了を待機
while video_file.state.name == "PROCESSING":
    print(".", end="", flush=True)
    time.sleep(2)
    video_file = genai.get_file(video_file.name)

if video_file.state.name == "FAILED":
    raise ValueError("動画の処理に失敗しました")

print(f"\n処理完了: {video_file.uri}")
```

## 動画の要約を生成

```python
model = genai.GenerativeModel("gemini-3-flash-preview")

# 動画全体の要約
response = model.generate_content([
    video_file,
    "この動画の内容を日本語で要約してください。主要なポイントを箇条書きでまとめてください。"
])

print(response.text)
```

## タイムライン・チャプターの生成

```python
response = model.generate_content([
    video_file,
    """この動画の詳細なタイムラインを作成してください。
    各セクションについて:
    - 開始時間（MM:SS形式）
    - タイトル
    - 30字以内の内容説明
    をJSONで返してください。"""
])

import json
timeline = json.loads(response.text)
for chapter in timeline:
    print(f"{chapter['time']} - {chapter['title']}: {chapter['description']}")
```

## 特定の内容を動画内で検索

```python
response = model.generate_content([
    video_file,
    "この動画の中で、画面にコードが表示されている場面はありますか？あれば、何時何分何秒頃か教えてください。"
])
print(response.text)
```

## 音声文字起こし

```python
response = model.generate_content([
    video_file,
    "この動画の音声を文字起こしして、話者ごとにまとめてください。"
])
print(response.text)
```

## URL から直接分析（YouTube 対応）

File API を使わず、YouTube の URL を直接渡すことも可能です。

```python
response = model.generate_content([
    {
        "file_data": {
            "mime_type": "video/youtube",
            "file_uri": "https://www.youtube.com/watch?v=XXXXXXXXX"
        }
    },
    "この動画の主なトピックと重要なポイントを日本語でまとめてください。"
])
print(response.text)
```

## 処理コストの目安

動画の長さとモデルによって処理コストが変わります。Gemini 3 Flash を使う場合、1 分間の動画はおよそ 3,000〜8,000 トークン相当の処理になります。長時間動画には 3 Flash を、短い高品質分析には 3.1 Pro を使うのが費用対効果的です。

## まとめ

Gemini の Video Understanding 機能を使えば、動画コンテンツの分析・検索・要約を簡単に実装できます。

- File API でアップロード → GenerativeModel で分析
- 要約・タイムライン・文字起こし・シーン検索に対応
- YouTube URL を直接渡す方法も使える
- 長時間動画には Gemini 3 Flash がコスト効率的

セミナー録画の要約、会議議事録の自動生成、動画コンテンツのインデックス化など、幅広いビジネスユースに活用できます。
