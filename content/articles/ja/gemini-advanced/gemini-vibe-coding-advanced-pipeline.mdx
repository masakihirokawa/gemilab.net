## はじめに

バイブコーディングの次のステップは、**開発パイプラインの完全自動化**です。Gemini API と Antigravity エージェントを組み合わせることで、要件定義から本番デプロイまでを自動化できます。

## 自動開発パイプラインのフロー

```
要件入力（自然言語）
  ↓
Gemini API: 要件分析＆デザイン生成
  ↓
Antigravity エージェント: コード生成＆統合
  ↓
CI/CD 自動実行（テスト・ビルド・デプロイ）
  ↓
本番環境デプロイ＆監視
```

## Gemini API での要件分析

```python
from gemini_api import GeminiClient
import os
import json

gemini = GeminiClient(api_key=os.getenv("GEMINI_API_KEY"))

def analyze_requirements(user_input: str) -> dict:
    response = gemini.generate_content(
        model="gemini-2.5-pro",
        prompt=f"要件を分析してJSON形式で返してください: {user_input}",
        temperature=0.3,
        max_tokens=2000
    )
    # JSON抽出して返却
    return json.loads(response.text)
```

## よくある質問（FAQ）

**Q1: 生成コードに問題があった場合？**

A: 「このコードに問題。XXX を修正してください」と指示すれば自動修正。

**Q2: 複数プロジェクト並行実行？**

A: `asyncio.gather()` で並行実行可。Gemini API quota に注意。

**Q3: 既存プロジェクトへの組み込み？**

A: 既存リポジトリ URL を指定でき、新機能だけを生成可能。
