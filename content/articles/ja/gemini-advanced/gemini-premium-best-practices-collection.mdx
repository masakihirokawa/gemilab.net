### Code Execution の安全な実装

**サンドボックス環境で実行し、結果の検証ロジックを後段に必ず配置する**

Code Executionは強力ですが、リスクも伴います。サンドボックス内で実行し、タイムアウトを設定してから、得られた結果に対して検証ロジックを実装することが必須です。

```python
import subprocess
import json

def safe_code_execution(code: str, timeout: int = 5) -> dict:
    """サンドボックスでPythonコードを実行"""
    try:
        result = subprocess.run(
            ["python", "-c", code],
            capture_output=True,
            timeout=timeout,
            text=True
        )

        if result.returncode != 0:
            return {"error": result.stderr, "success": False}

        # 結果の検証
        output = result.stdout.strip()
        if not output:
            return {"error": "Empty output", "success": False}

        return {"output": output, "success": True}
    except subprocess.TimeoutExpired:
        return {"error": "Timeout", "success": False}
```

### Veo 3 動画生成の最適化

**非同期ポーリングは指数バックオフで、プロトタイプは720p、本番は1080pで実施する**

Veo 3の生成は時間がかかるため、ポーリングには指数バックオフを使用し、サーバー負荷を軽減します。開発段階では720p（コスト削減）、本番デプロイ前に1080pで検証することで、効率と品質のバランスが取れます。

```python
import asyncio
import random

async def generate_video_with_backoff(prompt: str, resolution: str = "720p") -> str:
    """指数バックオフでビデオ生成"""

    # ビデオ生成ジョブ開始
    job = await client.video.create(
        model="veo-3",
        prompt=prompt,
        resolution=resolution
    )

    max_retries = 12
    base_delay = 2

    for attempt in range(max_retries):
        await asyncio.sleep(base_delay ** attempt + random.uniform(0, 1))

        status = await client.video.get_status(job.id)
        if status.state == "completed":
            return status.video_url
        elif status.state == "failed":
            raise Exception(f"Video generation failed: {status.error}")

    raise TimeoutError("Video generation timed out")
```

### エンタープライズ AI プラットフォーム設計

**マルチテナント分離を strict に実装し、API Key のローテーション自動化する**

マルチテナント環境では、テナント間のデータ漏洩を防ぐため、厳格に分離する必要があります。API Keyは定期的にローテーションして、漏洩リスクを最小化します。

```python
import secrets
from datetime import datetime, timedelta

class MultiTenantKeyManager:
    def __init__(self, rotation_days: int = 30):
        self.rotation_days = rotation_days
        self.keys = {}

    def create_tenant_key(self, tenant_id: str) -> str:
        """テナント用の新しい API Key を生成"""
        key = f"gml_{secrets.token_urlsafe(32)}"

        self.keys[key] = {
            "tenant_id": tenant_id,
            "created_at": datetime.now(),
            "rotated_at": datetime.now()
        }

        return key

    def rotate_expired_keys(self):
        """期限切れ Key を自動ローテーション"""
        now = datetime.now()

        for key, metadata in list(self.keys.items()):
            age = now - metadata["rotated_at"]
            if age > timedelta(days=self.rotation_days):
                # 古い Key を無効化、新規作成
                tenant_id = metadata["tenant_id"]
                del self.keys[key]
                self.create_tenant_key(tenant_id)
```

## 音声 AI・TTS のベストプラクティス

### Live API でのリアルタイム音声パイプライン

**WebSocket 接続に heartbeat を実装し、音声チャンクを 200ms 単位で送信する**

Live APIでは安定した接続が必須です。定期的にheartbeatを送信してコネクションを保持し、音声チャンクを200ms単位で分割してから送信することで、レイテンシと安定性のバランスが取れます。

```python
import asyncio
import websockets
import json

async def live_audio_stream(audio_stream):
    """Live API で リアルタイム音声処理"""

    uri = "wss://generativelanguage.googleapis.com/google.ai.generativelanguage.v1alpha.GenerativeService/BidiGenerateContent"

    async with websockets.connect(uri) as ws:
        # 接続開始
        await ws.send(json.dumps({"setup": {"model": "models/gemini-2.0-flash"}}))

        heartbeat_task = asyncio.create_task(send_heartbeat(ws))

        try:
            async for chunk in audio_stream:
                # 200ms 単位のチャンク処理
                await ws.send(json.dumps({
                    "realtime_input": {
                        "media_chunks": [{
                            "data": chunk,
                            "mime_type": "audio/pcm"
                        }]
                    }
                }))

                # レスポンス受信
                response = await ws.recv()
                yield json.loads(response)
        finally:
            heartbeat_task.cancel()

async def send_heartbeat(ws, interval: float = 10.0):
    """WebSocket keepalive heartbeat"""
    while True:
        await asyncio.sleep(interval)
        await ws.send(json.dumps({"heartbeat": {}}))
```

### TTS で感情表現と長文処理

**SSML タグで感情を制御し、長文は文単位で分割 → ストリーミング再生**

TTS の自然性はSSML（Speech Synthesis Markup Language）タグで制御できます。長文は文ごとに分割して処理し、ストリーミング再生することで、ユーザー体験が向上します。

def generate_expressive_speech(text: str, emotion: str = "neutral"):
    """感情表現を含む TTS"""

    ssml_markup = f"""<speak>
        <amazon:emotion name="{emotion}" intensity="medium">
            {text}
        </amazon:emotion>
    </speak>"""

    return client.text_to_speech.synthesize(
        ssml=ssml_markup,
        voice_config={"language_code": "ja-JP", "name": "ja-JP-Standard-A"}

import re

def stream_long_text_tts(long_text: str):
    """長文を文単位で分割してストリーミング"""

    # 文単位で分割（。、！、？）
    sentences = re.split(r'[。！？]', long_text)

    for sentence in sentences:
        if sentence.strip():
            audio = generate_expressive_speech(sentence + "。")

### Personal Intelligence でのプライバシー管理

**ユーザー同意を必ず取得し、パーソナライズデータの暗号化を実装**

Personal Intelligence機能を使用する場合、個人データの取扱いが極めて慎重である必要があります。明示的なユーザー同意を取得し、保存するパーソナライズデータはエンドツーエンド暗号化することで、プライバシーリスクが最小化されます。

```python
from cryptography.fernet import Fernet
import json

class PrivacyAwarePersonalIntelligence:
    def __init__(self):
        self.cipher_suite = Fernet(Fernet.generate_key())

    def request_user_consent(self, user_id: str, permissions: list) -> bool:
        """明示的なユーザー同意を取得"""
        consent_request = {
            "user_id": user_id,
            "permissions": permissions,
            "timestamp": datetime.now().isoformat()
        }
        # 同意確認ダイアログを表示
        return True  # ユーザーの明示的な同意

    def encrypt_personal_data(self, data: dict) -> bytes:
        """パーソナルデータを暗号化"""
        json_data = json.dumps(data).encode()
        return self.cipher_suite.encrypt(json_data)

    def decrypt_personal_data(self, encrypted_data: bytes) -> dict:
        """暗号化データを復号"""
        decrypted = self.cipher_suite.decrypt(encrypted_data)
        return json.loads(decrypted.decode())
```
