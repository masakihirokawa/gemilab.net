## Firebase Genkit × Gemini 開発ガイド

**Firebase Genkit**は、Googleが提供する新世代のAI開発フレームワークで、Gemini APIと深く統合されています。Genkitを使用することで、複雑なAIロジックを効率的に実装し、Firebaseのインフラストラクチャ上でスケーラブルにデプロイできます。このガイドでは、Genkitの基礎から実装パターンまで、実践的な内容を網羅します。

## Firebase Genkit の概要

Firebase Genkitは、AI駆動型アプリケーション開発を簡素化するために設計されたTypeScript/Node.js フレームワークです。以下のような特徴を持ちます：

### 主な特徴

- **Gemini統合**: Gemini APIとの連携がデフォルトで組み込まれている
- **Flows**: AI推論、データ取得、外部APIとの連携を統一的に処理
- **Prompts**: テンプレートベースのプロンプト管理で再利用性を向上
- **Firebase統合**: Firestore、Cloud Functions、Cloud Runとのシームレスな連携
- **デバッグツール**: DevUI で実行結果をリアルタイム検査

## Genkitのセットアップ

### インストール

```bash
# Node.js プロジェクトを初期化
npm init -y
npm install -D typescript @types/node ts-node

# Firebase Genkit をインストール
npm install @genkit-ai/core @genkit-ai/firebase
npm install @genkit-ai/gemini
```

### 初期設定

```typescript
// genkit.config.ts
import { defineConfig } from '@genkit-ai/core';
import { firebase } from '@genkit-ai/firebase';
import { googleAI } from '@genkit-ai/gemini';

export default defineConfig({
  plugins: [
    firebase(),
    googleAI({
      apiKey: process.env.GOOGLE_API_KEY,
    }),
  ],
  logLevel: 'debug',
});
```

## Prompts による効率的なプロンプト管理

Genkitのprompts機能を使用することで、プロンプトをコード内に直接記述するのではなく、外部ファイルで管理できます。

### プロンプトファイルの作成

```yaml
# src/prompts/summarizer.prompt.yaml
name: summarizer
description: "テキストを要約します"
model: gemini-1.5-pro
config:
  temperature: 0.5
  maxOutputTokens: 500

input:
  schema:
    type: object
    properties:
      text:
        type: string
        description: "要約対象のテキスト"
      language:
        type: string
        description: "出力言語"

template: |
  あなたは優秀な文章要約エキスパートです。
  以下のテキストを {{language}} で要約してください。
  簡潔で、重要なポイントを逃さないようにしてください。

  テキスト：
  {{text}}
```

### Promptの使用

```typescript
// src/flows/summarize.ts
import { prompt } from '@genkit-ai/core';
import { summarizer } from './prompts';

export const summarizeFlow = prompt(summarizer);
```

## Flows による複雑なロジック実装

Flowsは、複数のAI推論ステップ、データベースアクセス、外部API呼び出しを組み合わせた複合的な処理を実装するための機能です。

### 実装例：ブログ記事生成Flow

以下は、トピックからブログ記事を自動生成するFlowの実装例です：

```typescript
// src/flows/blog_generator.ts
import { defineFlow, streamFlow } from '@genkit-ai/core';
import { gemini } from '@genkit-ai/gemini';
import { firebaseAuth, firestore } from '@genkit-ai/firebase';

interface BlogGenerationInput {
  topic: string;
  language: string;
  sections: number;
}

interface BlogGenerationOutput {
  title: string;
  content: string;
  sections: BlogSection[];
  generatedAt: string;
}

interface BlogSection {
  heading: string;
  content: string;
  wordCount: number;
}

export const generateBlogFlow = defineFlow(
  {
    name: 'generateBlog',
    inputType: 'BlogGenerationInput' as any,
    outputType: 'BlogGenerationOutput' as any,
    description: 'トピックから完全なブログ記事を生成します',
  },
  async (input: BlogGenerationInput): Promise<BlogGenerationOutput> => {
    const db = firestore();

    // ステップ1: タイトルとアウトラインを生成
    const outlineResponse = await gemini.generateContent({
      model: 'gemini-1.5-pro',
      prompt: `
        以下のトピックについて、ブログ記事のタイトルと${input.sections}セクションのアウトラインを${input.language}で作成してください。
        JSON形式で返してください。

        トピック: ${input.topic}

        {
          "title": "記事のタイトル",
          "sections": [
            {"heading": "セクション1", "keyPoints": ["ポイント1", "ポイント2"]},
            ...
          ]
        }
      `,
    });

    const outline = JSON.parse(outlineResponse.text || '{}');

    // ステップ2: 各セクションのコンテンツを生成
    const sections: BlogSection[] = [];

    for (const section of outline.sections) {
      const sectionResponse = await gemini.generateContent({
        model: 'gemini-1.5-pro',
        prompt: `
          以下のセクションについて、詳細で有用なコンテンツを${input.language}で書いてください。
          300-500字程度を目安としてください。

          セクション: ${section.heading}
          キーポイント: ${section.keyPoints.join(', ')}
        `,
      });

      sections.push({
        heading: section.heading,
        content: sectionResponse.text || '',
        wordCount: (sectionResponse.text || '').split(/\s+/).length,
      });
    }

    // ステップ3: データベースに保存
    const docRef = await db.collection('blog_articles').add({
      title: outline.title,
      topic: input.topic,
      language: input.language,
      sections: sections,
      generatedAt: new Date(),
      status: 'published',
    });

    return {
      title: outline.title,
      content: sections.map(s => `## ${s.heading}\n\n${s.content}`).join('\n\n'),
      sections: sections,
      generatedAt: new Date().toISOString(),
    };
  }
);
```

### Flowの実行

```typescript
// 非同期実行
const result = await generateBlogFlow({
  topic: 'Gemini APIの活用例',
  language: '日本語',
  sections: 5,
});

console.log(result.title);
console.log(result.content);

// ストリーミング実行（段階的に結果を取得）
const stream = await streamFlow(generateBlogFlow, {
  topic: 'Firebase Genkitの基礎',
  language: '日本語',
  sections: 3,
});

for await (const chunk of stream) {
  console.log('進捗:', chunk);
}
```

## Cloud Functions での Genkit Flows のデプロイ

Genkitは、Google Cloud Functionsとの統合により、サーバーレスな環境でFlowを実行できます。

### Cloud Function の設定

```typescript
// functions/src/index.ts
import { onRequest } from 'firebase-functions/v2/https';
import { generateBlogFlow } from '../shared/flows/blog_generator';

export const blogGeneratorAPI = onRequest(
  { region: ['us-central1'], cors: true },
  async (req, res) => {
    try {
      const { topic, language, sections } = req.body;

      const result = await generateBlogFlow({
        topic,
        language,
        sections: sections || 5,
      });

      res.json({ success: true, data: result });
    } catch (error) {
      console.error('エラー:', error);
      res.status(500).json({ success: false, error: error.message });
    }
  }
);
```

```bash
firebase deploy --only functions
```

## [Gemini API](/articles/gemini-api/gemini-api-quickstart) との連携

Firebase GenkitはGemini APIを標準でサポートしており、モデル選択から出力制御まで柔軟に設定できます。

```typescript
// 高度なGemini設定の例
export const advancedAnalysisFlow = defineFlow(
  {
    name: 'advancedAnalysis',
    inputType: 'string' as any,
    outputType: 'string' as any,
  },
  async (input: string): Promise<string> => {
    const response = await gemini.generateContent({
      model: 'gemini-2.0-flash',
      prompt: input,
      config: {
        temperature: 0.7,
        topP: 0.95,
        topK: 40,
        maxOutputTokens: 2048,
      },
      safetySettings: [
        {
          category: 'HARM_CATEGORY_HATE_SPEECH',
          threshold: 'BLOCK_MEDIUM_AND_ABOVE',
        },
      ],
    });

    return response.text || '';
  }
);
```

## ベストプラクティス

### エラーハンドリング

```typescript
export const robustFlow = defineFlow(
  { name: 'robustFlow', inputType: 'string' as any },
  async (input: string) => {
    try {
      const response = await gemini.generateContent({
        model: 'gemini-1.5-pro',
        prompt: input,
      });

      if (!response.text) {
        throw new Error('空の応答が返されました');
      }

      return response.text;
    } catch (error) {
      console.error('Gemini API エラー:', error);
      // フォールバック処理またはユーザーへのエラーメッセージ
      throw new Error('処理に失敗しました。後でお試しください。');
    }
  }
);
```

### キャッシング

```typescript
import { cache } from '@genkit-ai/core';

export const cachedFlow = defineFlow(
  { name: 'cachedFlow', inputType: 'string' as any },
  async (input: string) => {
    const cacheKey = `analysis_${input}`;
    const cached = await cache.get(cacheKey);

    if (cached) {
      return cached;
    }

    const result = await gemini.generateContent({
      model: 'gemini-1.5-pro',
      prompt: input,
    });

    await cache.set(cacheKey, result.text, 3600); // 1時間キャッシュ
    return result.text;
  }
);
```

## よくある質問

### Q1: Firebase Genkit は無料で使用できますか？

A: Firebase Genkit自体は無料ですが、Gemini APIの利用には費用が発生します。使用量に応じた従量課金制となっており、公式サイトで料金表を確認できます。

### Q2: Genkit で複数の言語モデルを使い分けることはできますか？

A: はい。Genkitは複数のAIプロバイダー（Gemini、OpenAI、Anthropicなど）をサポートしており、Flow内で複数のモデルを組み合わせることができます。

### Q3: Firestore との統合方法は？

A: Firebase Genkitは@genkit-ai/firebaseプラグインを通じてFirestoreと統合されます。Flow内からfirestore()を呼び出し、通常のFirestoreクエリを実行できます。
