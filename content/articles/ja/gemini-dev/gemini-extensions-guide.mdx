## Gemini Extensions とは

Gemini Extensionsは、Geminiが外部のサービスやデータにアクセスし、より正確で最新の情報を提供できるようにする仕組みです。標準で利用できるExtensionsに加え、開発者がカスタムExtensionを構築することも可能です。

## 利用可能な標準Extensions

### Google Search

最新の情報を検索し、回答に反映します。ニュースや時事情報、リアルタイムデータが必要な質問に自動的に検索が適用されます。

**活用例:**
- 「今日の東京の天気は？」
- 「直近のAI関連ニュースをまとめて」
- 「〇〇社の最新の決算情報は？」

### Google Maps

場所の検索、経路案内、レストランやホテルの情報を取得します。

**活用例:**
- 「渋谷駅から東京タワーまでの行き方」
- 「この周辺でおすすめのイタリアンレストラン」
- 「営業時間とレビュー評価を教えて」

### YouTube

YouTube上の動画を検索し、関連コンテンツを提案します。

**活用例:**
- 「Pythonの機械学習チュートリアル動画を探して」
- 「この曲のミュージックビデオを見つけて」

### Google Flights & Hotels

フライトやホテルの検索・比較ができます。

**活用例:**
- 「来月の東京〜パリの最安フライト」
- 「バルセロナの4つ星ホテルを比較して」

## Extensionsの有効化

1. Geminiの設定画面を開く
2. 「Extensions」セクションに移動
3. 使用したいExtensionをオンにする

各Extensionは個別にオン/オフを切り替えられます。

## API での Tool 利用

開発者はGemini APIで `tools` パラメータを使って、同様の機能をプログラマティックに利用できます。

```python
import google.genai as genai

client = genai.Client(api_key="YOUR_API_KEY")

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents="東京の現在の天気を教えてください",
    config={
        "tools": [{"google_search": {}}]
    }
)

# Grounding metadata を確認
for candidate in response.candidates:
    if candidate.grounding_metadata:
        for chunk in candidate.grounding_metadata.grounding_chunks:
            print(f"Source: {chunk.web.uri}")
```

## カスタム Function Calling

標準Extensionsだけでなく、独自の関数を定義してGeminiから呼び出すことも可能です。

```python
# 関数の定義
get_weather = {
    "name": "get_weather",
    "description": "指定した都市の現在の天気を取得する",
    "parameters": {
        "type": "object",
        "properties": {
            "city": {"type": "string", "description": "都市名"},
            "unit": {"type": "string", "enum": ["celsius", "fahrenheit"]}
        },
        "required": ["city"]
    }
}

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents="東京の天気は？",
    config={
        "tools": [{"function_declarations": [get_weather]}]
    }
)
```

## ベストプラクティス

- **必要なExtensionsだけ有効にする**: 不要なExtensionsが有効だと、意図しない外部呼び出しが発生する可能性があります
- **Grounding情報の活用**: 検索結果のソースURLを確認して、情報の信頼性を検証しましょう
- **レイテンシの考慮**: Extension呼び出しはネットワークアクセスを伴うため、レスポンス時間が増加します
- **コスト管理**: API経由でのGrounding検索には追加コストがかかる場合があります
