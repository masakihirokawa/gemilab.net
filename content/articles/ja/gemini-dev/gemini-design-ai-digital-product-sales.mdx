## はじめに — デザインの壁を超える

[前編](/articles/gemini-dev/gemini-multi-channel-monetization-basics)では、Geminiを活用した5つの収益化チャネルを紹介しました。後編では、デジタルプロダクトの**制作から販売まで**を具体的に実践します。

「デザインスキルがないから売れるものが作れない」と感じている方も多いかもしれません。しかし、GeminiのコンテンツGen能力とCanva・Figmaのデザイン機能を組み合わせれば、プロ品質のプロダクトを個人で制作できます。

## 売れるデジタルプロダクトの4タイプ

### タイプ1: テンプレート集

ビジネス文書、SNS投稿、プレゼン資料などのテンプレートは、安定した需要があります。Geminiでコンテンツ構成を設計し、CanvaやFigmaでビジュアルを仕上げます。

### タイプ2: ガイドブック・ebook

特定のテーマに絞ったガイドブックは、Gemini Deep Researchの調査力を最大限に活かせるジャンルです。

### タイプ3: チェックリスト・ワークシート

実用的なチェックリストやワークシートは制作が比較的簡単で、初心者の第一歩に最適です。

### タイプ4: デザイン素材パック

アイコン、バナー、SNS画像のテンプレートなど、Canvaで制作したデザイン素材を販売します。

## 実践: テンプレート集の制作フロー

### Step 1: 市場リサーチ（Gemini Deep Research）

```javascript
// Gemini APIで市場リサーチを行う例
import { GoogleGenAI } from "@google/genai";

const ai = new GoogleGenAI({ apiKey: process.env.GEMINI_API_KEY });

const response = await ai.models.generateContent({
  model: "gemini-2.5-flash",
  contents: `デジタルプロダクト市場のリサーチを行ってください：
1. Gumroadで売れているNotionテンプレートの上位5ジャンル
2. 各ジャンルの平均価格帯
3. 差別化できそうなニッチな切り口を3つ提案
4. 初心者が参入しやすい順にランキング`,
});

console.log(response.text);
// → 市場データと参入戦略が出力される
```

### Step 2: コンテンツ設計（Gemini）

リサーチ結果をもとに、テンプレートの構成をGeminiで設計します。セクション構成、フィールド定義、使用方法の説明文まで一括で生成できます。

### Step 3: デザイン制作（Canva / Figma）

Geminiが生成した構成案をCanvaやFigmaに落とし込みます。

**Canvaの場合**: テンプレートを選んでカスタマイズするだけで、統一感のあるデザインが完成します。ブランドカラーやフォントを設定しておくと、シリーズ化する際にも一貫性を保てます。

**Figmaの場合**: より高度なカスタマイズが可能です。Auto LayoutやComponents機能を使えば、テンプレートの品質をさらに高められます。

### Step 4: 販売ページの作成

Stitchを使えば、AIが自動で販売ページを生成してくれます。商品説明、特徴の箇条書き、FAQ、購入ボタンまでを含むLPを数分で構築可能です。

### Step 5: Stripeで決済を設定

自前のサイトで販売する場合は、Stripe決済の設定が必要です。

```javascript
// Stripe決済リンクを作成する例
import Stripe from "stripe";

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY);

// 商品を作成
const product = await stripe.products.create({
  name: "フリーランス向け案件管理テンプレート集",
  description: "Notionベースの案件管理テンプレート10種セット",
});

// 価格を設定
const price = await stripe.prices.create({
  product: product.id,
  unit_amount: 1480, // ¥1,480
  currency: "jpy",
});

// 決済リンクを生成
const paymentLink = await stripe.paymentLinks.create({
  line_items: [{ price: price.id, quantity: 1 }],
});

console.log(`決済リンク: ${paymentLink.url}`);
// → https://buy.stripe.com/xxx のURLが生成される
```

## 価格設定の戦略

デジタルプロダクトの価格設定は収益を大きく左右します。

**エントリープロダクト（¥500〜1,500）** — 初めての購入者に向けた低価格帯。テンプレート単品やチェックリストに適しています。まずは購入体験を提供し、信頼を構築します。

**メインプロダクト（¥1,500〜5,000）** — テンプレート集やガイドブックなど、まとまった価値を提供するプロダクト。収益の柱になる価格帯です。

**プレミアムプロダクト（¥5,000〜15,000）** — 動画解説付きガイド、コンサルティング付きテンプレートなど、付加価値の高いプロダクト。少数でも大きな収益を生みます。

## 販売を加速するマーケティング

### noteでの発信

Geminiで記事を執筆し、noteで有料・無料のコンテンツを公開することで、プロダクトへの導線を作れます。無料記事でノウハウの一部を公開し、詳細はプロダクトで提供する形が効果的です。

### SNSでの告知

InstagramやTwitterで、プロダクトのプレビュー画像や使用例を定期的に投稿します。Canvaで見栄えの良い画像を作り、Geminiでキャプションを生成する流れが効率的です。

### メールマーケティング

購入者リストを構築し、新商品のリリースやアップデートを通知します。Geminiでメール文面を生成し、効果的なマーケティングオートメーションを構築しましょう。

## よくある質問

### Q: デザインスキルがなくても売れるプロダクトは作れますか？

はい。Canvaにはプロがデザインしたテンプレートが豊富に用意されています。色やテキストを変えるだけでオリジナルの商品が作れます。Geminiで構成案を考え、Canvaでデザインすれば、デザインスキルなしでも十分にプロ品質のプロダクトが完成します。

### Q: Gumroad、STORES、BASEのどれで販売するのがいいですか？

海外向けにも販売するならGumroad、日本市場メインならBASEやSTORESがおすすめです。Gumroadは手数料が低く、デジタルプロダクトに特化しています。まずは1つのプラットフォームで始めて、売上が安定してから拡大するのが効率的です。

### Q: Stripeの設定は難しいですか？

基本的な決済リンクの作成は非常に簡単です。Stripeダッシュボードから数クリックで設定でき、プログラミング不要です。APIを使ったカスタマイズをしたい場合も、Geminiにコードの書き方を相談できます。

### Q: 最初に作るプロダクトとしておすすめは何ですか？

チェックリストやワークシートがおすすめです。制作時間が短く、需要も安定しているため、最初の成功体験を得やすいジャンルです。まずは1つ完成させて販売し、フィードバックを集めてから次のプロダクトに進みましょう。
