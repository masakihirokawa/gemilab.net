## 概要

2026年3月、GoogleはAI分野における重大な技術革新を実現しました。**gemini-embedding-2-preview** — テキスト、画像、動画、音声、PDFを統一ベクトル空間にマッピングする史上初のマルチモーダルエンベディングモデルの発表です。

これまでのエンベディング技術は単一モダリティ（テキストのみなど）に特化していました。しかし現実世界のデータは多様な形式で存在します。gemini-embedding-2-previewはこの課題を解決し、すべてのモダリティを同じベクトル空間に投影することで、真の意味でのクロスモーダル検索と推論が可能になります。

この完全ガイドでは、gemini-embedding-2-previewの技術仕様から実装まで、あらゆる側面を詳しく解説します。

## マルチモーダルエンベディングとは

### 従来のエンベディングの限界

従来のテキストエンベディングモデルは、文字列をベクトル空間に変換するだけでした：

```
テキスト → エンコーダー → ベクトル [0.234, -0.512, 0.789, ...]
```

このアプローチの問題点：
- 画像や動画を直接エンベディングできない
- 異なるモダリティ間の関係性を捉えられない
- 「猫の動画」と「猫の画像」が意味的に関連していることを理解できない

### マルチモーダルエンベディングの革新性

gemini-embedding-2-previewは、複数のモダリティを**同じベクトル空間**に投影します：

```
テキスト        ┐
画像           ├→ マルチモーダル ├→ 統一ベクトル空間
動画           │  エンコーダー   │   [0.234, -0.512, 0.789, ...]
音声           │                │
PDF            ┘                ↓
              （すべてのベクトルは
               相互比較可能）
```

この設計により、次のような革新的な能力が実現します：

1. **クロスモーダル検索**：テキストクエリで画像を検索、画像クエリで動画を検索
2. **マルチモーダルRAG**：ドキュメント、画像、動画を統一されたベクトルベースで検索
3. **セマンティック類似性**：異なるモダリティ間の意味的な関連性を発見

## 統一ベクトル空間の理解

### 数学的背景

gemini-embedding-2-previewの核心は、複数のモダリティエンコーダーの出力を単一のベクトル空間に正規化することです：

```
モダリティ → 専門エンコーダー → 中間表現 → 正規化 → 統一空間ベクトル
  (テキスト)   (テキスト化)    (異なる次元数)  (768次元)
  (画像)      (視覚特徴抽出)   (異なる次元数)
  (動画)      (時系列特徴)     (異なる次元数)
```

### コサイン類似度による検索

統一されたベクトル空間では、すべてのベクトルがコサイン類似度により比較可能です：

```python
# 類似度スコア (0から1)
similarity = (embedding1 · embedding2) / (||embedding1|| × ||embedding2||)

# スコア解釈
# 0.95+ : ほぼ同じコンテンツ
# 0.80-0.95 : 強く関連
# 0.60-0.80 : 関連あり
# 0.40-0.60 : 弱く関連
# 0.00-0.40 : 関連なし
```

### 実際の動作例

```
入力：テキスト「黄色いラブラドール犬が公園で遊んでいる」
↓
出力：[0.234, -0.512, 0.789, ..., 0.156]（768次元ベクトル）

同じベクトル空間で：
✓ 黄色いラブラドールの動画: 類似度 0.92
✓ 犬が走っている画像: 類似度 0.88
✓ 公園のビデオ: 類似度 0.76
✓ 赤いシベリアンハスキーの画像: 類似度 0.64
✗ 猫の画像: 類似度 0.23
```

import base64
from pathlib import Path

def embed_image(image_path: str):
    """画像ファイルをエンベディング"""
    with open(image_path, 'rb') as f:
        image_data = base64.standard_b64encode(f.read()).decode('utf-8')

    # ファイルタイプを判定
    suffix = Path(image_path).suffix.lower()
    media_type_map = {
        '.jpg': 'image/jpeg',
        '.jpeg': 'image/jpeg',
        '.png': 'image/png',
        '.webp': 'image/webp',
        '.heic': 'image/heic'

    embedding = genai.embed_content(
        model="models/embedding-2-preview",
        content={
            "mime_type": media_type,
            "data": image_data
        }
    )

    return embedding['embedding']

### クロスモーダル類似度計算

```python
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

def calculate_similarity(embedding1, embedding2):
    """2つのベクトル間のコサイン類似度を計算"""
    embedding1 = np.array(embedding1).reshape(1, -1)
    embedding2 = np.array(embedding2).reshape(1, -1)
    return cosine_similarity(embedding1, embedding2)[0][0]

# テキストと画像の類似度を計算
text_query = "黄色い犬が公園で走っている"
text_query_embedding = genai.embed_content(
    model="models/embedding-2-preview",
    content=text_query
)['embedding']

image_embedding = embed_image("dog_in_park.jpg")

similarity = calculate_similarity(
    text_query_embedding,
    image_embedding
)
print(f"テキストと画像の類似度: {similarity:.3f}")
```

## 実践的ユースケース

### 1. クロスモーダル検索システム

```python
import json
from typing import List, Dict

class MultimodalSearchEngine:
    """マルチモーダル検索エンジン"""

    def __init__(self, model_name: str = "models/embedding-2-preview"):
        self.model_name = model_name
        self.indexed_items = []  # 本番はベクトルDBを使用

    def index_content(self, items: List[Dict]):
        """複数のコンテンツアイテムにインデックスを付与"""
        for item in items:
            content_type = item['type']  # 'text', 'image', 'video', etc.
            content_value = item['value']

            if content_type == 'text':
                embedding = genai.embed_content(
                    model=self.model_name,
                    content=content_value,
                    task_type="retrieval_document"
                )['embedding']
            elif content_type == 'image':
                with open(content_value, 'rb') as f:
                    img_data = base64.standard_b64encode(f.read()).decode()
                embedding = genai.embed_content(
                    model=self.model_name,
                    content={"mime_type": "image/jpeg", "data": img_data}
                )['embedding']

            self.indexed_items.append({
                'content': item,
                'embedding': embedding
            })

    def search(self, query: str, query_type: str = 'text', top_k: int = 5):
        """テキスト、画像、動画でクロスモーダル検索"""
        if query_type == 'text':
            query_embedding = genai.embed_content(
                model=self.model_name,
                content=query,
                task_type="retrieval_query"
            )['embedding']

        # 類似度スコアを計算して上位K件をソート
        scores = []
        for item in self.indexed_items:
            similarity = calculate_similarity(
                query_embedding,
                item['embedding']
            )
            scores.append({
                'item': item['content'],
                'score': similarity
            })

        return sorted(scores, key=lambda x: x['score'], reverse=True)[:top_k]

# 使用例
search_engine = MultimodalSearchEngine()
results = search_engine.search("犬が走っている", query_type='text', top_k=5)
```

### 2. 画像を含むRAGシステム

```python
class MultimodalRAG:
    """マルチモーダルドキュメント検索と生成"""

    def __init__(self):
        self.documents = []
        self.embeddings = []

    def add_document(self, doc_content: str, doc_metadata: Dict):
        """ドキュメント（テキスト＋画像）を追加"""
        embedding = genai.embed_content(
            model="models/embedding-2-preview",
            content=doc_content,
            task_type="retrieval_document"
        )['embedding']

        self.documents.append({
            'content': doc_content,
            'metadata': doc_metadata
        })
        self.embeddings.append(embedding)

    def retrieve(self, query: str, top_k: int = 3) -> List[Dict]:
        """関連ドキュメントを取得"""
        query_embedding = genai.embed_content(
            model="models/embedding-2-preview",
            content=query,
            task_type="retrieval_query"
        )['embedding']

        similarities = []
        for idx, doc_embedding in enumerate(self.embeddings):
            sim = calculate_similarity(query_embedding, doc_embedding)
            similarities.append((idx, sim))

        top_indices = sorted(similarities, key=lambda x: x[1], reverse=True)[:top_k]
        return [self.documents[idx] for idx, _ in top_indices]

    def generate_response(self, query: str):
        """取得したドキュメントに基づいて回答生成"""
        retrieved_docs = self.retrieve(query)

        context = "\n".join([
            f"ドキュメント {i+1}:\n{doc['content'][:500]}"
            for i, doc in enumerate(retrieved_docs)
        ])

        response = genai.generate_content(
            f"""以下の関連ドキュメントに基づいて、質問に答えてください。

【関連ドキュメント】
{context}

【質問】
{query}

【回答】"""
        )

        return response.text
```

                'similarity_score': score

def setup_pinecone_with_multimodal(api_key: str, index_name: str):

    vectors = []

    for item in items:
        if item['type'] == 'text':
            embedding = genai.embed_content(
                model="models/embedding-2-preview",
                content=item['content'],
                task_type="retrieval_document"
            )['embedding']
        elif item['type'] == 'image':
            with open(item['content'], 'rb') as f:
                img_data = base64.standard_b64encode(f.read()).decode()
            embedding = genai.embed_content(
                model="models/embedding-2-preview",
                content={"mime_type": "image/jpeg", "data": img_data}
            )['embedding']

        vectors.append((
            item['id'],
            embedding,
            {"type": item['type'], "source": item['source']}
        ))

    index.upsert(vectors=vectors)

## RAGパイプラインアーキテクチャの変更

### 従来のテキストのみRAG

```
ドキュメント → テキスト抽出 → テキストエンベディング → ベクトルDB
                                      ↓
クエリ → テキストエンベディング → ベクトル検索 → 上位K件取得 → LLMで生成
```

### 新しいマルチモーダルRAG

```
ドキュメント群（テキスト、画像、動画）
    ↓
マルチモーダルエンベディング（統一ベクトル空間）
    ↓
ハイブリッドベクトルDB（メタデータ付き）
    ↓
    ┌─────────────────────────┐
    │ 多様なクエリ形式をサポート │
    │ （テキスト/画像/動画）   │
    └─────────────────────────┘
    ↓
マルチモーダルコンテンツ検索
    ↓
    ┌─────────────────────────────────────┐
    │ 関連するすべてのモダリティを取得      │
    │ （テキスト + 画像 + 動画チャンク）  │
    └─────────────────────────────────────┘
    ↓
LLM（+ビジョン機能）で総合的に生成
    ↓
リッチな回答（テキスト + 画像参照 + 動画リンク）
```

### 実装例

```python
class AdvancedMultimodalRAG:
    """次世代型マルチモーダルRAG"""

    def __init__(self, vector_db_client):
        self.vector_db = vector_db_client
        self.model_name = "models/embedding-2-preview"

    def process_multimodal_document(self, doc_path: str):
        """複合ドキュメント（テキスト+画像）を処理"""
        chunks = self.extract_chunks(doc_path)
        processed_chunks = []

        for chunk in chunks:
            if chunk['type'] == 'text':
                embedding = self.embed_text(chunk['content'])
            elif chunk['type'] == 'image':
                embedding = self.embed_image(chunk['path'])

            processed_chunks.append({
                'content': chunk,
                'embedding': embedding,
                'modality': chunk['type']
            })

        return processed_chunks

    def multimodal_query(self, query: str, query_type: str = 'text'):
        """クエリのモダリティに応じて検索"""
        if query_type == 'text':
            query_embedding = genai.embed_content(
                model=self.model_name,
                content=query,
                task_type="retrieval_query"
            )['embedding']

        # ベクトルDBから関連チャンクを取得
        results = self.vector_db.query(
            query_embedding,
            top_k=10,
            include_metadata=True
        )

        # モダリティごとに結果を整理
        multimodal_context = self.organize_by_modality(results)

        return multimodal_context

    def generate_enriched_response(self, query: str):
        """マルチモーダルコンテキストから豊富な回答を生成"""
        context = self.multimodal_query(query)

        # テキストコンテキストを準備
        text_context = "\n".join([
            chunk['content']
            for chunk in context.get('text', [])[:3]
        ])

        # LLMで生成（ビジョン対応）
        response = genai.generate_content([
            f"以下の関連情報に基づいて回答してください：\n{text_context}\n\n質問：{query}",
            # 画像があれば追加
            *[{"mime_type": "image/jpeg", "data": img}
              for img in context.get('images', [])[:2]]
        ])

        return response.text
```
