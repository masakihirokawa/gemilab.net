## マルチモーダルとは

Geminiの最大の強みの一つが、テキストだけでなく画像、動画、音声、PDFなど複数の入力形式を同時に処理できるマルチモーダル能力です。この記事では、Gemini APIを使ったマルチモーダル処理の実装方法を解説します。

## 対応するメディアタイプ

| メディア | 形式 | 最大サイズ |
|---|---|---|
| 画像 | PNG, JPEG, WEBP, GIF | 20MB |
| 動画 | MP4, MPEG, MOV, AVI | 2GB |
| 音声 | MP3, WAV, AIFF, FLAC | 最大約9.5時間 |
| ドキュメント | PDF | 最大300ページ |

## 画像を使った例

### Base64エンコードで送信

```python
import google.genai as genai
import base64

client = genai.Client(api_key="YOUR_API_KEY")

with open("photo.jpg", "rb") as f:
    image_data = base64.b64encode(f.read()).decode("utf-8")

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[
        {"text": "この画像に写っているものを説明してください"},
        {
            "inline_data": {
                "mime_type": "image/jpeg",
                "data": image_data
            }
        }
    ]
)

print(response.text)
```

### URLから直接指定

Google Cloud Storageに保存した画像はURIで直接指定できます。

```python
response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[
        {"text": "この画像を分析してください"},
        {
            "file_data": {
                "mime_type": "image/jpeg",
                "file_uri": "gs://your-bucket/image.jpg"
            }
        }
    ]
)
```

## 動画分析

Geminiは動画の内容を理解し、要約や質問応答ができます。

```python
# File APIを使って動画をアップロード
video_file = client.files.upload(
    file="presentation.mp4",
    config={"display_name": "My Presentation"}
)

# アップロード完了を待機
import time
while video_file.state.name == "PROCESSING":
    time.sleep(5)
    video_file = client.files.get(name=video_file.name)

# 動画について質問
response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[
        {"text": "この動画のキーポイントを5つ挙げてください"},
        {"file_data": {"file_uri": video_file.uri, "mime_type": "video/mp4"}}
    ]
)
```

## 音声処理

音声ファイルの文字起こしや要約も可能です。

```python
audio_file = client.files.upload(file="meeting.mp3")

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[
        {"text": "この会議の音声を文字起こしし、決定事項をリストアップしてください"},
        {"file_data": {"file_uri": audio_file.uri, "mime_type": "audio/mp3"}}
    ]
)
```

## PDF分析

PDFドキュメントをアップロードして、内容の要約や質問応答ができます。

```python
pdf_file = client.files.upload(file="report.pdf")

response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[
        {"text": "このレポートの主要な発見と推奨事項を日本語で要約してください"},
        {"file_data": {"file_uri": pdf_file.uri, "mime_type": "application/pdf"}}
    ]
)
```

## 複数メディアの組み合わせ

Geminiの真価は、複数の種類のメディアを同時に処理できる点にあります。

```python
response = client.models.generate_content(
    model="gemini-2.5-pro",
    contents=[
        {"text": "この画像とPDFの内容を比較して、相違点を教えてください"},
        {"file_data": {"file_uri": image_file.uri, "mime_type": "image/png"}},
        {"file_data": {"file_uri": pdf_file.uri, "mime_type": "application/pdf"}}
    ]
)
```

## ベストプラクティス

- **モデル選択**: 高精度が必要な分析は `gemini-2.5-pro`、速度重視なら `gemini-2.5-flash`
- **ファイルサイズ**: 大きなファイルは File API 経由でアップロード、小さなファイルは inline_data で直接送信
- **プロンプト設計**: マルチモーダル入力でも、具体的な指示を出すことで精度が大幅に向上
- **コスト管理**: 動画や音声はトークン消費が多いため、必要な部分だけを切り出して送信
