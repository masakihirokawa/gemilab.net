## はじめに

テキストを自然な音声に変換する機能は、アプリケーションのアクセシビリティを大幅に向上させます。GoogleのGemini TTS APIは、最新の音声合成技術を活用し、複数の言語で自然で表現力のある音声を生成できます。

本ガイドでは、Gemini TTS APIの基本から実装方法、応用的な活用法まで、実践的なコード例を交えながら詳しく解説します。

## Gemini TTS APIとは

Gemini TTS APIは、テキストをMP3またはその他のフォーマットの音声ファイルに変換するGoogle提供のAPIです。以下の特徴があります：

- **自然な音声**: 最新の音声合成モデルにより、人間らしい発音を実現
- **複数の音声スタイル**: 異なるトーンと性別の音声オプションから選択可能
- **ストリーミング対応**: リアルタイム音声生成に対応
- **日本語対応**: 日本語を含む複数言語をネイティブに処理

## セットアップと環境準備

## 基本的な使い方

### Pythonでのシンプルな実装

```python
import google.generativeai as genai
import os

# APIキーの設定
genai.configure(api_key=os.environ.get("GOOGLE_API_KEY"))

# テキストを音声に変換
text = "こんにちは。Gemini TTS APIについて解説します。"

# TTS APIを使用
response = genai.generate_speech(
    model="models/gemini-1.5-flash",
    text=text,
    voice="ja-JP-Neural2-C"  # 日本語女性音声
)

# 音声ファイルを保存
with open("output.mp3", "wb") as f:
    f.write(response.audio_data)

print("✅ 音声ファイルが生成されました: output.mp3")
```

**期待される出力:**
- `output.mp3` ファイルが生成され、自然な日本語音声を再生可能

### JavaScriptでの実装

```javascript
const genai = require('@google/generative-ai');

const client = new genai.GoogleGenerativeAI(
  process.env.GOOGLE_API_KEY
);

async function textToSpeech() {
  const text = "Gemini TTS API is a powerful tool for speech generation.";
  
  try {
    const response = await client.generateSpeech({
      model: "models/gemini-1.5-flash",
      input: { text: text },
      voice: {
        languageCode: "en-US",
        name: "en-US-Neural2-C"
      },
      audioConfig: {
        audioEncoding: "MP3"
      }
    });

    const fs = require('fs');
    fs.writeFileSync('output.mp3', response.audioContent);
    console.log('✅ Audio file generated: output.mp3');
  } catch (error) {
    console.error('❌ Error:', error);
  }
}

textToSpeech();
```

**期待される出力:**
- `output.mp3` ファイルが生成され、英語の自然な音声を再生可能

## 音声スタイルの制御

Gemini TTS APIは複数の音声パラメータをサポートしており、スピーキングスタイルを細かく制御できます：

**日本語の音声オプション:**
- `ja-JP-Neural2-A` (男性、自然)
- `ja-JP-Neural2-B` (男性、少し高い)
- `ja-JP-Neural2-C` (女性、自然)
- `ja-JP-Neural2-D` (女性、少し高い)

**英語の音声オプション:**
- `en-US-Neural2-A` (男性)
- `en-US-Neural2-C` (女性)
- `en-US-Neural2-E` (男性、別のバリエーション)
- `en-US-Neural2-F` (女性、別のバリエーション)

## 実装パターン

### パターン1: Webアプリケーションへの統合

```python
from flask import Flask, request, send_file
import google.generativeai as genai
import io

app = Flask(__name__)
genai.configure(api_key=os.environ.get("GOOGLE_API_KEY"))

@app.route('/generate-speech', methods=['POST'])
def generate_speech():
    data = request.json
    text = data.get('text', '')
    voice = data.get('voice', 'ja-JP-Neural2-C')
    
    try:
        response = genai.generate_speech(
            model="models/gemini-1.5-flash",
            text=text,
            voice=voice
        )
        
        audio_file = io.BytesIO(response.audio_data)
        audio_file.seek(0)
        
        return send_file(
            audio_file,
            mimetype='audio/mpeg',
            as_attachment=True,
            download_name='speech.mp3'
        )
    except Exception as e:
        return {'error': str(e)}, 400

if __name__ == '__main__':
    app.run(debug=True)
```

### パターン2: バッチ処理による複数テキストの音声化

```python
import time

texts = [
    "最初のテキストです。",
    "次のテキストです。",
    "最後のテキストです。"
]

for i, text in enumerate(texts):
    try:
        response = genai.generate_speech(
            model="models/gemini-1.5-flash",
            text=text,
            voice="ja-JP-Neural2-C"
        )
        
        filename = f"batch_{i+1:03d}.mp3"
        with open(filename, "wb") as f:
            f.write(response.audio_data)
        
        print(f"✅ 生成: {filename}")
        time.sleep(1)  # API制限を考慮
    except Exception as e:
        print(f"❌ エラー (テキスト{i+1}): {e}")
```

## よくあるエラーと対処法

### エラー1: APIキーが無効

```
Error: 403 Forbidden - Invalid API key
```

**対処法:**
- APIキーが正しく設定されているか確認
- Google AI Studioで新しいキーを再生成
- 環境変数が正しく読み込まれているか確認

### エラー2: テキストが長すぎる

```
Error: 400 Bad Request - Input text exceeds maximum length
```

**対処法:**
- テキストを分割して複数回のAPIコールに分ける
- テキストの上限は通常5,000文字程度

### エラー3: 音声フォーマットが対応していない

```
Error: 400 Bad Request - Audio encoding not supported
```

**対処法:**
- MP3フォーマットを使用（最も安定）
- 必要に応じてffmpegで変換

## FAQ

### Q1: Gemini TTS APIの利用料金は？

Gemini TTS APIの利用料金は、Google AI Studio経由では通常無料（試験段階）で、Vertex AI経由では従量課金となります。詳細は公式ドキュメントを確認してください。

### Q2: 生成した音声をどのくらい保存できますか？

生成した音声ファイルはローカルに保存できますが、著作権に配慮が必要です。商用利用の場合はGoogleの利用規約を確認してください。

### Q3: オフライン環境で使用できますか？

いいえ、Gemini TTS APIはオンラインが必須です。ただし、一度生成した音声ファイルはオフラインで再生できます。

### Q4: リアルタイムで音声生成できますか？

API応答時間は通常1〜3秒程度です。ライブストリーミングのような極めてリアルタイムな用途には向きませんが、チャットボットやボイスアシスタントには十分対応できます。
