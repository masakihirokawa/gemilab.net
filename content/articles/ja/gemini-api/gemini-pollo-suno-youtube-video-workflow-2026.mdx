```

条件：
1. 再生数 100万回以上のチャンネルが少ない（競争少ない）

   25%以上増加している

4. RPM（1000回再生あたりの収入）が ¥1,500以上
   （テック系は$20-30が相場）

5つのテーマを提案し、各テーマについて：
- 推定月間検索ボリューム（YouTube Search）
```

## 第3章：Pollo AI による映像制作（複数パターン並列）

### 3.1 Pollo AI の賢い使い分け

**Pollo AI の複数機能を目的別に使い分け：**

```
【機能別活用マトリックス】

テキスト→動画（T2V）：
- 用途：スライドショー的な背景映像
- 例：「営業メール作成画面」「Gemini インターフェース」
- 推奨時間：3-5分間分を生成

画像→動画（I2V）：
- 用途：静止画を動画化（スライドショーより自然）
- 例：あなた自身の実写、オフィス風景
- 推奨時間：各素材から30秒-1分

Pollo Canvas：
- 用途：複数の要素を組み合わせた本格編集
- 例：背景 + テキスト + グラフ + トランジション
- 推奨所要時間：30-45分の編集

AI Viral Agent：
- 用途：1つの動画から複数バリエーション生成
- 例：YouTubeショーツ用、TikTok用、LinkedIn用
- 推奨タイミング：本編集完了後、別プラットフォーム最適化
```

### 3.2 Pollo での映像生成スケジュール

**前述のスクリプト4本に対応した映像生成：**

```
【Day 1：背景映像の一括生成】

テーマ1用：
Prompt1：「Professional businessman at corporate desk,
working on computer, typing email, modern office with plants」

Prompt2：「Close-up of email inbox interface on screen,
Gmail or similar, highlighted important emails」

Prompt3：「Split screen: before (stressed) vs after (happy)
using AI to manage emails」

※ 各テーマで 3-5 本の異なるシーン用途で生成
合計：約12本の背景映像（各1分）を生成

【Day 2：Pollo Canvas での組み立て】
- 各スクリプトに対応する背景映像を時系列で配置
- テキストオーバーレイを追加
- エフェクト・トランジション（200+から選択）を挿入
- 所要時間：各動画 30-40分 × 4本 = 2時間20分
```

### 3.3 Pollo の出力後の必須編集（人間）

**品質向上のための最終編集：**

```
【チェックリスト】
□ 映像がナレーションと適切に同期しているか
□ 背景が「AI っぽい」不自然さがないか
  （違和感がある場合は別の背景に置き換え）
□ カラーグレーディングが統一されているか
□ トランジションが過剰でないか
□ テキストオーバーレイのタイミングが読者対応か

【改善が必要な場合】
- 問題シーンを再生成（別プロンプトで微調整）
- または実写素材（自分が撮った映像）を挿入
- または実画面キャプチャ（Gemini 使用シーン）を挿入
```

## 第9章：よくある失敗と対策

### 9.1 デモネタイズになりやすいパターン

```
【失敗パターン1】

❌ AI開示をしない、または不十分
  「本当はAIで作ったけど、隠そう」という姿勢

- 概要欄に明確に「Gemini / Pollo / Suno を使用」
- 「すべて人間が検証しました」と明記
- 信頼構築が長期的な視点では有利

【失敗パターン3】
❌ 大量投稿（1日3本以上）
  スパム判定される可能性が高い

✅ 対策：
- 週2-3本のペースに抑える
- 品質と量のバランスを重視
```
