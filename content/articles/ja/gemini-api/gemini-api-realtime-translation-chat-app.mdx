## アーキテクチャ設計

今回構築するアプリは、以下の3つのレイヤーで構成されます。

**バックエンド（FastAPI + WebSocket）**: クライアントからのメッセージを受信し、Gemini APIで翻訳を実行してストリーミングで返す中核部分です。

**Gemini API連携層**: 翻訳プロンプトの最適化、言語検出、ストリーミングレスポンスの処理を担当します。

**フロントエンド（HTML + JavaScript）**: WebSocketでサーバーと常時接続し、翻訳結果をリアルタイムに表示するチャットUIです。

この設計により、ユーザーがメッセージを送信した瞬間から翻訳結果が逐次表示され、自然な会話体験を実現できます。

## Gemini API翻訳エンジンの実装

アプリの心臓部となる翻訳エンジンを実装します。Gemini APIのSystem Instructionsを活用して、高品質な翻訳を安定的に出力させます。

```python
# translator.py
import os
from google import genai

client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])

TRANSLATION_SYSTEM_PROMPT = """You are a professional real-time translator.
Your task is to translate the user's message into the specified target language.

Rules:
1. Translate naturally — do not produce literal, word-for-word translations.
2. Preserve the original tone and nuance (formal, casual, technical, etc.).
3. If the source language is the same as the target language, return the text unchanged.
4. Output ONLY the translated text — no explanations or annotations.
5. For technical terms, keep them in their original form if that is the common usage in the target language.
"""

LANGUAGE_DETECT_PROMPT = """Detect the language of the following text and return ONLY the ISO 639-1 language code (e.g., 'en', 'ja', 'fr', 'de', 'zh', 'ko', 'es').
Do not include any explanation. Return only the two-letter code.

Text: {text}"""


def detect_language(text: str) -> str:
    """Detect the language of the input text."""
    response = client.models.generate_content(
        model="gemini-2.5-flash",
        contents=LANGUAGE_DETECT_PROMPT.format(text=text),
    )
    return response.text.strip().lower()[:2]


async def translate_stream(text: str, target_lang: str):
    """Translate text using Gemini API with streaming response.

    Yields translated text chunks as they are generated.
    """
    # Translate the text using Gemini API with streaming
    # Expected output: translated text in the target language
    prompt = f"Translate the following text into {target_lang}:\n\n{text}"

    async for chunk in client.aio.models.generate_content_stream(
        model="gemini-2.5-flash",
        contents=prompt,
        config=genai.types.GenerateContentConfig(
            system_instruction=TRANSLATION_SYSTEM_PROMPT,
            temperature=0.3,
            max_output_tokens=2048,
        ),
    ):
        if chunk.text:
            yield chunk.text
```

ここでのポイントは、`temperature` を `0.3` と低めに設定していることです。翻訳タスクでは創造性よりも正確性が重要なため、出力のばらつきを抑えています。

## FastAPI + WebSocket サーバーの構築

次に、WebSocketを使ったリアルタイム通信サーバーを構築します。

```python
# main.py
import json
import asyncio
from fastapi import FastAPI, WebSocket, WebSocketDisconnect
from fastapi.staticfiles import StaticFiles
from fastapi.responses import HTMLResponse
from translator import detect_language, translate_stream

app = FastAPI(title="Gemini Translate Chat")

# Store connected clients
# Key: WebSocket, Value: dict with user info
connected_clients: dict[WebSocket, dict] = {}

SUPPORTED_LANGUAGES = {
    "ja": "Japanese",
    "en": "English",
    "zh": "Chinese",
    "ko": "Korean",
    "fr": "French",
    "de": "German",
    "es": "Spanish",
    "pt": "Portuguese",
    "it": "Italian",
    "ru": "Russian",
}


@app.websocket("/ws/{username}/{target_lang}")
async def websocket_endpoint(
    websocket: WebSocket, username: str, target_lang: str
):
    await websocket.accept()
    connected_clients[websocket] = {
        "username": username,
        "target_lang": target_lang,
    }

    try:
        while True:
            data = await websocket.receive_text()
            message = json.loads(data)
            original_text = message.get("text", "")

            if not original_text.strip():
                continue

            # Detect source language
            source_lang = detect_language(original_text)

            # Broadcast original message to all clients
            await broadcast(
                {
                    "type": "message",
                    "username": username,
                    "text": original_text,
                    "source_lang": source_lang,
                }
            )

            # Translate for each connected client with a different language
            for client_ws, client_info in connected_clients.items():
                client_target = client_info["target_lang"]
                if client_target == source_lang:
                    continue  # No translation needed

                # Stream translation to each client
                asyncio.create_task(
                    stream_translation_to_client(
                        client_ws,
                        original_text,
                        client_target,
                        username,
                    )
                )

    except WebSocketDisconnect:
        del connected_clients[websocket]


async def stream_translation_to_client(
    client_ws: WebSocket,
    text: str,
    target_lang: str,
    username: str,
):
    """Stream translated chunks to a specific client."""
    try:
        # Send translation start signal
        await client_ws.send_text(
            json.dumps(
                {
                    "type": "translation_start",
                    "username": username,
                    "target_lang": target_lang,
                }
            )
        )

        # Stream translated chunks
        async for chunk in translate_stream(text, target_lang):
            await client_ws.send_text(
                json.dumps(
                    {
                        "type": "translation_chunk",
                        "username": username,
                        "chunk": chunk,
                    }
                )
            )

        # Send translation end signal
        await client_ws.send_text(
            json.dumps(
                {
                    "type": "translation_end",
                    "username": username,
                }
            )
        )
    except Exception:
        pass  # Client may have disconnected


async def broadcast(message: dict):
    """Broadcast a message to all connected clients."""
    disconnected = []
    for client_ws in connected_clients:
        try:
            await client_ws.send_text(json.dumps(message))
        except Exception:
            disconnected.append(client_ws)
    for ws in disconnected:
        del connected_clients[ws]


# Serve the frontend
app.mount("/static", StaticFiles(directory="static"), name="static")


@app.get("/")
async def get_index():
    with open("static/index.html", "r") as f:
        return HTMLResponse(content=f.read())
```

このサーバーでは、各クライアントが接続時に自分の表示言語を指定し、他のユーザーからメッセージが送信されるたびに、Gemini APIがリアルタイムで翻訳してストリーミング配信します。

## よくある質問（FAQ）

### Q: 対応言語数に制限はありますか？

Gemini APIは100以上の言語をサポートしています。今回のサンプルでは7言語を設定していますが、`SUPPORTED_LANGUAGES`辞書に言語コードを追加するだけで簡単に拡張できます。Gemini 2.5 Flashは特に多言語タスクでの精度が高いモデルです。

### Q: 翻訳の精度をさらに向上させるにはどうすればいいですか？

System Instructionsにドメイン固有の用語集や文脈情報を追加することで、翻訳精度を大幅に向上させることが可能です。たとえば、医療分野のチャットであれば医学用語の対訳表を含めるとよいでしょう。また、[プロンプトエンジニアリング](/articles/gemini-basics/gemini-prompt-engineering-guide)のテクニックを活用して、翻訳の品質を段階的に改善できます。

### Q: 無料枠でどのくらい使えますか？

Gemini 2.5 Flashは無料枠で毎分15リクエスト、1日1,500リクエストまで利用できます（2026年3月時点）。個人利用や開発テストには十分な量です。本番運用で大量のリクエストが見込まれる場合は、[APIのレート制限と課金管理](/articles/gemini-api/gemini-api-rate-limiting-quota-management)の記事も参考にしてください。

### Q: WebSocketの代わりにSSE（Server-Sent Events）は使えますか？

はい、サーバーからクライアントへの一方向のストリーミングだけで十分な場合はSSEも有効な選択肢です。ただし、チャットアプリのように双方向通信が必要な場合はWebSocketの方が適しています。
