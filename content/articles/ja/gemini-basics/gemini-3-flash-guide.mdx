## Gemini 3 Flash とは

Gemini 3 FlashはGoogleが開発した最新の大規模言語モデルです。従来の「速いけど性能は控えめ」というFlashシリーズのイメージを一新し、**ProレベルのAI推論をFlashの速度とコストで実現**しています。

主な特徴は以下の通りです。

- **1Mトークンのコンテキストウィンドウ**: 膨大なテキスト・コード・ドキュメントを一度に処理可能
- **マルチモーダル入力**: テキスト、画像、音声、動画、PDFに対応
- **設定可能な推論レベル**: minimal・low・medium・highの4段階で思考深度を調整
- **自動コンテキストキャッシュ**: 繰り返し処理のコスト削減
- **構造化出力・ツール使用**: 実用的なエージェント開発に対応

## 利用方法

### Google AI Studio から試す

最も手軽な方法は[Google AI Studio](https://aistudio.google.com/)にアクセスし、モデルセレクターで `gemini-3-flash-preview` を選択することです。APIキーなしでブラウザ上で試用できます。

### Gemini API（Python）

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-3-flash-preview")

response = model.generate_content(
    "Pythonでフィボナッチ数列を生成する関数を書いてください。"
)
print(response.text)
```

### 推論レベルの設定

Gemini 3 Flashは「思考モード」を4段階で調整できます。複雑な問題には高い推論レベルを、高速応答が必要な場面には低い推論レベルを選択できます。

```python
from google.generativeai.types import GenerationConfig

config = GenerationConfig(
    thinking_config={"thinking_budget": "high"}  # minimal / low / medium / high
)

response = model.generate_content(
    "次の数学の問題を解いてください: ...",
    generation_config=config
)
print(response.text)
```

### マルチモーダル（画像入力）

```python
import PIL.Image

image = PIL.Image.open("screenshot.png")

response = model.generate_content(
    ["この画像に写っているバグを特定し、修正方法を提案してください。", image]
)
print(response.text)
```

**Q: Gemini 3 Flashは一般公開されていますか？**

**Q: 無料枠はありますか？**
A: Google AI Studioの無料ティアで一定量まで無料試用が可能です。最新の上限はAI Studioの公式サイトをご確認ください。

**Q: 既存のGemini 2.5 Flashコードはそのまま使えますか？**
A: モデル名を `gemini-3-flash-preview` に変更するだけで移行できます。APIの基本的な呼び出し方は変わりません。

**Q: Gemini 3 Flash Liteとの違いは？**
A: Gemini 3.1 Flash Liteはより低コスト・高速に特化した派生モデルです。軽量な処理には3.1 Flash Lite、高性能が必要なタスクには3 Flashを使い分けてください。
