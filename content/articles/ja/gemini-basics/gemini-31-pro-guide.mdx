## 最大の新機能：3段階思考システム

Gemini 3.1 Proの最も重要なアップデートは、**3段階の思考モード（Thinking Budget）**の導入です。

従来のGeminiは「低計算（速い）」と「高計算（深い）」の2段階でしたが、3.1 Proでは中間の「**Medium**」が加わり、開発者は用途に応じてレイテンシと推論深度のバランスを数学的に最適化できるようになりました。

### 思考モードの使い分け

| モード | 特徴 | 適した用途 |
|--------|------|------------|
| **Low** | 高速・低コスト | 簡単な質問応答、テキスト生成 |
| **Medium** | バランス型 | 要約、コードレビュー、標準的な推論 |
| **High** | 高精度・深い思考 | 数学的証明、複雑なエージェント処理 |

## 使い方：Gemini API でモデルを呼び出す

### Python SDK での基本的な呼び出し

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

# Gemini 3.1 Pro Preview を指定
model = genai.GenerativeModel("gemini-3.1-pro-preview")

response = model.generate_content(
    "量子コンピュータの基本原理を初心者向けに説明してください。",
    generation_config=genai.GenerationConfig(
        temperature=0.7,
        max_output_tokens=2048,
    )
)

print(response.text)
```

### 思考モードを指定する（Thinking Budget）

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel(
    "gemini-3.1-pro-preview",
    generation_config={
        "thinking_config": {
            "thinking_budget": "medium"  # "low" / "medium" / "high"
        }
    }
)

response = model.generate_content(
    "次の数列のパターンを解析し、次の5つの値を予測してください: 1, 1, 2, 3, 5, 8, 13, 21"
)

print(response.text)
```

### REST API での呼び出し

```bash
curl -X POST \
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-3.1-pro-preview:generateContent?key=YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "parts": [{"text": "Pythonで非同期処理を実装するベストプラクティスを教えてください。"}]
    }],
    "generationConfig": {
      "thinkingConfig": {
        "thinkingBudget": "high"
      }
    }
  }'
```

## 推論性能：ARC-AGI-2で業界最高水準

Gemini 3.1 Proは、AIの汎用推論能力を測る難関ベンチマーク**ARC-AGI-2**で**77.1%**のスコアを達成しました。これはGemini 3 Proの2倍以上の性能向上であり、現時点でトップクラスの結果です。

ARC-AGI-2は人間でも解くのが難しいパターン認識・論理推論問題で構成されており、このスコアはモデルが真の意味での「新しい問題の解決能力」を持ち始めていることを示しています。

## マルチモーダル活用例

Gemini 3.1 Proは**テキストだけでなく、画像・音声・動画・PDF・コードリポジトリ**まで対応した真のマルチモーダルモデルです。

### 画像解析の例

```python
import google.generativeai as genai
import PIL.Image

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-3.1-pro-preview")

# 画像ファイルを読み込む
image = PIL.Image.open("diagram.png")

response = model.generate_content([
    "この図の内容を詳しく分析し、改善点があれば提案してください。",
    image
])

print(response.text)
```

### PDFドキュメント解析

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

# PDFをアップロード
uploaded_file = genai.upload_file("report.pdf", mime_type="application/pdf")

model = genai.GenerativeModel("gemini-3.1-pro-preview")

response = model.generate_content([
    "このレポートの要点を3つ挙げ、実行可能なアクションプランを提案してください。",
    uploaded_file
])

print(response.text)
```
