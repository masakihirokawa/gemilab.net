# Gemini Canvas 完全ガイド — AIとリアルタイムで共同編集する新しい働き方

## はじめに

Gemini の Canvas（キャンバス）は、従来のチャット形式を超えた新しいAIインタラクションの形です。通常の会話では、AIの出力はチャットウィンドウに一方向的に表示されるだけでした。しかし Canvas を使えば、AIが生成したテキストやコードを専用のエディタ上でリアルタイムに確認し、自分で編集を加え、さらにAIに修正を依頼するという双方向の作業が可能になります。

本記事では、Canvas の基本的な使い方から、実践的なユースケースまでを初心者向けに詳しく解説します。

## Canvas とは何か

Canvas は、Gemini のチャットインターフェースの横にサイドパネルとして開く編集スペースです。Google ドキュメントのような編集画面がチャットの隣に表示され、AIと「一緒に作業する」感覚でコンテンツを作成できます。

従来のチャット形式との最大の違いは、出力の「持続性」です。通常のチャットでは、AIの回答は会話の流れの中に埋もれていきます。一方 Canvas では、作業中のドキュメントやコードが常に画面上に表示されたまま保持され、何度でも修正・加筆できます。

## Canvas の起動方法

Canvas を使い始めるのはとても簡単です。

Gemini（gemini.google.com）にアクセスし、チャット画面で以下のような指示を出すだけで、自動的に Canvas が開きます。

```
ブログ記事を書いてください：テーマは「リモートワークの生産性向上」
```

また、明示的に Canvas を指定することもできます。

```
Canvas を開いて、プレゼン用のアウトラインを作成してください
```

Canvas が起動すると、画面の右側に大きな編集パネルが表示されます。左側のチャットでAIに指示を出しながら、右側の Canvas でリアルタイムにコンテンツが生成される様子を確認できます。

## 基本操作

### テキストの編集

Canvas 上に表示されたテキストは、通常のテキストエディタと同じように直接編集できます。誤字の修正、文章の追加、段落の入れ替えなど、自由に手を加えられます。

編集した箇所はAIにも認識されるため、「先ほど私が修正した部分を踏まえて、次の段落を書いてください」といった指示も可能です。

### 部分選択による修正依頼

Canvas の強力な機能の一つが、テキストの一部を選択してAIに修正を依頼する機能です。

特定の段落や文章を選択した状態で、チャットに修正指示を入力します。

```
この段落をもっとカジュアルなトーンに書き換えてください
```

AIは選択された部分だけを書き換え、残りのコンテンツはそのまま維持します。ドキュメント全体を書き直す必要がないため、効率的に仕上げていけます。

### バージョン管理

Canvas には作業の履歴が保存されます。AIが変更を加えるたびにバージョンが作られるため、前の状態に戻したい場合は簡単にロールバックできます。大幅な書き換えを試す際も、元の内容を失う心配はありません。

## テキスト作成での活用

### ブログ記事の執筆

Canvas はブログ記事の作成に最適です。最初にアウトラインをAIに作成させ、セクションごとに内容を膨らませていくワークフローが効率的です。

手順の一例を紹介します。

まず、チャットで「AIを使った家計管理についてのブログ記事のアウトラインを Canvas で作成してください」と指示します。Canvas にアウトラインが表示されたら、各セクションの順序を自分で並べ替えます。次に「最初のセクションを500文字程度で書いてください」と依頼し、生成されたテキストを読んで必要に応じて自分で手を加えます。この流れを繰り返すことで、自分の意図を反映した高品質な記事が完成します。

### ビジネス文書の作成

提案書、報告書、メールのドラフトなど、ビジネス文書の作成にも Canvas は力を発揮します。

特に便利なのが「トーンの調整」です。下書きをAIに生成させた後、「もっとフォーマルなトーンにしてください」「具体的な数値を追加してください」「結論を先に持ってきてください」といった指示を出すことで、目的に合った文書に仕上げられます。

### 多言語コンテンツの作成

日本語で書いた文章を Canvas 上で英語に翻訳させ、さらにネイティブチェック的な修正を依頼するといった使い方も可能です。原文と翻訳文を見比べながら作業できるため、翻訳の品質を高めやすくなります。

## コード編集での活用

Canvas はコードの作成・編集にも対応しています。

### コードの生成と修正

プログラミング言語を指定してコードを生成させ、Canvas 上で直接編集できます。

```
Python で、CSV ファイルを読み込んで売上データを月別に集計するスクリプトを Canvas で書いてください
```

生成されたコードに対して、「エラーハンドリングを追加してください」「コメントを日本語で付けてください」といった追加指示を出せます。

### シンタックスハイライト

Canvas のコードモードでは、適切なシンタックスハイライトが適用されます。コードの可読性が高まり、構文エラーにも気づきやすくなります。

### コードレビューの依頼

自分で書いたコードを Canvas にペーストし、AIにレビューを依頼することも効果的です。

```
このコードのパフォーマンスを改善できる箇所を指摘してください
```

AIがコード上の改善点をインラインで指摘してくれるため、具体的にどこをどう直せばよいかが明確になります。

## 効果的に使うためのコツ

### 段階的に作業する

一度にすべてを完成させようとせず、段階的に進めるのが Canvas 活用のコツです。まずはラフな構造を作り、セクションごとに内容を充実させ、最後に全体のトーンや表現を調整するという3段階のアプローチが効果的です。

### 自分の編集を積極的に行う

AIの出力をそのまま使うのではなく、自分の言葉や知識を積極的に書き加えましょう。Canvas は「AIに全部やらせる」ツールではなく、「AIと一緒に作る」ツールです。自分のオリジナリティを加えることで、はるかに価値のあるコンテンツになります。

### 明確な指示を心がける

Canvas 上での修正依頼は、できるだけ具体的にしましょう。「もっと良くして」ではなく、「専門用語を一般的な言い回しに置き換えてください」のように指示するほど、期待通りの結果が得られます。

## Google ドキュメントへのエクスポート

Canvas で作成したコンテンツは、Google ドキュメントにエクスポートできます。チームメンバーとの共有や、さらなる編集が必要な場合に便利です。Canvas の上部にある共有ボタンから「Google ドキュメントにエクスポート」を選択するだけで完了します。

## まとめ

Gemini Canvas は、AIとの共同作業をより直感的かつ効率的にする機能です。チャット形式では難しかった「途中経過を見ながら一緒に作り上げる」という体験が実現され、テキスト作成からコード編集まで幅広い用途に活用できます。

まずは簡単なブログ記事の下書きから試してみてください。AIと一緒に編集する新しいワークフローが、あなたの作業効率を大きく変えてくれるはずです。
