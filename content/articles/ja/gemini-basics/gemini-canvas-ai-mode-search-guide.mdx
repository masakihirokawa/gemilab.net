## Gemini Canvas × AI Modeとは？

2026年3月、Googleは **Google検索のAI Mode** に **Canvas（キャンバス）** 機能を全米ユーザー向けに正式公開しました。これまでCanvasはGeminiアプリ内で利用できる機能でしたが、今回の統合により、**検索画面から直接** コード生成・文書作成・プロジェクト整理が行えるようになりました。

「ちょっとしたアプリを作りたい」「調べものをそのままレポートにまとめたい」——そんな場面で、もうブラウザタブを行き来する必要はありません。AI Modeの中で調査から制作までをシームレスに完結できます。

この記事では、Gemini Canvas × AI Modeの基本的な使い方から、実践的な活用テクニックまでをわかりやすく解説します。

## AI Modeを有効にする方法

Canvas機能を使うには、まずGoogle検索のAI Modeを有効にする必要があります。

### ステップ1: AI Modeへのアクセス

1. [google.com](https://google.com) にアクセスします
2. 検索バーの上部にある **「AI Mode」** タブをクリックします
3. 初回利用時はオプトインの確認画面が表示されるので、同意して進みます

### ステップ2: Canvasを開く

AI Mode内でCanvasを開くには、以下の手順を踏みます。

```
1. AI Modeの入力欄横にある「+」ボタン（ツールメニュー）をクリック
2. 表示されるメニューから「Canvas」を選択
3. サイドパネルが展開され、Canvasのワークスペースが表示される
```

これでCanvasが利用可能になります。プロンプトを入力すれば、検索結果やGoogleのナレッジグラフの情報を活用しながら制作を進められます。

## Canvasでできる3つのこと

### 1. アプリやゲームをその場で作成（Vibe Coding）

Canvasの最も注目される機能が、**自然言語でアプリを作れる** ことです。「BMI計算ツールを作って」「しりとりゲームを作って」のように話しかけるだけで、HTMLやJavaScriptのコードが自動生成されます。

```javascript
// Canvas で生成されるアプリの例
// 「シンプルなタイマーアプリを作って」とプロンプトすると
// 以下のようなコードが自動生成される

function startTimer(seconds) {
  const display = document.getElementById('timer');
  let remaining = seconds;

  const interval = setInterval(() => {
    const min = Math.floor(remaining / 60);
    const sec = remaining % 60;
    display.textContent = `${min}:${sec.toString().padStart(2, '0')}`;

    if (remaining <= 0) {
      clearInterval(interval);
      display.textContent = "Time's up!";
    }
    remaining--;
  }, 1000);
}

// 期待される出力:
// ブラウザ上でカウントダウンタイマーが動作し、
// 0になると "Time's up!" と表示される
```

生成されたアプリはCanvas内でプレビュー・テスト・修正が可能です。さらに、**共有リンクを生成** して他の人に見せることもできます。

### 2. ドキュメントの作成・編集

検索で得た情報を元に、長文のドキュメントを作成・編集できます。

- **レポート作成**: 調査テーマを入力すると、Web上の情報を整理した下書きが生成される
- **文章のリライト**: 既存のテキストをペーストして「もっとカジュアルに」「ビジネス文書風に」と指示できる
- **翻訳・要約**: 検索で見つけた英語の記事を日本語に要約する使い方も便利

### 3. リサーチ・プロジェクトの整理

Canvasは単なるエディタではなく、**検索エンジンと連動したワークスペース** です。

- 授業のノートをアップロードして学習ガイドを自動作成
- リサーチレポートをWebページやクイズに変換
- 旅行の計画をCanvasで整理し、予算計算ツールも同時に作成

このように、「調べる → まとめる → 形にする」という一連の作業をAI Mode内で完結できるのが最大の特徴です。

## 実践チュートリアル: Canvasで簡易Webアプリを作る

ここでは、実際にCanvasを使って簡単なWebアプリを作る手順を紹介します。

### ステップ1: AI Modeでテーマを検索

AI Modeの検索バーに「体重管理アプリを作りたい」と入力します。AI Modeが関連する健康情報やBMI計算の知識を検索から取得し、背景情報を提示してくれます。

### ステップ2: Canvasでアプリ生成

ツールメニュー（+）からCanvasを開き、以下のようにプロンプトを入力します。

```
身長と体重を入力するとBMIを計算し、
結果に応じてアドバイスを表示するWebアプリを作ってください。
デザインはシンプルでモダンなものにしてください。
```

### ステップ3: プレビューとテスト

Canvasのサイドパネル内でアプリのプレビューが表示されます。実際に値を入力してテストし、問題があればチャットで修正を依頼します。

```
「結果にグラフを追加して」
「色をダークモードに変えて」
「入力バリデーションを追加して」
```

このように対話的に改善を繰り返せるのが、Canvasの大きな魅力です。

### ステップ4: 共有

完成したアプリは **共有リンク** を発行して、友人やチームメンバーに共有できます。コードをダウンロードして自分のサーバーにデプロイすることも可能です。

## Geminiアプリ内Canvasとの違い

Geminiには以前から **Geminiアプリ内のCanvas** が存在しますが、AI Mode版のCanvasにはいくつかの重要な違いがあります。

| 項目 | Geminiアプリ内Canvas | AI Mode Canvas |
|------|---------------------|----------------|
| アクセス方法 | gemini.google.com | google.com（AI Modeタブ） |
| 検索連携 | 限定的 | Google検索・ナレッジグラフと完全連携 |
| 主な用途 | コード編集・文書作成 | 検索→制作のワンストップ作業 |
| リアルタイム共同編集 | 対応 | 今後対応予定 |
| モバイル対応 | フル対応 | 現在はデスクトップ中心 |

既にGeminiアプリのCanvasに慣れている方は、AI Mode版を「検索エンジンに組み込まれたCanvas」と理解するとわかりやすいでしょう。Geminiアプリ版Canvasの詳細については、[Gemini Canvas リアルタイム共同編集ガイド](/articles/gemini-basics/gemini-canvas-realtime-collaborative-editing)もご参照ください。

- **フリーランサー**: クライアント向けの提案書をCanvas上で作成

- **個人開発者**: ちょっとしたツールやユーティリティを検索→制作で即完成

## 現在の制限事項

2026年3月時点での制限事項も把握しておきましょう。

- **対応言語**: 英語のみ（日本語対応は今後のアップデートで予定）
- **対応地域**: 米国のみ
- **モバイル**: 一部機能がデスクトップ限定
- **リアルタイム共同編集**: AI Mode版ではまだ未対応
- **データ永続性**: Canvas内で作成したプロジェクトの保存期間に制限あり

日本語環境での利用を待っている方も多いかと思いますが、VPNを使った米国アクセスでは利用可能な場合があります。正式な国際展開のスケジュールは今後のGoogleの発表を待ちましょう。

## FAQ

### Q1: AI ModeのCanvasは無料で使えますか？

はい、AI ModeのCanvas機能は無料で利用できます。Google検索にログインしていれば、追加のサブスクリプションは不要です。ただし、一部の高度な機能（大規模なコード生成など）はGoogle AI ProやUltraプランの方がより高い利用上限で使えます。

### Q2: Canvasで生成したコードの品質はどの程度ですか？

Canvasが生成するコードは、簡単なWebアプリやツール程度であれば十分に実用的です。Geminiの最新モデルが裏側で動いており、HTML/CSS/JavaScript を中心に、すぐ動作するコードを出力します。本格的なプロダクション開発には [Google AI Studio](/articles/gemini-basics/gemini-complete-guide-2026) や専用のIDEを使うことをお勧めします。

### Q3: 日本からCanvasを使うにはどうすればよいですか？

2026年3月時点では、AI Mode Canvasは米国で英語のみの提供です。日本からの利用は公式にはまだサポートされていませんが、Googleは国際展開を計画中と発表しています。一方、Geminiアプリ内のCanvasは日本からもアクセス可能ですので、同様の機能が必要な場合はそちらをご利用ください。

### Q4: 作成したアプリやドキュメントはエクスポートできますか？

はい、Canvasで作成したコードはダウンロードが可能です。また、ドキュメントはコピーして他のアプリに貼り付けたり、共有リンクを通じて公開したりできます。

## まとめ

Gemini Canvas × AI Modeは、**Google検索の中で調査から制作までを完結できる** 画期的な機能です。これまで別々のツールで行っていた「検索 → 情報整理 → コンテンツ制作」のワークフローが、一つの画面に統合されました。

現時点では米国・英語限定ですが、今後の国際展開により、日本のユーザーにとっても身近なツールになることが期待されます。まずはGeminiアプリ内の[Canvas機能](/articles/gemini-basics/gemini-canvas-realtime-collaborative-editing)で操作に慣れておくと、AI Mode版がリリースされたときにスムーズに活用できるでしょう。
