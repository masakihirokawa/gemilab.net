## Google Gemini TTS で高品質な音声を生成

Google Gemini には2つのTTS品質があります。

### Flash TTS（低遅延・スピード重視）
```python
import anthropic

client = anthropic.Anthropic(api_key="YOUR_GEMINI_API_KEY")

text = """
ブログ記事の内容がここに入ります。
Gemini TTS API は高品質な日本語音声を提供します。
"""

# Gemini Flash TTS で音声生成
response = client.messages.create(
    model="gemini-2.0-flash",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": f"以下のテキストを自然な日本語で朗読してください:\n{text}"}
    ]
)

print(response.content[0].text)
# 出力例: 音声ファイルのURL または base64エンコード音声
```

**推奨用途**: リアルタイム配信、コスト最小化

### Pro TTS（高音質・自然度重視）
- より自然な抑揚と間合い
- 複雑な漢字・固有名詞への対応が強い
- 1000文字あたり約$0.002（格安）

**選び方のコツ**:
- 短編・ニュース系 → Flash TTS
- 深掘りガイド・長編 → Pro TTS

## Veo 3.1で背景動画を秒単位で自動生成

ナレーション音声に合わせて、背景動画が必要です。Veo 3.1はテキストから動画を生成できます。

```python
# Veo 3.1 APIの利用イメージ
import requests

prompt = "未来的な図書館、本が浮かぶ、淡い光"
duration = 8  # 秒
resolution = "720p"  # 最大 4K

response = requests.post(
    "https://api.google.com/v1/veo/generate",
    json={
        "prompt": prompt,
        "duration": duration,
        "resolution": resolution,
    },
    headers={"Authorization": f"Bearer {API_KEY}"}
)

print(f"生成動画: {response.json()['video_url']}")
# 出力例: https://storage.googleapis.com/veo-videos/abc123.mp4
```

**Veo 3.1の特徴**:
- 最大8秒、720p～4Kまで自由に生成
- 日本語プロンプト対応
- 生成速度: 平均30秒～2分
- コスト: $0.01～0.05/本

**朗読動画向けの活用方法**:
1. ブログ記事の「段落ごと」にキーワード抽出
2. キーワードに合わせてVeo 3.1で背景動画生成
3. ナレーション音声と重ねる

## Vrewで自動字幕生成とテキストベース編集

YouTube動画は **字幕がある方が視聴完了率が30%以上高い** というデータがあります。Vrewなら自動で対応できます。

### Vrewでできること
- 音声から自動字幕生成（60+ 言語対応、日本語の精度99%）
- テキストベースでのカット編集（デザイナー不要）
- 自動レイアウト・テンプレート適用
- 複数トラック同時編集

### 実際の手順
1. Vrewにアップロード → 自動字幕生成（1～2分）
2. 編集画面でテキストを修正（必要に応じて）
3. フォント・色・背景を選択
4. MP4で書き出し（YouTubeアップロード対応）

**推奨設定**:
- フォントサイズ: 48pt～56pt（スマホでも読みやすく）
- 背景透明度: 40～60%（動画が見える程度に）
- 字幕表示時間: 音声長＋0.2秒（違和感なし）

## Gemini を使ったスクリプト最適化

ブログ記事をそのままナレーションすると、冗長になったり、聞き辛くなったりします。Gemini を使ってスクリプト化しましょう。

```
[Gemini プロンプト例]

以下のブログ記事を「YouTube朗読動画用のナレーションスクリプト」に変換してください。

要件:
1. 300～500文字
2. 自然な日本語の話言葉
3. 句読点は「、」と「。」のみ
4. 数字は「アラビア数字」のまま
5. 固有名詞は括弧で強調: 「Gemini API」→「Gemini（ジェミナイ） API」

元記事:
[ブログ記事の本文をペースト]
```

Gemini の結果を Vrew にペーストして自動字幕化すると、ナレーションと字幕が完璧に同期します。

## 実際のワークフロー手順（全5ステップ）

### ステップ1: ブログ記事の選定・準備
- 2000～3000字の記事を選ぶ（5～8分のナレーション長）
- タイトル、見出し、重要文を抽出

### ステップ2: Gemini でスクリプト生成
- 上記のプロンプトで変換
- 読み上げテスト（自分で音読して違和感チェック）

### ステップ3: Gemini TTS で音声化
- Flash TTS → Pro TTS の順に試す
- 納得いくまで何度でも再生成可能（コスト数円）

### ステップ4: Veo 3.1 で背景動画生成
- ブログ記事のキーワード＆イメージから背景プロンプト作成
- Veo 3.1で生成（平均1～2分待機）

### ステップ5: Vrew で最終編集
- 音声ファイル + 動画ファイル をインポート
- 自動字幕生成・調整
- YouTubeサムネイル用スクリーンショット取得
- MP4書き出し

**全体所要時間**: 30分～1時間

## 初心者向けFAQ

### Q1: 「Gemini Flash TTS で日本語の品質は十分?」
**A**: 短編・ニュース形式なら十分です。ただし長編やナレーションのしっかり感を出したい場合は Pro TTS がおすすめ。無料で両方試せるので、まずFlashで試してみてください。

### Q2: 「Veo 3.1 で生成した動画は著作権フリー?」
**A**: はい。Google が生成した動画はあなたの著作物として扱えます。YouTubeにアップロード、商用利用OK。ただし人物の顔認識には要注意（実在の人物にそっくりな生成は避ける）。

### Q3: 「毎日朗読動画を作りたいなら、コストはいくら?」
**A**: 1本あたり目安:
- Gemini TTS: $0.002～0.01
- Veo 3.1: $0.01～0.05
- Vrew: 月額$7～15（無料版でも可）
- **合計**: $0.012～0.06/本 = 月額$0.36～1.80（1日1本の場合）

ほぼ無料に近いです。

### Q4: 「既存の YouTube チャンネルをこのワークフローで拡大できる?」
**A**: できます。実例として、毎日技術ニュースを朗読するチャンネルが Veo + Gemini で登録者を3倍に増やした例があります。ポイントは **継続** と **一定の品質** です。
