# Gemini の画像生成機能の使い方 — プロンプトのコツと実践テクニック

「Gemini で画像が生成できるって聞いたけど、どうやって使うの？」「プロンプトって何を書いたらいい？」という質問は非常に多いです。

実は、画像生成は「運に頼る」のではなく「テクニック」です。このガイドでは、誰でも理想に近い画像を生成できるプロンプトの書き方を解説します。

## Gemini の画像生成とは

Gemini には「ImageFX」という画像生成ツールが統合されています。

**特徴：**
- テキストだけから画像を生成
- 無料版でも月に何十枚も生成可能
- Google アカウントがあればすぐに使える
- 日本語での指示が可能

## プロンプトの基本構造

良い画像を生成するコツは「正確で詳しい指示」です。

**基本的な構造：**

```
[被写体] が [状況/背景] で [スタイル] というスタイルで描かれている、[特徴1]、[特徴2] な画像。
```

では、具体例で説明します。

### 例 1: 初心者向け（シンプル）

```
黄色い犬が公園で遊んでいる、明るい晴れた日の風景。
```

これでも画像が生成されますが、もっと具体的にできます。

### 例 2: 中級者向け（詳しい）

```
黄色いラブラドール・レトリーバーが、青い空の元、公園の草原で元気にボールを咥えて走っている。
日光が降り注ぎ、暖かい春の午後という雰囲気。
写真的なリアルな描画スタイル。
```

この方が、理想に近い画像が生成される確率が高まります。

## プロンプトの主要な要素

### 1. 被写体（何を描くか）

最も大切です。ここが曖昧だと、話になりません。

**良い例:**
- 茶色い短毛の猫
- 赤いタートルネックを着た 30 代女性
- 古い木造の一軒家

**悪い例:**
- 人
- かわいいもの
- 自然

被写体は、できるだけ具体的に。

### 2. 背景・環境

被写体だけでなく、周囲の環境も大切です。

**例：**
- 「オフィスのデスク」か「カフェ」か「図書館」か → 全く違う画像
- 「晴れ」か「曇り」か「夜」か → 雰囲気が全く変わる
- 「都市」か「田舎」か「宇宙」か → 背景が大きく異なる

### 3. スタイル・雰囲気

同じ被写体でも、描画スタイルで全く違う画像になります。

**スタイルの種類：**
- 写真的（photorealistic）
- イラスト風
- 油絵風
- 水彩画風
- 3D CGI
- アニメ風
- 漫画風
- スケッチ風

**例：**
- 「写真的なリアルな」
- 「柔らかい水彩画のような」
- 「ジブリアニメ風の」

### 4. 特徴・ディテール

細部にこだわることで、完成度が上がります。

**例：**
- 色合い：「暖色系」「寒色系」「パステルカラー」
- 感情：「楽しそう」「落ち着いた」「冒険的」
- 照明：「逆光」「スポットライト」「自然光」
- アングル：「正面から」「俯瞰」「ローアングル」

## 用途別プロンプト集

では、実際の用途に合わせたプロンプト例を紹介します。

### 用途 1: SNS 投稿用のアイキャッチ画像

Instagram や Twitter に投稿する画像が欲しい場合。

**例 1:「ヨガ・ウェルネス系」**
```
ヨガマットの上で瞑想をしている 20 代女性。
緑色のヨガウェア。
背景は明るい朝日が差し込む広い白いスタジオ。
リラックスした表情。
柔らかいパステル調の色合い、雑誌の表紙のような洗練されたプロフェッショナルな写真風。
```

**例 2:「コーヒー・ライフスタイル系」**
```
カフェのテーブルに置かれた温かいコーヒーカップ。
手帳、ペン、小さな観葉植物が一緒に。
暖かい午後の逆光が差し込んでいる。
ほっこりした、落ち着いた雰囲気。
温かみのある色合い、Instagream映えする写真風。
```

**例 3:「ビジネス系」**
```
モダンなオフィスで、複数の人がミーティングをしている。
ホワイトボードにカラフルなペンで図を描いている。
自然光が窓から差し込んでいる。
協力的で前向きな雰囲気。
明るく清潔感のある、プロフェッショナルな写真風。
```

### 用途 2: ブログのアイキャッチ

ブログ記事の上に置く画像。記事のテーマに合わせる必要があります。

**例 1:「初心者向けプログラミング記事」**
```
机の上のノートパソコンの画面にプログラミングコードが映っている。
コップにはコーヒー。
手帳とペンも一緒に置いてある。
学習中の雰囲気。
明るい、親しみやすい、優しい色合い。
イラスト風かやや柔らかい写真風。
```

**例 2:「子育てハウツー記事」**
```
お母さんと小さな子どもが一緒に読書をしている。
お母さんは優しく微笑んでいて、子どもは楽しそう。
背景はリビングで、温かい照明が当たっている。
愛情的で温かい雰囲気。
パステル調のやさしい色合い。
イラスト風か柔らかい写真風。
```

### 用途 3: プレゼンテーション資料用

PowerPoint や Keynote に挿入する画像。

**例 1:「データ分析プレゼン」**
```
グラフとチャートが浮かんでいる、未来的で技術的な背景。
青と紫のネオンカラーをメインに。
3D CGI スタイル。
前向きで革新的な雰囲気。
```

**例 2:「環境問題プレゼン」**
```
地球が手のひらに乗っている。
手のひらは大切に包み込むようなポーズ。
背景は緑色が基調。
希望と責任感を感じさせるイラスト風。
自然な、温かい色合い。
```

### 用途 4: 個人ブランディング用

LinkedId や自社サイトに使う画像。

**例 1:「コンサルタント向け」**
```

背景は高いビルが立ち並ぶ都市風景。
昼間、青空。

プロフェッショナルな写真風。
```

**例 2:「クリエイター向け」**
```

背景はカラフルなアトリエ。
楽しそう、創造的な雰囲気。
活気に満ちた、ポップな色合い。
イラスト風。
```

## よくある失敗と対策

### 失敗 1: 被写体の指定が曖昧

**悪い例：**
「人が座っている」

**改善：**
「30 歳代のアジア人女性が、ソファに座っている。黒い髪で、カジュアルな服装。」

### 失敗 2: スタイルの指定がない

**悪い例：**
「カフェのテーブルの上のコーヒー」

**改善：**
「カフェのテーブルの上のコーヒー。暖かい色合いの写真風。プロフェッショナルな雑誌の表紙のような仕上がり。」

### 失敗 3: 矛盾した指示

**悪い例：**
「暗い夜の場面なのに、明るい昼間の風景」

**改善：**
「夜のカフェ。暖かい室内照明に照らされている。外の窓には夜景が見える。」

### 失敗 4: 負の表現を使ってしまう

画像生成 AI は「〜しない」という負の指示に弱いです。

**悪い例：**
「医者が注射をしていない。患者が痛くない顔をしている。」

**改善：**
「医者が患者に温かく微笑みかけている。患者は安心した表情をしている。」

## テクニック：より良い画像を生成するコツ

### テクニック 1: 複数回、別のプロンプトで試す

画像生成は、実は「運」の要素もあります。

同じプロンプトでも、何度か実行すると、時々素晴らしい画像が出てくることがあります。

気に入らなかったら、何度も試してみてください。

### テクニック 2: 「参考スタイル」を明記する

「〜のような」という表現を使うと、スタイルが指定しやすい。

**例：**
- 「Vogue 雑誌のファッションフォトのような」
- 「ジブリ映画『千と千尋の神隠し』のような」
- 「Instagram インフルエンサーの投稿のような」

AI は「参考」を理解するのが得意です。

### テクニック 3: 色合いを詳しく指定

同じ被写体でも、色合いで雰囲気が全く変わります。

**例：**
- 「暖色系、オレンジとゴールドをメイン」
- 「寒色系、青と紫」
- 「モノトーン、グレースケール」
- 「パステルカラー」

### テクニック 4: 感情を明記する

被写体の「気持ち」を指定することで、画像の雰囲気が定まります。

**例：**
- 「楽しそう」「悲しそう」「真剣」「リラックス」
- 「希望に満ちた」「落ち込んだ」「冒険的」「静かな」

### テクニック 5: サイズ・アスペクト比を考える

用途によって、必要なサイズが異なります。

- **SNS**: スクエア（1:1）や縦長（9:16）
- **ブログ**: 横長（16:9）
- **プレゼン**: 横長（16:9）

Gemini では、生成後にサイズ調整できることが多いです。

## 実践的なプロンプト作成フロー

では、実際に「良いプロンプト」を作るステップを示します。

### ステップ 1: 目的を明確にする

「何に使う画像なのか」を明確に。
- SNS 投稿？
- ブログのアイキャッチ？
- プレゼン資料？

### ステップ 2: 被写体を詳しく決める

「何を中心に描きたいか」を明確に。
- 人？ 物？ 風景？
- 年齢、性別、服装は？
- 大きさや位置は？

### ステップ 3: 背景・環境を決める

「どこで、どんな時間帯か」を決定。
- 室内？ 屋外？
- 昼間？ 夜？
- 季節は？

### ステップ 4: スタイル・色合いを決める

「どんな雰囲気か」を決定。
- 写真的？ イラスト？
- 暖色系？ 寒色系？
- 明るい？ 暗い？

### ステップ 5: プロンプトを文章化して実行

テンプレートに当てはめて、プロンプトを完成。

### ステップ 6: 複数回試して、最高のものを選ぶ

気に入らなければ、プロンプトを微調整して再実行。

## 最後に：実験と試行錯誤

Gemini の画像生成は、奥が深いです。

最初は「なんか違う画像が出てきた」と感じるかもしれません。

でも「プロンプトのこの部分を変えたら、こういう画像が出た」という試行錯誤を繰り返すと、だんだん理想の画像に近づきます。

大切なのは「丁寧な指示」と「試行錯誤」です。

ぜひ、試してみてください。きっと、素晴らしい画像が生成できます。
